# 一致性指数（Concordance Index, C-index）

## 1. 方法概览

### 1.1 一句话本质

C-index 从所有能够判断结局先后的个体对中，计算模型把更早发生事件者排在更高风险位置的比例。

### 1.2 定义

一致性指数衡量预测模型的区分度：随机取一对可比较个体，模型给「更早发生事件者」更高风险的概率。它是 ROC 曲线下面积在生存数据上的推广。

### 1.3 它主要解决什么问题

- 研究问题：预测模型能否把高风险个体与低风险个体正确排序。
- 适用任务：生存/风险预测模型的区分度评估与比较。
- 常见医学场景：预后评分、复发风险模型、列线图的判别能力评价。

### 1.4 直觉理解

好的风险模型应给「结局更差的人」更高的风险分。C-index 就是把所有能比较的个体两两配对，数一数模型排序正确的比例——0.5 相当于抛硬币，1.0 为完美排序。

## 2. 核心思想与原理

### 2.1 根本困难

生存结局既有事件发生时间，又有删失。对两个患者，如果一人先发生事件，就能判断谁结局更早；但若一人在另一人发生事件前已经失访，就无法确定两人的真实先后。

### 2.2 关键洞察

只在结局先后可判定的个体对中检查风险排序：先发生事件者风险分更高记为一致，方向相反记为不一致，风险并列通常记半分。Harrell C 直接计算这些对；Uno C 再以删失分布的逆概率权重修正重删失造成的偏倚。

### 2.3 它回答与不回答的问题

C-index 回答“谁更早发生事件，模型能否排对”，不回答“患者在 3 年时发生事件的概率是否准确”，也不衡量按某阈值采取治疗能否带来净获益。因此应与校准和 DCA 联合报告。

## 3. 数学形式

### 3.1 核心公式

$$
C=\frac{\#\{\text{可比对且预测排序一致}\}}{\#\{\text{可比对}\}}
$$

即在所有可比较对 $(i,j)$（结局先后可判定）中，预测风险与实际结局排序一致的比例。

### 3.3 参数或统计量含义

- 可比对：两个体的事件先后可确定（考虑删失后仍可判定者）。
- 一致对：实际先发生事件者被赋予更高风险。
- $C=0.5$ 无区分度，$C=1$ 完美；二分类结局时 $C=\mathrm{AUC}$。
- 时依 C-index 可评估某一时间窗内的区分度。

### 3.4 关键假设

- 风险评分与结局排序的方向一致。
- 删失为非信息性；重删失会使 Harrell C 偏乐观（可用 Uno C 校正）。
- 只衡量排序（区分度），不衡量校准。

## 4. 手把手算例

考虑 4 位患者，风险分越大表示越危险：

| 患者 | 观察时间 | 事件 | 风险分 |
| --- | ---: | ---: | ---: |
| A | 2 | 1 | 0.90 |
| B | 4 | 1 | 0.60 |
| C | 6 | 0 | 0.30 |
| D | 3 | 0 | 0.95 |

可比较对及其排序如下：

- A–B：A 在时间 2 发生事件且风险更高，一致。
- A–C：A 在 C 删失前发生事件且风险更高，一致。
- A–D：A 在 D 删失前发生事件，但 D 风险更高，不一致。
- B–C：B 在 C 删失前发生事件且风险更高，一致。
- B–D：D 在 B 发生事件前已删失，真实先后未知，不可比较。
- C–D：两者均未观察到事件，不可比较。

因此 4 个可比较对中有 3 个一致：

$$
C=\frac{3}{4}=0.75
$$

若 A 与 D 风险分相同，通常给该对 0.5 分，此时 $C=(3+0.5)/4=0.875$。这个算例也说明删失比例和删失发生时点会改变哪些个体对进入分母。

## 5. 数据形式与输入输出

### 5.1 适合的数据形式

- 自变量类型：模型输出的风险评分/线性预测子。
- 因变量类型：二分类或时间到事件（含删失）。
- 数据结构：每个体的预测风险 + 实际结局（+ 生存时间与事件指示）。
- 是否适合高维数据：对模型输出评分即可，与维度无关。
- 是否适合缺失较多数据：需完整的评分与结局。
- 是否适合删失数据：适用（Harrell/Uno C）。
- 是否适合重复测量数据：需时依 C-index 扩展。

### 5.2 示例表格

| 个体 | 风险评分 | 生存时间 | 事件 |
| --- | --- | --- | --- |
| 1 | 2.1 | 12 | 1 |
| 2 | 0.4 | 40 | 0 |
| 3 | 1.5 | 20 | 1 |
| 4 | 0.9 | 40 | 0 |

### 5.3 输入与产出

#### 输入

- 输入数据：预测风险评分、结局（及生存时间/事件）。
- 关键变量：风险评分、time、event。
- 需要预处理的内容：确认评分方向、选择 Harrell 或 Uno C。

#### 产出

- 模型对象/统计结果：C-index 及置信区间。
- 参数估计：一致对比例。
- 预测结果：无（是评估指标）。
- 不确定性指标：C 的 95% CI，两模型 C 的比较检验。

## 6. 适用场景

- 适合：比较模型区分能力、报告预后模型判别度。
- 不适合：评估校准（用 [[校准曲线（Calibration Curve）]]）或临床净获益（用 [[决策曲线分析（Decision Curve Analysis, DCA）]]）。
- 使用前需要特别检查的点：删失比例、是否需 Uno C、是否需时依评估。

## 7. 实现

### 7.1 Python

常用包：

- `lifelines`

```python
from lifelines.utils import concordance_index

# 事件时间、预测风险(越大越危险取负号)、事件指示
c = concordance_index(df["time"], -df["risk_score"], df["event"])
print("C-index =", round(c, 3))
```

### 7.2 R

常用包：

- `survival`

```r
library(survival)
fit <- coxph(Surv(time, event) ~ risk_score, data = df)
summary(fit)$concordance          # C-index 及标准误
# 或直接:
concordance(Surv(time, event) ~ risk_score, data = df)
```

## 8. 结果如何解释

- 核心结果看什么：C 值大小与置信区间是否明显高于 0.5。
- 每个主要参数如何解释：约定俗成——$0.5$ 无用、$0.7$ 尚可、$0.8$ 良好、$\gt 0.9$ 很强（依场景而定）；这些阈值不能替代具体临床场景判断。
- 临床或医学意义如何表达：C 高说明模型能有效排序高低风险，但不代表预测概率准确。
- 常见误读：把高 C 当成校准好；忽视 Harrell C 在重删失下偏乐观。

## 9. 假设诊断与稳健性

- **确认评分方向**：先用几个已知高低风险病例核对方向；软件对“更长生存分”和“更高风险分”的约定可能相反。
- **描述删失机制**：报告删失比例、随访长度和风险集变化；删失与预后相关时，普通 Harrell C 的解释会受损。
- **比较估计量**：重删失或不同随访分布下同时计算 Harrell C 与 Uno C，说明时间截断点 $\tau$ 的选择。
- **使用独立预测**：在独立测试集或 out-of-fold 预测上计算；训练集 C-index 会因过拟合而偏高。
- **量化不确定性**：用患者层面的 bootstrap 给出置信区间；比较模型时对同一重抽样内的差值取区间，而非只看两个点估计。
- **检查时间异质性**：比例风险不成立或排序能力随时间变化时，补充时间依赖 AUC/C-index 曲线。
- **联合评价**：同步报告校准曲线、Brier 分数与决策曲线，避免把排序好误写成绝对风险准确或临床有用。

## 10. 推荐可视化

- 不同模型 C-index 的对比（含 CI）森林图。
- 时依 C-index 曲线。
- 风险分层的 Kaplan-Meier 曲线佐证区分度。

## 11. 优势、局限与常见坑

### 优势

- 单一、可解释的区分度指标，广泛用于预后模型。
- 处理删失，二分类时退化为 AUC。
- 便于跨模型比较。

### 局限

- 只反映排序，不反映校准与临床效用。
- Harrell C 在重删失时偏乐观。
- 对已很强的模型，改进的边际增量常很小。

### 常见坑

- 只报 C-index 而不报校准与净获益。
- 用训练集 C（乐观）而非验证/交叉验证 C。
- 评分方向搞反导致 $C\lt 0.5$。

## 12. 与相近方法的区别

- 和 [[ROC曲线与AUC（Receiver Operating Characteristic and AUC）]] 的区别：AUC 针对固定二分类结局，C-index 推广到含删失的生存结局。
- 和 [[校准曲线（Calibration Curve）]] 的区别：C 看区分度，校准看概率准确性，二者互补。
- 和 [[决策曲线分析（Decision Curve Analysis, DCA）]] 的区别：DCA 评估临床净获益而非排序能力。

## 13. 医学研究中的典型应用

- 肿瘤预后评分/列线图的判别能力评价。
- 心血管风险模型的区分度比较。
- 新生物标志物加入后区分度是否提升。

## 14. 关键术语

- **可比较对**：根据事件与删失信息能够确定结局先后的两个个体。
- **一致对**：更早发生事件者同时被模型赋予更高风险。
- **不一致对**：实际事件顺序与预测风险顺序相反的个体对。
- **Harrell C**：直接基于可比较对的经典一致性指数。
- **Uno C**：以删失分布逆概率加权、对重删失更稳健的一致性指数。
- **风险评分**：用于排序的模型输出；数值方向必须明确。
- **时间依赖区分度**：允许模型判别能力随预测时间点变化的评价。
- **乐观偏倚**：在模型开发数据上评估造成的性能高估。
