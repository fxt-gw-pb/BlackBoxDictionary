# 贝叶斯回归（Bayesian Regression）

## 1. 方法概览

### 1.1 一句话本质

贝叶斯回归把回归系数当作**有先验分布的随机量**，用数据把先验更新成后验分布——输出不再是一个点估计加区间，而是参数与预测的整条概率分布，不确定性天然内建。

### 1.2 定义

贝叶斯回归在回归系数上放置先验分布，通过贝叶斯定理结合似然得到后验分布，用后验做参数推断与预测。先验起正则化作用，后验预测分布给出自然的不确定性。

### 1.3 它主要解决什么问题

- 研究问题：如何在纳入先验知识的同时，直接得到「参数取各值的概率」与带不确定性的预测？
- 适用任务：小样本回归、需要正则化与不确定性量化、层次/分层建模。
- 常见医学场景：小样本或稀疏数据的效应估计、纳入既往证据作先验、需要预测区间指导决策。

### 1.4 直觉与类比

频率派问「如果把研究重复无数次，估计会怎样散」；贝叶斯派直接问「看了这批数据后，参数最可能是多少、有多确定」。就像医生看病：先有基于经验的先入判断（先验），再根据检查结果（似然）更新成最终判断（后验）。数据越多，最终判断越被数据主导；数据越少，先验的影响越大。

## 2. 核心思想与原理

### 2.1 它到底在解决什么根本困难

频率派的系数是固定未知常数，不确定性靠「假想重复抽样」间接表达（置信区间），小样本时正则化与不确定性量化都不自然，也难以正式纳入既往知识。根本困难是：**如何直接、连贯地表达「参数本身的不确定性」，并把先验证据融进来？**

### 2.2 关键洞察

把参数看成随机变量，用**贝叶斯定理**更新信念：

$$
\text{后验}\propto\text{似然}\times\text{先验}.
$$

先验对系数施加「收缩」——高斯先验等价于 Ridge 惩罚（后验众数 = Ridge 解），拉普拉斯先验等价于 Lasso。数据一到，后验在「先验」和「数据」之间做**精度加权平均**：谁更「确定」（精度高），后验就更靠向谁。最终得到的不是一个数，而是整条后验分布，预测也变成后验预测分布，区间自然浮现。

### 2.3 与朴素/相邻做法的对比

- 相对 **OLS**：贝叶斯给分布而非点估计，小样本更稳、可纳入先验。
- 相对 **Ridge/Lasso**：正则化是贝叶斯的「后验众数（MAP）」特例；贝叶斯还额外给完整不确定性。
- 相对**频率派 CI**：贝叶斯的**可信区间**可直接说「参数有 95% 概率在此」，解释更直观。

## 3. 数学形式

### 3.1 核心公式

$$
p(\boldsymbol{\beta}\mid D)\ \propto\ p(D\mid\boldsymbol{\beta})\,p(\boldsymbol{\beta})
$$

高斯似然 + 高斯先验（共轭）时后验仍是高斯。这个式子在说：看到数据后对系数的信念（后验），正比于「数据在该系数下的可能性（似然）」乘以「事先的信念（先验）」。

### 3.2 推导脉络

- 似然：$y\mid\boldsymbol{\beta}\sim N(X\boldsymbol{\beta},\sigma^2 I)$；先验：$\boldsymbol{\beta}\sim N(0,\tau^2 I)$。
- 共轭性使后验也是高斯，后验均值是「先验均值与数据估计的精度加权平均」。
- 后验众数（MAP）恰为 Ridge 解，惩罚强度 $\lambda=\sigma^2/\tau^2$——**先验方差越小、正则化越强**。
- 非共轭/复杂模型用 MCMC 或变分推断近似后验；预测用后验预测分布 $p(y^*\mid D)$。

### 3.3 参数与统计量含义

- 先验 $p(\boldsymbol{\beta})$：事前信念，控制正则化强度。
- 似然 $p(D\mid\boldsymbol{\beta})$：数据与参数的契合度。
- 后验 $p(\boldsymbol{\beta}\mid D)$：更新后的完整信念分布。
- 可信区间：后验的分位数区间；后验预测：带不确定性的预测。

### 3.4 关键假设（含违反后果）

| 假设 | 含义 | 违反后会怎样 | 如何粗查 |
| --- | --- | --- | --- |
| 先验合理 | 先验不与数据严重冲突 | 后验被误导（小样本尤甚） | 先验敏感性分析 |
| 似然/模型正确 | 数据生成模型对 | 后验有偏 | 后验预测检查 |
| 采样收敛 | MCMC 收敛 | 后验不可信 | R̂、迹图 |

## 4. 手把手算例

用共轭正态更新演示「后验 = 先验与数据的精度加权平均」。设某系数先验 $N(\text{均值}=0,\ \text{精度}=1)$，数据给出的估计为 $10$、精度 $4$（精度=1/方差）。

**一步步计算：**

- 后验精度 = 先验精度 + 数据精度 $=1+4=5$。
- 后验均值 = $\dfrac{\text{先验精度}\times\text{先验均值}+\text{数据精度}\times\text{数据估计}}{\text{后验精度}}=\dfrac{1\times0+4\times10}{5}=\dfrac{40}{5}=8$。
- 后验方差 $=1/5=0.2$，后验标准差 $\approx0.45$；95% 可信区间约 $8\pm0.88$。

**结论：** 数据把估计从「若无先验的 10」**收缩**到 8——被先验（均值 0）往回拉了一点，因为先验也贡献了信息。若数据精度更高（样本更大），如精度 40，后验均值 $=\dfrac{0+40\times10}{41}\approx9.8$，几乎完全由数据主导，先验影响可忽略。这精确演示了贝叶斯的核心机制：**后验是先验与数据的精度加权折中，数据越多越向数据靠拢**，而这份「收缩」正是正则化的来源。

## 5. 数据形式与输入输出

### 5.1 适合的数据形式

- 自变量类型：连续、分类均可。
- 因变量类型：连续（也可扩展到广义贝叶斯 GLM）。
- 数据结构：表格；层次结构可用分层先验。
- 是否适合高维数据：适合（先验即正则化）。
- 是否适合缺失较多数据：可在模型内联合建模缺失。
- 是否适合删失数据：可用贝叶斯生存模型扩展。
- 是否适合重复测量数据：适合（分层/多层模型）。

### 5.2 示例表格

| 量 | 先验 | 数据 | 后验 |
| --- | --- | --- | --- |
| 均值 | 0 | 10 | 8 |
| 精度 | 1 | 4 | 5 |

### 5.3 输入与产出

#### 输入

- 输入数据：协变量 + 结局 + 先验设定。
- 关键变量：先验分布、似然、超参数。
- 需要预处理的内容：先验选择、标准化、采样设置。

#### 产出

- 模型对象/统计结果：后验分布（样本/解析）。
- 参数估计：后验均值/众数与可信区间。
- 预测结果：后验预测分布。
- 不确定性指标：可信区间、后验预测区间。

## 6. 适用场景

- 适合：小样本、需先验、需完整不确定性、层次数据。
- 不适合：无合理先验且样本充足（OLS 更简）、计算资源受限的超大数据。
- 使用前需要特别检查的点：先验合理性与敏感性、采样收敛、后验预测检查。

## 7. 实现

### 7.1 Python

常用包：

- `scikit-learn`（BayesianRidge，快速近似）
- `pymc`（完整 MCMC）

```python
from sklearn.linear_model import BayesianRidge
import numpy as np

X = np.random.randn(50, 3); y = X @ [1.0, -2.0, 0.5] + np.random.randn(50)*0.5
m = BayesianRidge().fit(X, y)
mean, std = m.predict(X[:3], return_std=True)   # 预测 + 不确定性
print(mean, std)
```

### 7.2 R

常用包：

- `rstanarm`

```r
library(rstanarm)
fit <- stan_glm(y ~ x1 + x2, data = df,
                prior = normal(0, 2.5))   # 高斯先验
posterior_interval(fit, prob = 0.95)      # 可信区间
```

## 8. 结果如何解读

- 核心结果看什么：后验均值/中位数、可信区间、后验预测。
- 每个主要参数如何解读：可信区间可直接作概率陈述（「95% 概率在此」）。
- 临床或医学意义如何表达：报告后验估计 + 可信区间，说明先验来源。
- 常见误读：把可信区间当频率派 CI；用与数据冲突的强先验却不做敏感性分析。

## 9. 假设诊断与稳健性

- 先验敏感性：换几个合理先验，看后验是否稳健（小样本尤其重要）。
- 收敛诊断：MCMC 看 $\hat{R}\approx1$、有效样本量、迹图。
- 后验预测检查：模拟数据与观测对比，查模型误设。
- 与频率派对照：弱先验下后验应接近 OLS，可作 sanity check。

## 11. 优势、局限与常见坑

### 优势

- 完整不确定性、可直接概率解释。
- 自然纳入先验知识、内建正则化。
- 适合小样本与层次结构。

### 局限

- 先验选择带主观性。
- 计算量大（MCMC）。
- 门槛高于最小二乘。

### 常见坑

- 用强先验主导小样本却不报敏感性。
- 忽视 MCMC 收敛诊断。
- 把可信区间与置信区间混为一谈。

## 12. 与相近方法的区别

- 和 **OLS**：贝叶斯给分布、纳先验、正则化；OLS 给点估计。
- 和 **Ridge/Lasso**：正则化是贝叶斯 MAP 的特例。
- 和**频率派 CI**：可信区间可作概率陈述，解释不同。
- 如何选择：需先验/小样本/完整不确定性 → 贝叶斯；样本足、求简 → OLS/正则化。

## 13. 医学研究中的典型应用

- 小样本试验纳入既往证据作先验的效应估计。
- 需要预测区间指导临床决策的风险预测。
- 多中心/分层数据的层次贝叶斯建模。

## 14. 关键术语

- **先验（Prior）**：见数据前对参数的信念分布。
- **似然（Likelihood）**：给定参数时数据的概率。
- **后验（Posterior）**：结合先验与似然后的更新信念。
- **共轭（Conjugate）**：先验与后验同族，可解析更新。
- **可信区间（Credible Interval）**：后验分位区间，可作概率陈述。
- **收缩（Shrinkage）**：先验把估计往其均值拉，即正则化。
