# 多项式回归（Polynomial Regression）

## 1. 方法概览

### 1.1 一句话本质

多项式回归是线性回归最直接的「弯曲」扩展：把 $x,x^2,x^3,\dots$ 当作新的自变量一起塞进线性回归，就能拟合曲线——而模型对系数依然是线性的，仍用普通最小二乘求解。

### 1.2 定义

多项式回归通过引入自变量的幂次项，用线性回归框架拟合自变量与连续结局之间的非线性关系。它是「对参数线性、对自变量非线性」的模型。

### 1.3 它主要解决什么问题

- 研究问题：自变量与结局呈曲线关系（U 形、饱和、加速）时如何建模？
- 适用任务：连续结局的非线性趋势拟合与预测。
- 常见医学场景：年龄与某指标的 U 形关系、剂量-反应的非线性、生长曲线。

### 1.4 直觉与类比

直线只能表达「一路上升」或「一路下降」。但很多关系会拐弯——比如 BMI 与死亡风险是 U 形（太瘦太胖都不好）。多项式回归的窍门是：既然直线不够，就给回归多喂几样「原料」——不只喂 $x$，还喂 $x^2$（让它能弯一次）、$x^3$（能弯两次）。回归照旧是那台线性机器，但有了这些原料，就能画出曲线。

## 2. 核心思想与原理

### 2.1 它到底在解决什么根本困难

真实关系常是弯的，直线拟合会系统性地偏（在曲线两端要么高估要么低估）。根本困难是：**如何在不放弃线性回归那套成熟、可解的框架的前提下，拟合弯曲的关系？**

### 2.2 关键洞察

区分两种「线性」：模型对**参数** $\beta$ 线性（可用 OLS），不代表对**自变量** $x$ 线性。把 $x^2,x^3$ 视为新造的特征列，模型 $y=\beta_0+\beta_1 x+\beta_2 x^2+\dots$ 仍是关于 $\beta$ 的线性模型，正规方程照样解。代价是**次数的选择**是一场偏差-方差权衡：次数太低欠拟合（抓不住弯曲），太高过拟合（追噪声、两端剧烈摆动）。

### 2.3 与朴素/相邻做法的对比

- 相对**线性回归**：多项式能拟合曲线，是特征扩展后的线性回归。
- 相对**限制性立方样条（RCS）**：多项式是**全局**的，高次时两端（外推）会剧烈震荡（龙格现象）；样条**分段局部**、两端线性，更稳。
- 相对**局部加权回归（LOWESS）**：LOWESS 完全非参数、更灵活；多项式给出简洁的参数方程。

## 3. 数学形式

### 3.1 核心公式

$$
y_i=\beta_0+\beta_1 x_i+\beta_2 x_i^2+\dots+\beta_d x_i^d+\varepsilon_i
$$

这个式子在说：把自变量的各次幂作为独立的项加权求和；$d$ 是多项式次数，控制曲线能弯几次。

### 3.2 推导脉络

- 构造设计矩阵 $X=[1,x,x^2,\dots,x^d]$，问题化为普通多元线性回归。
- 用 OLS 闭式解 $\hat{\boldsymbol{\beta}}=(X^\top X)^{-1}X^\top y$。
- 次数 $d$ 用交叉验证或信息准则（AIC）选择；幂次高度共线，宜**中心化/标准化** $x$ 或用正交多项式改善数值稳定。
- 次数越高，模型越灵活但方差越大、边界外推越危险（龙格现象）。

### 3.3 参数与统计量含义

- $d$：多项式次数（关键超参数）。
- $\beta_j$：第 $j$ 次幂项系数；单个系数**无直观意义**，需看整条曲线。
- $R^2$/调整 $R^2$：拟合优度（高次易虚高）。
- 交叉验证误差：选次数的依据。

### 3.4 关键假设（含违反后果）

| 假设 | 含义 | 违反后会怎样 | 如何粗查 |
| --- | --- | --- | --- |
| 次数合适 | $d$ 匹配真实弯曲 | 欠拟合/过拟合 | 交叉验证 |
| 误差同方差正态 | 同线性回归 | 推断失真 | 残差图 |
| 不外推 | 预测在数据范围内 | 尾部剧烈发散 | 看 x 范围 |

## 4. 手把手算例

3 个点呈明显 U 形：$x=\{-1,0,1\}$，$y=\{2,1,2\}$。直线拟合会失败（斜率 0、$R^2=0$），二次多项式能完美拟合。

**一步步计算（拟合 $y=\beta_0+\beta_1 x+\beta_2 x^2$）：**

- 由对称性猜 $\beta_1=0$（数据关于 $x=0$ 对称）。
- 在 $x=0$：$y=\beta_0=1$，故 $\beta_0=1$。
- 在 $x=1$：$y=\beta_0+\beta_2=2\Rightarrow\beta_2=1$；在 $x=-1$ 校验：$1+0+1=2$ ✓。
- 拟合方程：$\hat{y}=1+x^2$，三点全中，$R^2=1$。

**结论：** 同样 3 个点，**直线只能给出一条水平线（完全抓不住 U 形）**，而加入 $x^2$ 后立刻完美拟合——这就是特征扩展的威力。但要警惕另一面：若把次数一路加到很高，模型会穿过每个点却在点与点之间、尤其在数据边界剧烈摆动（龙格现象），对新数据预测极差。所以次数要靠交叉验证节制，且**绝不向数据范围外外推**。

## 5. 数据形式与输入输出

### 5.1 适合的数据形式

- 自变量类型：连续（做幂次扩展）。
- 因变量类型：连续型。
- 数据结构：表格。
- 是否适合高维数据：多变量高次会维度爆炸，慎用。
- 是否适合缺失较多数据：需先处理。
- 是否适合删失数据：不适合。
- 是否适合重复测量数据：不适合；用混合模型。

### 5.2 示例表格

| x | y | 直线拟合 | 二次拟合 |
| --- | --- | --- | --- |
| −1 | 2 | 1.67 | 2 |
| 0 | 1 | 1.67 | 1 |
| 1 | 2 | 1.67 | 2 |

### 5.3 输入与产出

#### 输入

- 输入数据：连续自变量 + 结局。
- 关键变量：待扩展变量、次数 $d$。
- 需要预处理的内容：中心化/标准化、决定次数。

#### 产出

- 模型对象/统计结果：系数、$R^2$、CV 误差。
- 参数估计：$\hat{\boldsymbol{\beta}}$。
- 预测结果：拟合曲线与预测。
- 不确定性指标：预测区间（边界处很宽）。

## 6. 适用场景

- 适合：单变量、低次（2–3 次）的平滑弯曲关系。
- 不适合：高次外推、复杂局部形状（用样条/LOWESS）、多变量高次。
- 使用前需要特别检查的点：次数选择、中心化、不外推。

## 7. 实现

### 7.1 Python

常用包：

- `numpy`
- `scikit-learn`

```python
import numpy as np
x = np.array([-1, 0, 1]); y = np.array([2, 1, 2])
coef = np.polyfit(x, y, deg=2)      # [1, 0, 1] → 1 + 0*x + 1*x^2
print(coef)
print(np.polyval(coef, 0.5))        # 预测 x=0.5
```

### 7.2 R

常用包：

- `stats`

```r
x <- c(-1,0,1); y <- c(2,1,2)
fit <- lm(y ~ poly(x, 2, raw = TRUE))   # 或 poly(x,2) 正交多项式
summary(fit)
```

## 8. 结果如何解读

- 核心结果看什么：拟合曲线形状、交叉验证误差、是否过拟合。
- 每个主要参数如何解读：单系数无意义，看整条曲线与拐点位置。
- 临床或医学意义如何表达：描述关系形状（U 形、饱和）与最低/拐点位置。
- 常见误读：追求高 $R^2$ 而过拟合；解读单个高次系数；向数据外外推。

## 9. 假设诊断与稳健性

- 次数选择：用交叉验证/AIC，别只看 $R^2$。
- 共线性/数值：中心化或用正交多项式避免高次幂强共线。
- 边界与外推：预测尽量落在数据范围内，边界预测区间会很宽。
- 与样条对比：若尾部行为怪异，改用 RCS 更稳。

## 10. 推荐可视化

- 散点 + 不同次数拟合曲线对比。
- 训练/验证误差随次数变化（选次数）。
- 拟合曲线 + 预测区间（看边界发散）。

## 11. 优势、局限与常见坑

### 优势

- 简单，直接扩展线性回归。
- 给出简洁的参数化曲线。
- 低次时可解释、稳定。

### 局限

- 高次过拟合、边界剧烈摆动（龙格现象）。
- 外推极不可靠。
- 幂次强共线。

### 常见坑

- 用高次追高 $R^2$。
- 不中心化导致数值不稳。
- 向数据范围外预测。

## 12. 与相近方法的区别

- 和**线性回归**：多项式是加了幂次特征的线性回归。
- 和**限制性立方样条**：样条分段局部、两端线性，比全局多项式稳得多。
- 和**局部加权回归**：LOWESS 非参数、更灵活；多项式给参数方程。
- 如何选择：轻度弯曲 → 低次多项式；复杂/尾部敏感 → 样条；探索性 → LOWESS。

## 13. 医学研究中的典型应用

- 年龄/BMI 与结局的 U 形或饱和关系初探。
- 剂量-反应的非线性拟合。
- 生长/时间趋势的曲线建模。

## 14. 关键术语

- **次数（Degree, $d$）**：多项式最高幂，控制弯曲能力。
- **特征扩展**：用 $x$ 的幂次造新自变量。
- **对参数线性**：模型关于系数线性，可用 OLS。
- **龙格现象**：高次多项式在边界剧烈震荡。
- **过拟合**：过度贴合训练数据、泛化差。
