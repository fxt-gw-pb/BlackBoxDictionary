# 有序Logistic回归（Ordinal Logistic Regression）

## 1. 方法概览

### 1.1 一句话本质

有序 Logistic 回归（比例优势模型）对「累积概率」建 logit，并让所有切点共享同一组斜率——于是一个比值比就能概括「协变量把个体推向更高等级」的整体倾向。

### 1.2 定义

有序 Logistic 回归用于有序分类结局（如轻/中/重），通过对累积概率建立 logit 模型估计协变量效应。在比例优势假设下，各切点的斜率相同。

### 1.3 它主要解决什么问题

- 研究问题：协变量如何影响一个有自然顺序的分级结局？
- 适用任务：有序结局的回归、累积比值比估计、有序风险预测。
- 常见医学场景：疾病严重度（轻/中/重）、肿瘤分级、疼痛/功能量表、Likert 评分。

### 1.4 直觉与类比

结局像楼层：1 楼、2 楼、3 楼有高低之分。若把它当连续数字回归，等于假设「1 到 2 楼」和「2 到 3 楼」的间距相等——未必成立；若当无序多类，又浪费了楼层的高低顺序。有序 Logistic 的做法是：在每个楼层之间设一道「门槛」，看协变量把人往楼上推的整体倾向，并假设它对每道门槛的作用相同，于是用一个数（OR）概括。

## 2. 核心思想与原理

### 2.1 它到底在解决什么根本困难

有序结局卡在「连续」与「无序多类」之间：当连续回归会假设等间距、且可能预测出界；当多项 Logistic 会浪费顺序信息、参数翻倍。根本困难是：**如何既利用类别的顺序、又不假设等间距，且尽量省参数？**

### 2.2 关键洞察

把有序结局在每个「切点」处二分——「$\le j$」对「$\gt j$」，得到 $J-1$ 条**累积 logit**。比例优势假设让这些累积 logit **平行**（共享同一斜率 $\boldsymbol{\beta}$，只有截距 $\alpha_j$ 不同）。于是一个协变量对「向更高等级移动」的效应，可以用单一累积比值比 $\exp(\beta)$ 概括，既保留顺序、又极省参数。

### 2.3 与朴素/相邻做法的对比

- 相对**把有序当连续做线性回归**：不假设等间距、不会预测出界。
- 相对**多项 Logistic**：利用顺序、参数更少（多项要为每个非参照类各估一组系数）。
- 相对**二分类 Logistic**：把多个切点「打包」成一个模型，避免任意二分损失信息。

## 3. 数学形式

### 3.1 核心公式

$$
\operatorname{logit}\big(P(Y\le j\mid x)\big)=\alpha_j-\boldsymbol{\beta}^\top x,\qquad j=1,\dots,J-1
$$

这个式子在说：对「结局不超过第 $j$ 级」的累积概率建 logit；每个切点有自己的截距 $\alpha_j$（随 $j$ 递增），但斜率 $\boldsymbol{\beta}$ 所有切点共享。负号约定使 $\beta\gt0$ 表示协变量增大更高等级的概率。

### 3.2 推导脉络

- 对每个切点 $j$ 定义累积 logit；共 $J-1$ 条。
- 比例优势假设：各切点斜率相同 → 平行的累积 logit 线，仅截距不同。
- 用最大似然估计 $\alpha_j$ 与 $\boldsymbol{\beta}$；$\exp(\beta)$ 是「跨任一切点」的累积 OR。
- 若平行假设不成立，可放宽为偏比例优势模型或改用多项 Logistic。

### 3.3 参数与统计量含义

- $\alpha_j$：第 $j$ 切点截距，随 $j$ 单调递增。
- $\boldsymbol{\beta}$：共享斜率；$\exp(\beta_k)$ = 累积比值比。
- $J$：有序类别数。
- 比例优势检验：验证平行假设。

### 3.4 关键假设（含违反后果）

| 假设 | 含义 | 违反后会怎样 | 如何粗查 |
| --- | --- | --- | --- |
| 比例优势 | 各切点斜率相同 | 单一 OR 有偏 | 平行线检验、逐切点拟合 |
| 结局有序 | 类别有自然顺序 | 用错模型 | 领域判断 |
| 观测独立 | 无相关结构 | SE 失真 | 看设计 |

## 4. 手把手算例

3 个有序等级（1<2<3）的模型，切点截距 $\alpha_1=-1$、$\alpha_2=1$，某协变量系数 $\beta=0.5$。对协变量取值 $x=2$ 的个体，算各等级概率。

**一步步计算（$\operatorname{logit}(P(Y\le j))=\alpha_j-\beta x$）：**

- 累积 1：$\operatorname{logit}(P(Y\le1))=\alpha_1-\beta x=-1-0.5\times2=-2$，故 $P(Y\le1)=\dfrac{1}{1+e^{2}}=0.12$。
- 累积 2：$\operatorname{logit}(P(Y\le2))=\alpha_2-\beta x=1-1=0$，故 $P(Y\le2)=\dfrac{1}{1+e^{0}}=0.50$。
- 分级概率：$P(Y=1)=0.12$；$P(Y=2)=0.50-0.12=0.38$；$P(Y=3)=1-0.50=0.50$。
- 累积比值比：$\exp(\beta)=\exp(0.5)\approx1.65$。

**结论：** 这名个体最可能落在等级 3（概率 0.50）。系数的累积 OR=1.65 意味着：协变量每升 1 单位，「落到更高等级」相对「更低等级」的比值增大约 65%，**且这个倍数对每一道切点都一样**（这正是比例优势的含义）。若逐切点单独拟合发现斜率相差很大，就说明平行假设不成立，应改用偏比例优势或多项模型。

## 5. 数据形式与输入输出

### 5.1 适合的数据形式

- 自变量类型：连续、二分类、多分类均可。
- 因变量类型：有序分类（≥3 个有序等级）。
- 数据结构：每行一个体，含有序结局与协变量。
- 是否适合高维数据：需配合正则化。
- 是否适合缺失较多数据：需先处理。
- 是否适合删失数据：不适合。
- 是否适合重复测量数据：需有序混合模型扩展。

### 5.2 示例表格

| 病例 | 协变量 x | 疗效等级 |
| --- | --- | --- |
| 1 | 2 | 好转 |
| 2 | 0 | 稳定 |
| 3 | 3 | 治愈 |

### 5.3 输入与产出

#### 输入

- 输入数据：有序结局（有序因子）+ 协变量。
- 关键变量：结局等级顺序、协变量矩阵。
- 需要预处理的内容：设定有序因子、编码、比例优势检验。

#### 产出

- 模型对象/统计结果：切点截距、系数、累积 OR。
- 参数估计：$\alpha_j$、$\boldsymbol{\beta}$。
- 预测结果：各等级概率、累积概率。
- 不确定性指标：系数标准误、OR 区间、比例优势检验。

## 6. 适用场景

- 适合：有序等级结局、各切点效应近似一致。
- 不适合：无序多分类（用多项 Logistic）、比例优势明显不成立。
- 使用前需要特别检查的点：比例优势假设、等级是否真有序、各等级样本是否充足。

## 7. 实现

### 7.1 Python

常用包：

- `statsmodels`

```python
import pandas as pd
from statsmodels.miscmodels.ordinal_model import OrderedModel

y = pd.Categorical(df["grade"],
                   categories=["恶化","稳定","好转","治愈"], ordered=True)
mod = OrderedModel(y.codes, df[["age","treat"]], distr="logit")
res = mod.fit(method="bfgs", disp=False)
print(res.summary())
```

### 7.2 R

常用包：

- `MASS`

```r
library(MASS)
df$grade <- factor(df$grade,
                   levels = c("恶化","稳定","好转","治愈"), ordered = TRUE)
fit <- polr(grade ~ age + treat, data = df, Hess = TRUE)
exp(cbind(OR = coef(fit), confint(fit)))   # 累积 OR
```

## 8. 结果如何解读

- 核心结果看什么：各协变量的累积 OR、方向与区间；比例优势检验。
- 每个主要参数如何解读：OR>1 表示协变量升高使结局落到更高等级的比值增大（对所有切点相同）。
- 临床或医学意义如何表达：如「治疗使达到更好疗效等级的比值提高约 65%」。
- 常见误读：把有序 OR 当成某一具体等级的风险；忽视比例优势不成立。

## 9. 假设诊断与稳健性

- 比例优势：正式的平行线检验（如 Brant 检验），或逐切点二分拟合看斜率是否接近。
- 违反处理：偏比例优势模型（部分变量放松平行）、多项 Logistic、或连续潜变量模型。
- 稀疏等级：某等级样本太少会使估计不稳，可合并相邻等级。
- 方向核对：确认等级编码顺序，避免 OR 方向反转。

## 10. 推荐可视化

- 各协变量 OR 森林图。
- 不同协变量水平下各等级概率的堆叠条形图。
- 比例优势诊断（逐切点斜率对比）。

## 11. 优势、局限与常见坑

### 优势

- 利用顺序信息，比多项 Logistic 省参数。
- 单一 OR 概括效应，易沟通。
- 软件成熟。

### 局限

- 依赖比例优势假设。
- 假设不成立时结论有偏。
- 等级极不平衡时估计不稳。

### 常见坑

- 不检验比例优势就报单一 OR。
- 把有序结局当连续做线性回归。
- 等级顺序编码错误致方向反转。

## 12. 与相近方法的区别

- 和**二分类 Logistic**：后者二分类，前者有序多分类。
- 和**多项 Logistic**：多项不假设顺序、参数更多；有序利用顺序更简洁。
- 和 **GLM**：属累积 link 的 GLM 扩展。
- 如何选择：结局有序且平行成立 → 有序 Logistic；无序或平行不成立 → 多项 Logistic。

## 13. 医学研究中的典型应用

- 疾病严重度分级的危险因素分析。
- 治疗对功能/疼痛量表等级的效应。
- 肿瘤分级与分子标志物关联建模。

## 14. 关键术语

- **累积 logit**：对「$\le j$」累积概率的 logit。
- **比例优势（平行线）假设**：各切点共享斜率。
- **切点截距（$\alpha_j$）**：每个等级门槛的基线。
- **累积比值比**：跨切点的 $\exp(\beta)$。
- **偏比例优势模型**：放松部分变量平行假设的推广。
