# 置信区间（Confidence Interval）

## 1. 方法概览

### 1.1 一句话本质

置信区间给点估计配上一段范围，回答「这个数字有多可信」；95% 指的是**方法的长期表现**——反复抽样构造的区间里，约 95% 会罩住真值。

### 1.2 定义

置信区间是围绕点估计构造的一段取值范围，用来表达对未知参数的估计不确定性。它把「我们估计的效应有多大」和「这个估计有多精确」一次说清，是比 p 值信息量更高的报告方式。

### 1.3 它主要解决什么问题

- 研究问题：一个估计值有多可信？真值大概落在什么范围？
- 适用任务：均值/比例/风险比等参数的区间估计、效应量不确定性表达、等效/非劣判断。
- 常见医学场景：疗效差异的区间、患病率区间、OR/HR/RR 的 95% CI 报告。

### 1.4 直觉与类比

点估计像用一根针指出「就是这里」，但抽样有随机性，针尖不可能刚好落在真值上。置信区间改用「一段渔网」去罩：网越窄越精确。关键要理解——这段网是随数据变的，「95%」说的是「用这套撒网法，长期有 95% 的网能罩住鱼」，而不是「这条鱼有 95% 概率在这张网里」。

## 2. 核心思想与原理

### 2.1 它到底在解决什么根本困难

一个点估计（如均值 130）本身**不带任何不确定性信息**——它可能非常稳，也可能因样本小而摇摆不定。根本困难是：**怎样把「估计的抖动幅度」量化并附在结论上？** 否则读者无法判断该不该相信这个数字。

### 2.2 关键洞察

估计量有一个「抽样分布」——想象把研究重复无数次，每次的估计会围绕真值散开，散开的宽度就是**标准误（SE）**。已知这个分布，就能反过来在点估计两侧划出一段范围，使它以指定概率罩住真值。区间宽度 $\propto \mathrm{SE}\propto 1/\sqrt{n}$，所以样本翻 4 倍，区间才窄一半。

### 2.3 与朴素/相邻做法的对比

- 相对**只报点估计**：CI 额外给出精度，避免「看起来精确、其实很虚」。
- 相对 **p 值/假设检验**：CI 同时给方向、大小、精度；「CI 是否含无效值」还蕴含了检验结论，信息更全。
- 相对**贝叶斯可信区间**：贝叶斯区间可直接说「真值有 95% 概率在此」；频率派 CI 不能这样说，解释框架不同。

## 3. 数学形式

### 3.1 核心公式

大样本 Wald 型区间与小样本正态均值的 $t$ 区间：

$$
\hat{\theta}\pm z_{1-\alpha/2}\,\mathrm{SE}(\hat{\theta}),\qquad
\bar{x}\pm t_{1-\alpha/2,\,n-1}\,\frac{s}{\sqrt{n}}
$$

这个式子在说：区间 = 点估计 ± 「临界值 × 标准误」；临界值决定信心水平（95% 对应约 1.96 或对应的 $t$ 值），标准误决定宽度。

### 3.2 推导脉络

- 以正态均值为例：$\dfrac{\bar{x}-\mu}{s/\sqrt{n}}\sim t_{n-1}$。
- 于是 $P\!\left(-t_{1-\alpha/2}\le \dfrac{\bar{x}-\mu}{s/\sqrt{n}}\le t_{1-\alpha/2}\right)=1-\alpha$。
- 把不等式里的 $\mu$ 解出来，得到 $\bar{x}\pm t_{1-\alpha/2}\,s/\sqrt{n}$ 覆盖 $\mu$ 的概率为 $1-\alpha$——注意「随机的是区间端点，不是 $\mu$」。
- 比例、率、OR/HR 等换成对应的估计量与标准误；边界附近（比例接近 0/1）Wald 近似失效，改用 Wilson 或精确区间。

### 3.3 参数与统计量含义

- $\hat{\theta}$：点估计（均值、比例、系数等）。
- $\mathrm{SE}(\hat{\theta})$：标准误，估计量的抽样波动。
- $z_{1-\alpha/2}$、$t_{1-\alpha/2,n-1}$：置信水平对应的临界值（95% 时 $z=1.96$）。
- $1-\alpha$：置信水平（如 0.95）；区间宽度反映精度。

### 3.4 关键假设（含违反后果）

| 假设 | 含义 | 违反后会怎样 | 如何粗查 |
| --- | --- | --- | --- |
| 抽样分布近似 | 估计量近正态或用 $t$ | 小样本/偏态时覆盖率偏离 | QQ 图、改用 bootstrap |
| 独立观测 | 观测互不影响 | SE 低估、区间过窄 | 看设计；聚类用稳健 SE |
| 边界近似有效 | 参数不贴近边界 | 比例近 0/1 时 Wald 失效 | 改用 Wilson/精确区间 |

## 4. 手把手算例

某组 16 名患者收缩压均值 $\bar{x}=130$，样本标准差 $s=12$，求 95% 置信区间。

**一步步计算：**

- 标准误：$\mathrm{SE}=s/\sqrt{n}=12/\sqrt{16}=12/4=3$。
- 临界值：$t_{0.975,\,15}=2.131$（自由度 $n-1=15$）。
- 半宽（误差幅度）：$2.131\times 3=6.39$。
- 区间：$130\pm 6.39=(123.6,\ 136.4)$。

**结论：** 我们对总体均值的最佳猜测是 130，但考虑抽样波动，它合理地落在约 124 到 136 之间。若把样本量增到 64（4 倍），$\mathrm{SE}$ 降到 1.5、半宽降到约 3.2，区间收窄一半——**这直观展示了「精度随 $\sqrt{n}$ 提升」**。注意 95% 说的是方法长期覆盖率，而非「真值有 95% 概率在 124–136 内」。

## 5. 数据形式与输入输出

### 5.1 适合的数据形式

- 自变量类型：视所估参数而定。
- 因变量类型：连续、二分类、计数、时间到事件均可（各有对应区间）。
- 数据结构：任意可估计参数的数据。
- 是否适合高维数据：单参数区间；多参数需联合置信域或多重校正。
- 是否适合缺失较多数据：需先处理缺失并传播不确定性。
- 是否适合删失数据：生存参数有专门区间（如 KM 的对数变换区间）。
- 是否适合重复测量数据：需用稳健/聚类标准误。

### 5.2 示例表格

| 参数 | 点估计 | 95% CI |
| --- | --- | --- |
| 均值 | 130 | 123.6 ~ 136.4 |
| 均值差 | −8.0 | −11.2 ~ −4.8 |
| 风险比 HR | 0.72 | 0.58 ~ 0.90 |

### 5.3 输入与产出

#### 输入

- 输入数据：样本数据，或点估计 + 标准误。
- 关键变量：估计量、标准误、置信水平。
- 需要预处理的内容：选择合适的区间方法（Wald / t / Wilson / bootstrap）。

#### 产出

- 模型对象/统计结果：区间上下限。
- 参数估计：点估计。
- 预测结果：无（区间是推断，不是预测；预测区间是另一概念）。
- 不确定性指标：区间宽度本身即不确定性度量。

## 6. 适用场景

- 适合：几乎所有效应量的报告，替代或补充 p 值。
- 不适合：把区间当作真值的概率分布来解读、边界近似失效却仍用 Wald。
- 使用前需要特别检查的点：抽样分布近似是否成立、比例/率是否需专用区间、是否需多重比较调整。

## 7. 实现

### 7.1 Python

常用包：

- `scipy`
- `statsmodels`

```python
import numpy as np
from scipy import stats

x = np.random.normal(130, 12, 16)
m, se = x.mean(), stats.sem(x)          # sem = s/sqrt(n)
lo, hi = stats.t.interval(0.95, len(x) - 1, loc=m, scale=se)
print(f"均值={m:.1f}, 95% CI=({lo:.1f}, {hi:.1f})")

from statsmodels.stats.proportion import proportion_confint
print(proportion_confint(34, 100, method="wilson"))   # 比例用 Wilson
```

### 7.2 R

常用包：

- `stats`

```r
x <- rnorm(16, 130, 12)
t.test(x)$conf.int            # 均值 95% CI
prop.test(34, 100)$conf.int   # 比例区间(含连续性校正)
```

## 8. 结果如何解读

- 核心结果看什么：区间是否包含无效值（均差 0、比值 1）、区间宽度。
- 每个主要参数如何解读：不含无效值≈在该水平显著；宽区间提示证据不足。
- 临床或医学意义如何表达：报告区间两端是否都具临床意义（如下限是否仍有效）。
- 常见误读：说「真值有 95% 概率落在此区间」（频率派下不成立）；把不显著等同于无效应。

## 9. 假设诊断与稳健性

- 分布近似：小样本或强偏态时，正态/ $t$ 近似可能不准，改用 bootstrap 区间。
- 比例/率边界：接近 0 或 1 时 Wald 区间会越界或过窄，用 Wilson、Clopper-Pearson。
- 相关/聚类数据：普通 SE 会低估，用稳健或聚类标准误，否则区间过窄。
- 多重比较：同时报很多区间时，单个 95% 的「同时覆盖率」会下降，需校正。

## 10. 推荐可视化

- 森林图（多个效应量的点估计与区间）。
- 误差棒图。
- 区间宽度随样本量变化的示意。

## 11. 优势、局限与常见坑

### 优势

- 同时传达效应方向、大小与精度，信息量高于 p 值。
- 支持等效/非劣性判断。
- 各类参数都有对应构造方法。

### 局限

- 依赖抽样分布近似或重抽样。
- 频率派解释常被误读为概率陈述。
- 边界参数需专用方法。

### 常见坑

- 比例接近 0 或 1 仍用 Wald 区间。
- 把 95% CI 解读为真值概率。
- 多重比较下不调整。

## 12. 与相近方法的区别

- 和 **p 值/假设检验**：CI 给效应大小与精度，检验只给是否显著，二者互补。
- 和 **Bootstrap**：分布近似难成立时，用 bootstrap 构造区间。
- 和**贝叶斯可信区间**：贝叶斯区间可直接作概率陈述，解释不同。
- 如何选择：能算解析 SE 且近似成立就用经典 CI；否则 bootstrap；想要概率解释用贝叶斯。

## 13. 医学研究中的典型应用

- 疗效差异、OR/RR/HR 的 95% CI 报告（期刊规范要求）。
- 患病率、敏感度/特异度的区间估计。
- 等效性与非劣效性试验的区间判定。

## 14. 关键术语

- **点估计（Point Estimate）**：对参数的单一最佳猜测，如样本均值。
- **标准误（Standard Error, SE）**：估计量的抽样波动，决定区间宽度。
- **置信水平（Confidence Level, $1-\alpha$）**：方法长期覆盖真值的比例，如 95%。
- **覆盖率（Coverage）**：区间真正罩住参数的实际比例。
- **误差幅度（Margin of Error）**：临界值 × 标准误，即区间半宽。
- **Wilson 区间**：比例参数比 Wald 更可靠的区间构造。
