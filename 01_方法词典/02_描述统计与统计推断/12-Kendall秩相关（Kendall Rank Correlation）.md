# Kendall秩相关（Kendall Rank Correlation）

## 1. 方法概览

### 1.1 一句话本质

Kendall $\tau$ 直接数「排序一致的对」减去「排序矛盾的对」，再归一化：它回答「随机挑两个个体，两个变量给出相同大小顺序的概率，比相反顺序高多少」。

### 1.2 定义

Kendall 秩相关系数 $\tau$ 基于所有观测对的「和谐/不和谐」计数，度量两变量单调关联的强弱与方向，取值 $[-1,1]$，对结点与小样本更稳健。

### 1.3 它主要解决什么问题

- 研究问题：两个变量的大小排序有多一致？
- 适用任务：单调关联量化（尤其小样本、多结点、有序数据）。
- 常见医学场景：两位评分者的等级排序一致性、有序分级与结局、样本量小的关联分析。

### 1.4 直觉与类比

任取两名患者，比较他们在两个指标上的高低。如果「指标 A 更高的那位，指标 B 也更高」，这一对就「和谐」（顺序一致）；反之为「不和谐」。把所有配对都这样投票，和谐票占多数则 $\tau\gt 0$。$\tau$ 就是「和谐 − 不和谐」占总对数的净比例，非常直观。

## 2. 核心思想与原理

### 2.1 它到底在解决什么根本困难

Spearman 用「秩差平方」，其数值不易直接翻译成概率意义；小样本、大量结点时秩差公式也不稳。根本困难是：**能不能用一个基于「排序一致性」、可直接解释为概率、且对结点稳健的度量？**

### 2.2 关键洞察

把「相关」还原到最基本的比较单位——**每一对观测**。对每对判断顺序是否一致（和谐 C）或相反（不和谐 D），$\tau=\dfrac{C-D}{\text{总对数}}$ 直接等于「和谐概率 − 不和谐概率」。这个「计数配对」的定义天然可解释、对异常值免疫（只看顺序不看数值），并有针对结点的修正版（$\tau_b$）。

### 2.3 与朴素/相邻做法的对比

- 相对 **Spearman**：两者都测单调；Kendall 数值更小但更易解释（概率含义）、结点/小样本更稳，Spearman 更常见、计算更快。
- 相对 **Pearson**：Kendall 抓排序一致、稳健；Pearson 抓线性、敏感。
- 经验换算：无结点时约 $\tau\approx\frac{2}{3}\rho_{\text{Spearman}}$（故同数据 $\tau$ 通常小于 $\rho$）。

## 3. 数学形式

### 3.1 核心公式

$$
\tau=\frac{C-D}{\binom{n}{2}}=\frac{C-D}{n(n-1)/2}
$$

其中 $C$ 是和谐对数、$D$ 是不和谐对数。这个式子在说：把所有 $\binom{n}{2}$ 对里「顺序一致减去顺序相反」的净数，除以总对数，得到 $[-1,1]$ 的净一致率。

### 3.2 推导脉络

- 对每对 $(i,j)$，若 $(x_i-x_j)(y_i-y_j)\gt 0$ 记为和谐、$\lt 0$ 记为不和谐。
- $\tau_a=\dfrac{C-D}{n(n-1)/2}$；这直接是「和谐概率 − 不和谐概率」的样本估计。
- 有结点时分母改用几何平均校正，得 $\tau_b$（避免结点稀释相关）。
- 大样本下 $\tau$ 近似正态，方差有解析式，用于检验与置信区间。

### 3.3 参数与统计量含义

- $C,D$：和谐、不和谐对数。
- $\tau$（$\tau_a$/$\tau_b$）：净一致率，方向 + 强度。
- $\binom{n}{2}$：总配对数。
- p 值：排序一致性是否显著。

### 3.4 关键假设（含违反后果）

| 假设 | 含义 | 违反后会怎样 | 如何粗查 |
| --- | --- | --- | --- |
| 单调 | 关系单调 | 非单调被漏 | 看散点图 |
| 可排序 | 变量可比大小 | 用 $\tau_b$ 处理结点 | 看重复值 |
| 观测独立 | 成对独立 | p 值失真 | 看设计 |

## 4. 手把手算例

沿用前两张卡的数据：$x=\{1,2,3,4,5\}$，$y=\{2,3,5,4,6\}$。因 $x$ 已递增，只需看每对里 $y$ 是否也递增（和谐）。

**一步步计算**（共 $\binom{5}{2}=10$ 对，按 $x$ 顺序，$y$ 序列为 $2,3,5,4,6$）：

| 对 (i,j) | $y_i,y_j$ | 一致? |
| --- | --- | --- |
| (1,2) | 2,3 | 和谐 |
| (1,3) | 2,5 | 和谐 |
| (1,4) | 2,4 | 和谐 |
| (1,5) | 2,6 | 和谐 |
| (2,3) | 3,5 | 和谐 |
| (2,4) | 3,4 | 和谐 |
| (2,5) | 3,6 | 和谐 |
| (3,4) | 5,4 | **不和谐** |
| (3,5) | 5,6 | 和谐 |
| (4,5) | 4,6 | 和谐 |

- 和谐 $C=9$，不和谐 $D=1$。
- $\tau=\dfrac{C-D}{10}=\dfrac{9-1}{10}=0.80$。

**结论：** $\tau=0.80$ 意味着「随机取两名个体，两指标顺序一致的比例比相反的高 80 个百分点」。唯一的不和谐对来自 $y$ 序列里 $5$ 与 $4$ 的一次逆序。对照同数据的 Pearson $r=0.90$、Spearman $\rho=0.90$：$\tau$ 数值更小是**正常**的（量纲不同），但它给出了最直观的概率解释。

## 5. 数据形式与输入输出

### 5.1 适合的数据形式

- 自变量类型：两个连续或有序变量。
- 因变量类型：不区分。
- 数据结构：成对观测。
- 是否适合高维数据：配对计数计算量随 $n^2$ 增长，超大样本较慢。
- 是否适合缺失较多数据：需成对完整。
- 是否适合删失数据：不适合。
- 是否适合重复测量数据：需考虑组内相关。

### 5.2 示例表格

| 对 | 一致 | 类别 |
| --- | --- | --- |
| (3,4) | 5 vs 4 逆序 | 不和谐 |
| 其余 9 对 | 同序 | 和谐 |

### 5.3 输入与产出

#### 输入

- 输入数据：两列成对变量。
- 关键变量：两变量。
- 需要预处理的内容：处理结点（选 $\tau_b$）、缺失。

#### 产出

- 模型对象/统计结果：$\tau$、p 值。
- 参数估计：$\tau$（净一致率）。
- 预测结果：无。
- 不确定性指标：$\tau$ 的置信区间（可选）。

## 6. 适用场景

- 适合：单调关联，尤其小样本、多结点、有序数据、需概率解释时。
- 不适合：非单调关系、需要线性解释力。
- 使用前需要特别检查的点：散点图看单调性、结点是否需 $\tau_b$。

## 7. 实现

### 7.1 Python

常用包:

- `scipy`

```python
from scipy import stats

x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 4, 6]
tau, p = stats.kendalltau(x, y)
print(f"tau={tau:.2f}, p={p:.3f}")     # tau=0.80
```

### 7.2 R

常用包:

- `stats`

```r
x <- c(1,2,3,4,5); y <- c(2,3,5,4,6)
cor.test(x, y, method = "kendall")     # 默认 tau_b
```

## 8. 结果如何解读

- 核心结果看什么：$\tau$ 的符号与大小、p 值。
- 每个主要参数如何解读：$\tau$ 可读成「净一致率」；与 Spearman 换算约 $\tau\approx\frac23\rho$。
- 临床或医学意义如何表达：报告排序一致的概率优势，强调非因果。
- 常见误读：直接把 $\tau$ 与 $r$/$\rho$ 比大小（量纲不同）；忽略非单调关系。

## 9. 假设诊断与稳健性

- 单调性：散点图确认。
- 结点：大量相等值用 $\tau_b$（几何平均校正）。
- 稳健性：只看顺序，对异常值免疫。
- 小样本：比 Spearman 更稳，有精确分布。

## 10. 推荐可视化

- 散点图（原值）。
- 和谐/不和谐对的可视化。

## 11. 优势、局限与常见坑

### 优势

- 直观的概率解释（净一致率）。
- 对结点、小样本更稳健。
- 对异常值免疫。

### 局限

- 数值小于 Pearson/Spearman，易被误读。
- 大样本计算较慢（$O(n^2)$，虽有 $O(n\log n)$ 算法）。
- 只抓单调关系。

### 常见坑

- 与 $r$/$\rho$ 直接比大小。
- 多结点仍用 $\tau_a$。
- 忽略非单调关系。

## 12. 与相近方法的区别

- 和 **Spearman**：都测单调；Kendall 更易解释与抗结点，Spearman 更常见更快。
- 和 **Pearson**：Kendall 排序一致、稳健；Pearson 线性、敏感。
- 和**一致性系数**：评分者一致性另有 ICC/加权 Kappa。
- 如何选择：小样本/多结点/要概率解释 → Kendall；一般单调 → Spearman；线性干净 → Pearson。

## 13. 医学研究中的典型应用

- 两位评分者对病例等级排序的一致性。
- 有序分级与连续指标的关联。
- 小样本或多并列等级的关联分析。

## 14. 关键术语

- **和谐对（Concordant）**：两变量给出相同大小顺序的观测对。
- **不和谐对（Discordant）**：两变量顺序相反的观测对。
- **$\tau$（tau）**：和谐减不和谐的净比例。
- **$\tau_b$**：处理结点的 Kendall 修正版。
- **净一致率**：$\tau$ 的概率解释。
