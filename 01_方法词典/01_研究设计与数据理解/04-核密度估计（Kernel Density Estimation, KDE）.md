# 核密度估计（Kernel Density Estimation, KDE）

## 1. 方法概览

### 1.1 一句话本质

核密度估计给每个观测放上一个小小的、光滑的「山包」（核），再把所有山包叠加平均，得到一条连续平滑的密度曲线——本质是把直方图的硬台阶抹圆。

### 1.2 定义

KDE 是一种非参数密度估计方法，用核函数在每个数据点周围铺一个平滑权重，叠加后估计出连续的概率密度曲线，无需假设分布属于某个参数族。

### 1.3 它主要解决什么问题

- 研究问题：数据背后的概率密度长什么样（是否单峰、偏态、多峰）？
- 适用任务：平滑的分布可视化、多组分布叠加比较、密度形状探索。
- 常见医学场景：叠加比较多组生物标志物分布、展示患者某指标的平滑分布、混合亚群的多峰识别。

### 1.4 直觉与类比

直方图把数据硬塞进格子，边界一刀切、形状全靠格子。KDE 换个思路：每个观测不再是「掉进某个格子」，而是「在自己周围晕开一小片概率」（像墨滴在纸上晕开）。所有墨滴叠起来，就是一条光滑的浓淡曲线——数据密集处颜色深（密度高），稀疏处颜色浅。

## 2. 核心思想与原理

### 2.1 它到底在解决什么根本困难

直方图有两个硬伤：**箱边界是人为的**（同一点落在左箱还是右箱，结果不同），以及**输出是不连续的台阶**。根本困难是：**如何得到一条既平滑、又不依赖任意箱边界的密度曲线？** KDE 用「以每个数据点为中心」代替「固定格子」，从根上消除了箱边界问题。

### 2.2 关键洞察

把直方图的「计数」换成「加权求和」：离查询点 $x$ 越近的观测，贡献越大；越远，贡献越小。这个「远近权重」由核函数 $K$ 给出。关键旋钮是**带宽 $h$**（山包的胖瘦）：$h$ 大 → 曲线过度平滑、抹掉真峰（偏差大）；$h$ 小 → 曲线毛刺、把噪声当峰（方差大）。经验上**带宽比核函数的选择重要得多**。

### 2.3 与朴素/相邻做法的对比

- 相对**直方图**：KDE 连续、无箱边界依赖，但仍需选带宽（对应箱宽）。
- 相对**参数拟合**（如假设正态）：KDE 不强加分布形状，能揭示多峰等参数模型看不到的结构。
- 代价：边界处有偏差（bounded 变量如年龄 ≥0 会「漏」概率到负区）；高维时维度灾难严重。

## 3. 数学形式

### 3.1 核心公式

$$
\hat{f}_h(x)=\frac{1}{n\,h}\sum_{i=1}^{n} K\!\left(\frac{x-x_i}{h}\right)
$$

这个式子在说：$x$ 处的密度 = 把每个观测按「它离 $x$ 有多近」加权（权重由核 $K$ 给），求和再标准化。常用高斯核 $K(u)=\frac{1}{\sqrt{2\pi}}e^{-u^2/2}$。

### 3.2 推导脉络

- 从密度定义出发：$f(x)\approx\dfrac{P(x-h\le X\le x+h)}{2h}$，用落入窗口的比例估计——这正是「移动直方图」。
- 把「硬窗口」（矩形核，落入算 1、否则算 0）换成**平滑核**（如高斯），让贡献随距离连续衰减，得到平滑曲线。
- 偏差-方差分析给出最优带宽约 $h^\star\propto n^{-1/5}$。**Silverman 经验法则**：$h\approx 0.9\,\min(s,\ \mathrm{IQR}/1.34)\,n^{-1/5}$，是实用默认值。
- 带宽的自动选择还可用交叉验证或 plug-in 方法。

### 3.3 参数与统计量含义

- $h$（带宽）：核的宽度，控制平滑程度，最关键的参数。
- $K$：核函数（高斯、Epanechnikov 等），决定权重随距离衰减的形状，影响较小。
- $\hat{f}_h(x)$：$x$ 处的密度估计，曲线下面积为 1。
- $n^{-1/5}$：最优带宽随样本量缩小的速度（比直方图的 $n^{-1/3}$ 慢，反映 KDE 更高效）。

### 3.4 关键假设（含违反后果）

| 假设 | 含义 | 违反后会怎样 | 如何粗查 |
| --- | --- | --- | --- |
| 密度平滑 | 真密度连续光滑 | 尖峰/跳变处失真 | 结合直方图判断 |
| 无硬边界 | 变量取值无界 | 边界附近密度被低估/漏出 | 用有界核或变换 |
| 带宽合适 | $h$ 匹配数据 | 过平滑抹峰 / 欠平滑毛刺 | 试多个 $h$ |

## 4. 手把手算例

用 3 个观测 $\{2,3,5\}$、高斯核、带宽 $h=1$，估计 $x=4$ 处的密度。

**一步步计算**（$\hat{f}(4)=\dfrac{1}{n h}\sum_i \varphi\!\big(\tfrac{4-x_i}{h}\big)$，$\varphi$ 为标准正态密度）：

- 观测 2：$u=(4-2)/1=2$，$\varphi(2)=0.054$。
- 观测 3：$u=(4-3)/1=1$，$\varphi(1)=0.242$。
- 观测 5：$u=(4-5)/1=-1$，$\varphi(-1)=0.242$。
- 求和：$0.054+0.242+0.242=0.538$。
- 标准化：$\hat{f}(4)=\dfrac{0.538}{3\times 1}=0.179$。

**结论：** $x=4$ 处的密度约 0.18，主要由近邻点 3、5 贡献（各 0.242），远处的 2 只贡献 0.054——**「近处影响大、远处影响小」正是核方法的核心机制**。若把带宽 $h$ 调大到 2，三个点的贡献会更平均、曲线更平；调小到 0.5，则只有几乎重合的点才互相支撑、曲线更尖锐。

## 5. 数据形式与输入输出

### 5.1 适合的数据形式

- 自变量类型：单个连续变量（可多组叠加）。
- 因变量类型：不区分。
- 数据结构：一维数值数组（也有二维 KDE）。
- 是否适合高维数据：低维尚可，高维有维度灾难。
- 是否适合缺失较多数据：需先处理缺失。
- 是否适合删失数据：不直接适用。
- 是否适合重复测量数据：混合层级宜分层。

### 5.2 示例表格

| 观测 $x_i$ | $u=(4-x_i)/h$ | $\varphi(u)$ |
| --- | --- | --- |
| 2 | 2 | 0.054 |
| 3 | 1 | 0.242 |
| 5 | −1 | 0.242 |

### 5.3 输入与产出

#### 输入

- 输入数据：一维连续向量。
- 关键变量：带宽 $h$、核函数。
- 需要预处理的内容：处理缺失、对有界变量考虑变换。

#### 产出

- 模型对象/统计结果：连续密度曲线 $\hat{f}_h$。
- 参数估计：无（非参数）。
- 预测结果：任意点密度值。
- 不确定性指标：可用 bootstrap 给密度置信带。

## 6. 适用场景

- 适合：平滑分布可视化、多组分布叠加、多峰探索。
- 不适合：需要精确尾部/边界密度、高维密度、删失数据。
- 使用前需要特别检查的点：带宽是否合适、变量是否有硬边界、样本量是否够。

## 7. 实现

### 7.1 Python

常用包：

- `scipy`
- `seaborn`

```python
import numpy as np
from scipy.stats import gaussian_kde

x = np.array([2, 3, 5])
kde = gaussian_kde(x, bw_method="silverman")
print("f_hat(4) =", float(kde(4)))     # 与手算数量级一致
# 可视化: seaborn.kdeplot(x)
```

### 7.2 R

常用包：

- `stats`

```r
x <- c(2, 3, 5)
d <- density(x, bw = "SJ")   # Sheather-Jones 带宽
plot(d, main = "")
```

## 8. 结果如何解读

- 核心结果看什么：峰的数量与位置、偏态、多组曲线的分离与重叠。
- 每个主要参数如何解读：多峰提示可能存在亚群；峰的位置即数据「扎堆」处。
- 临床或医学意义如何表达：叠加多组 KDE 直观展示分布差异，比只报均值更丰富。
- 常见误读：把过小带宽的毛刺当真峰；把边界处的密度下降当真实（可能是边界偏差）。

## 9. 假设诊断与稳健性

- 带宽敏感性：至少试「默认、更大、更小」三个带宽，看峰是否稳健存在。
- 边界偏差：对 ≥0 的变量（年龄、浓度），密度会漏到负区；可先做对数变换或用有界核。
- 与直方图互证：把 KDE 叠在直方图上，两者一致才可信。
- 稳健性：对少数异常值不如 ECDF 稳健，极端值会拉出小鼓包。

## 11. 优势、局限与常见坑

### 优势

- 连续平滑、无箱边界依赖。
- 多组叠加比较美观直观。
- 能揭示多峰等结构。

### 局限

- 带宽选择敏感。
- 边界偏差、高维失效。
- 不如 ECDF 对异常值稳健。

### 常见坑

- 带宽太小把噪声当峰。
- 对有界变量不处理边界，误读负区密度。
- 只给 KDE 不给样本量，掩盖小样本不稳。

## 12. 与相近方法的区别

- 和**直方图**：同为密度估计，KDE 平滑、无箱边界；直方图简单、有台阶。
- 和**经验分布函数**：ECDF 估累积分布、无需调参；KDE 估密度、需带宽。
- 和**高斯混合模型（GMM）**：GMM 用有限个高斯拟合、可给簇归属；KDE 是非参数、每点一个核。
- 如何选择：只为看形状用 KDE；要读分位数用 ECDF；要建模亚群用 GMM。

## 13. 医学研究中的典型应用

- 叠加比较多组连续指标（如不同亚型的标志物）分布。
- 展示患者某指标的平滑分布用于报告。
- 探索是否存在多峰（潜在亚群）。

## 14. 关键术语

- **核函数（Kernel, $K$）**：以每个观测为中心的对称权重函数，如高斯核。
- **带宽（Bandwidth, $h$）**：核的宽度，控制平滑程度，KDE 最关键的参数。
- **偏差-方差权衡**：带宽大偏差大、带宽小方差大的取舍。
- **Silverman 法则**：由样本标准差与四分位距给默认带宽的经验公式。
- **边界偏差**：有界变量在边界附近密度被系统性低估的问题。
