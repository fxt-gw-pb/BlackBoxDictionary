Kolmogorov-Smirnov Test

Kolmogorov-Smirnov检验（Kolmogorov-Smirnov Test）

§1

方法概览

1.1 一句话本质

KS 检验用「两条累积分布曲线之间的最大垂直距离」作为分布是否相同的证据——距离越大，越有理由说它们来自不同分布。

1.2 定义

Kolmogorov-Smirnov 检验是基于经验分布函数的非参数检验，有两种用法：单样本版检验数据是否服从某指定分布（拟合优度）；两样本版检验两组数据是否来自同一分布。

1.3 它主要解决什么问题

  • 研究问题：一份数据是否服从某分布（如正态）？两组数据的整体分布是否不同？
  • 适用任务：分布拟合优度检验、两样本分布比较、正态性初查。
  • 常见医学场景：检查指标是否近似正态以决定用参数还是非参数方法、比较两组连续指标的整体分布差异。

1.4 直觉与类比

把两条累积曲线想成两条并行的登山路径。如果两条路是「同一座山」（同一分布），它们应当处处贴近；哪怕某段暂时分开，也不会拉开太大。KS 统计量就是量出「两条路在哪一点分得最开、开了多远」。分得越开，越说明它们是两座不同的山。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

比较两个分布，最朴素的做法是比均值（t 检验）或比方差，但那只抓住分布的某一个侧面——两个分布可以均值相同却形状迥异。根本困难是：如何一次性比较分布的「整体」，而不只是某个矩？ KS 用整条累积曲线的最大偏离来概括「整体差异」。

2.2 关键洞察

累积分布函数唯一地刻画一个分布——两个分布相同当且仅当 CDF 处处相同。于是「分布差异」可以浓缩成「两条 CDF 的最大距离」这一个数 $D$。更妙的洞察是：在原假设（分布相同、连续）下，这个 $D$ 的分布不依赖于具体是什么分布（distribution-free），因为经由概率积分变换 $F(X)\sim\mathrm{Uniform}(0,1)$，一切连续分布都被「拉平」成均匀分布——所以同一张临界值表对任何连续分布都适用。

2.3 与朴素/相邻做法的对比

  • 相对 t 检验/秩和检验：它们查「位置（中位/均值）」差异；KS 查「整体分布」差异，对形状/尺度差异也敏感。
  • 相对 Shapiro-Wilk：Shapiro-Wilk 专为正态性设计、小样本功效更高；KS 更通用但功效较低。
  • 相对卡方拟合优度：卡方要分箱、适合离散；KS 不分箱、适合连续。
§3

数学形式

3.1 核心公式

单样本（数据 vs 理论 $F_0$）与两样本（两经验分布）的最大距离：

$$ D_n=\sup_x\,\big|F_n(x)-F_0(x)\big|,\qquad D_{m,n}=\sup_x\,\big|F_{1,m}(x)-F_{2,n}(x)\big| $$

这个式子在说：$D$ 是「所有 $x$ 上、两条曲线纵向差的绝对值」的最大者——就是那道最宽的缝。

3.2 推导脉络

  • 因 ECDF 是阶梯函数，上确界只可能出现在观测点处，故 $D$ 只需在各数据点检查（每点比较跳跃前后两个值）。
  • 原假设下用概率积分变换把任意连续 $F_0$ 变成均匀分布，故 $D_n$ 的分布与 $F_0$ 无关。
  • 大样本下 $\sqrt{n}\,D_n$ 收敛到 Kolmogorov 分布，其分布函数 $P(\sqrt{n}D_n\le t)=1-2\sum_{k\ge 1}(-1)^{k-1}e^{-2k^2t^2}$，据此算 p 值与临界值。
  • 若参照分布的参数是从同一数据估计的（如用样本均值方差检验正态），标准 KS 过于保守，应改用 Lilliefors 校正。

3.3 参数与统计量含义

  • $F_n(x)$：样本的经验分布函数（ECDF）。
  • $D$：KS 统计量，即最大分布距离，取值 0 到 1。
  • p 值：原假设下观察到不小于该 $D$ 的概率。
  • 特性：对分布中部差异较敏感，对尾部相对不敏感。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
连续分布 变量连续、无大量结点 结点多则检验保守、失真 看重复值比例
参数非估计 $F_0$ 参数事先给定 用数据估参后过保守 改用 Lilliefors
独立同分布 观测独立 p 值失真 看设计
§4

手把手算例

单样本 KS：检验 5 个观测 $\{1,2,3,4,9\}$ 是否服从 $\mathrm{Uniform}(0,10)$（理论 $F_0(x)=x/10$）。

一步步计算（在每个观测点比较 ECDF 与 $F_0$，注意 ECDF 在该点跳跃前后各比一次）：

$x_i$ $F_0=x/10$ $F_n$（跳后） $F_n$（跳前） $\|F_n^{后}-F_0\|$ $\|F_0-F_n^{前}\|$
1 0.1 0.2 0.0 0.1 0.1
2 0.2 0.4 0.2 0.2 0.0
3 0.3 0.6 0.4 0.3 0.1
4 0.4 0.8 0.6 0.4 0.2
9 0.9 1.0 0.8 0.1 0.1
  • 取全表最大值：$D=\max=0.4$，出现在 $x=4$。
  • 查 $n=5$、$\alpha=0.05$ 的 KS 临界值约 $0.563$。
  • 因 $D=0.4\lt 0.563$，不拒绝「服从均匀分布」。

结论： 最大缝隙 0.4 出在 $x=4$（数据在低端偏挤，比均匀预期靠左），但 $n=5$ 太小，临界值高达 0.56——即便看到偏离，小样本也没有足够功效下结论。这正好说明 KS（及一切分布检验）在小样本上「不显著」往往只是证据不足，而非「证明服从」。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：单个连续变量（单样本）或两组连续变量（两样本）。
  • 因变量类型：不区分。
  • 数据结构：一维数值数组或两数组。
  • 是否适合高维数据：一维检验；多维需专门扩展。
  • 是否适合缺失较多数据：需先处理缺失。
  • 是否适合删失数据：不直接适用。
  • 是否适合重复测量数据：独立性破坏时不适用。

5.2 示例表格

检验类型 输入 原假设
单样本 数据 + 理论分布 数据服从该分布
两样本 两组数据 两组同分布

5.3 输入与产出

输入

  • 输入数据：一维数值向量（及参照分布或第二组）。
  • 关键变量：待检数据、参照分布或第二组数据。
  • 需要预处理的内容：确认连续、避免用同一数据估参再检验（否则需校正）。

产出

  • 模型对象/统计结果：KS 统计量 $D$、p 值。
  • 参数估计：无。
  • 预测结果：无。
  • 不确定性指标：p 值。
§6

适用场景

  • 适合：连续变量的分布拟合优度、两组整体分布比较。
  • 不适合：小样本（功效低）、对尾部差异敏感的问题、离散/多结点数据、用同一数据估参又检验（改用 Lilliefors）。
  • 使用前需要特别检查的点：参照分布参数是否来自同一数据、结点比例、样本量。
§7

实现

常用包：

  • scipy
from scipy import stats
import numpy as np

x = np.array([1, 2, 3, 4, 9])
print(stats.kstest(x, "uniform", args=(0, 10)))   # 单样本 vs U(0,10)
y = np.array([2, 3, 5, 6, 8])
print(stats.ks_2samp(x, y))                        # 两样本

常用包：

  • stats
x <- c(1, 2, 3, 4, 9)
ks.test(x, "punif", min = 0, max = 10)   # 单样本
y <- c(2, 3, 5, 6, 8)
ks.test(x, y)                            # 两样本
§8

结果如何解读

  • 核心结果看什么：$D$ 的大小与 p 值。
  • 每个主要参数如何解读：p 小说明有证据认为分布不同；$D$ 反映最大分布偏离处（可回看 ECDF 找出在哪偏）。
  • 临床或医学意义如何表达：拒绝正态性提示后续应考虑非参数方法或变换。
  • 常见误读：不拒绝 ≠ 证明分布相同；大样本下极小差异也会显著（要看 $D$ 的实际大小）。
§9

假设诊断与稳健性

  • 参数是否估计：若用样本均值方差检验正态，必须用 Lilliefors 或 Anderson-Darling，否则 p 偏大、过保守。
  • 结点：大量重复值破坏「连续」假设，使 KS 保守；离散数据改用卡方拟合优度。
  • 样本量：小样本功效低（如上例），大样本又对无关紧要的偏离过度敏感——务必结合 $D$ 的实际大小与 ECDF 图判断。
  • 尾部：KS 对尾部不敏感，若关心尾部差异改用 Anderson-Darling。
§10

推荐可视化

  • 两条 ECDF 叠加，标出最大缝隙 $D$ 的位置与高度。
  • ECDF 与理论 CDF 叠加。
  • 两样本 ECDF 差异曲线。
§11

优势、局限与常见坑

优势

  • 非参数、无需假设具体分布族。
  • 对整体分布差异敏感，直观（基于 ECDF）。
  • 有单样本与两样本两种形式。

局限

  • 小样本功效低。
  • 对尾部差异不如中部敏感。
  • 参数由数据估计时标准版过保守，需校正。

常见坑

  • 用同一数据估计正态参数后直接 ks.test(x,"pnorm")（应用 Lilliefors）。
  • 把「不显著」当作「服从该分布」。
  • 忽略结点对连续假设的破坏。
§12

与相近方法的区别

  • 和经验分布函数（ECDF）：ECDF 是描述工具，KS 在其上做假设检验。
  • 和 Shapiro-Wilk：后者专为正态性、小样本功效更高；KS 更通用但功效较低。
  • 和 Anderson-Darling：AD 给尾部更高权重，检尾部差异更灵；KS 偏中部。
  • 和 Wilcoxon 秩和检验：秩和聚焦位置差异，KS 检整体分布差异。
§13

医学研究中的典型应用

  • 建模前检查连续变量是否近似正态。
  • 比较两组患者某指标的整体分布是否不同。
  • 校验模拟/插补数据与原始分布的一致性。
§14

关键术语

  • 拟合优度（Goodness of Fit）：数据与某理论分布吻合程度的检验。
  • KS 统计量 $D$：两条累积分布曲线的最大垂直距离。
  • 上确界（sup）：一组数的最小上界，此处即最大缝隙。
  • 概率积分变换：$F(X)$ 服从均匀分布的性质，是 KS「分布无关」的原因。
  • Lilliefors 校正：参照分布参数由数据估计时对 KS 的修正版本。
  • Kolmogorov 分布：$\sqrt{n}D_n$ 的极限分布，用于算 p 值。
§15

相关方法

  • 经验分布函数（Empirical Cumulative Distribution Function, ECDF）
  • 核密度估计（Kernel Density Estimation, KDE）
  • Wilcoxon秩和检验（Wilcoxon Rank-Sum Test）
