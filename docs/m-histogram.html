直方图

Histogram
状态 · ◑ 已建 难度 · basic 标签 · 医学统计 / 数据科学 / 描述统计 / 分布探索 / 可视化

直方图（Histogram）

§1

方法概览

1.1 一句话本质

直方图把连续变量的取值范围切成一排等宽格子，数一数每格里落了多少观测——它其实是密度函数最朴素的「分段常数」估计。

1.2 定义

直方图通过把数据分箱（bin）并统计每箱的频数或频率密度，用相邻柱形展示一个连续变量的分布形状。它是认识数据的第一张图，也是核密度估计的前身。

1.3 它主要解决什么问题

  • 研究问题：一个连续变量长什么样——中心在哪、多分散、是否偏态、有几个峰、有无异常值？
  • 适用任务：分布探索、建模前的分布假设检查、异常值与截断的初步识别。
  • 常见医学场景：年龄、血压、生物标志物浓度的分布描述与数据质控。

1.4 直觉与类比

想象把每个观测当成一颗豆子，按它的数值扔进对应的格子里，格子越高说明落进去的豆子越多。堆出来的形状，就是数据分布的「侧影」。格子的宽窄决定这张侧影是粗糙还是平滑——这正是直方图最需要小心的地方。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

我们想知道数据背后的概率密度，但密度是个连续函数，样本只有有限个点。根本困难是：如何用有限的点估计一条连续的密度曲线？ 直方图给出最简单的答案——在每个小区间上用「落入的比例」近似该处的密度，得到一条阶梯状的密度估计。

2.2 关键洞察

密度在一个小区间上的平均值 ≈ 该区间的概率 / 区间宽度。于是「频数 / (总数 × 箱宽)」就是密度的一个估计。关键机制在于箱宽 $h$ 的取舍：$h$ 太大，把细节抹平（偏差大）；$h$ 太小，每箱点太少、忽高忽低（方差大）。这就是统计学里无处不在的偏差-方差权衡在最简单场景的现身。

2.3 与朴素/相邻做法的对比

  • 相对「只报均值和标准差」：直方图保留了分布形状（偏态、多峰），而两个数字会把它们藏起来。
  • 相对核密度估计（KDE）：直方图是「硬边界、分段常数」，KDE 用平滑核得到连续曲线，是直方图的光滑化推广。
  • 相对箱线图：箱线图擅长多组比较与异常值，但看不出多峰；直方图能看出多峰但不便并排比较。
§3

数学形式

3.1 核心公式

设箱宽 $h$、第 $k$ 个箱区间 $B_k=[x_0+kh,\,x_0+(k+1)h)$，落入的频数与频率密度为：

$$ n_k=\sum_{i=1}^{n}\mathbf{1}\{x_i\in B_k\},\qquad \hat{f}(x)=\frac{n_k}{n\,h},\quad x\in B_k $$

这个式子在说：某点的密度估计 = 它所在箱的观测数，除以「总数 × 箱宽」；除以 $nh$ 是为了让所有柱形的面积之和等于 1，从而成为合格的密度。

3.2 推导脉络

  • 密度定义：$f(x)=\lim_{h\to 0}\dfrac{P(x\le X\lt x+h)}{h}$。
  • 用样本频率估计概率：$\hat{P}(X\in B_k)=n_k/n$。
  • 除以箱宽还原为「密度」：$\hat{f}(x)=\dfrac{n_k/n}{h}$，即上式。
  • 箱宽选择有经验法则：Sturges（$k\approx\log_2 n+1$，假设近正态）、Scott（$h=3.49\,s\,n^{-1/3}$）、Freedman-Diaconis（$h=2\,\mathrm{IQR}\,n^{-1/3}$，用四分位距，对偏态与异常值更稳健）。

3.3 参数与统计量含义

  • $h$（或箱数）：平滑程度的旋钮，最关键的选择。
  • $n_k$：第 $k$ 箱的频数；$n_k/n$ 为频率。
  • $\hat{f}(x)$：频率密度，使不同箱宽下纵轴可比、可叠加理论密度。
  • 起点 $x_0$：箱的对齐位置，也会影响形状（尤其小样本）。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
变量可分箱 是数值型连续量 类别变量不该用直方图 用条形图代替
样本量足够 每箱有一定观测 小样本形状不稳、锯齿 换更宽箱或用 KDE
观测同质 非多层混合 混合层级掩盖真结构 分层/分面
§4

手把手算例

对 10 个收缩压读数手工做一张直方图：数据 = 118, 122, 125, 128, 130, 131, 134, 138, 142, 149。取箱宽 $h=10$，起点 $x_0=110$。

一步步计算：

  • 分箱计数：
  • $[110,120)$：118 → 1 个
  • $[120,130)$：122,125,128 → 3 个
  • $[130,140)$：130,131,134,138 → 4 个
  • $[140,150)$：142,149 → 2 个
  • 频率（频数/10）：0.1, 0.3, 0.4, 0.2。
  • 频率密度（频率/$h$）：0.010, 0.030, 0.040, 0.020。
  • 校验面积：$(0.010+0.030+0.040+0.020)\times 10=1.0$ ✓。

结论： 分布中心在 130 附近、略右偏（右侧有 142、149 拖尾）。若把箱宽改成 5，$[120,125)$ 只有 1 个、$[125,130)$ 有 2 个……柱形会更细碎、噪声更大——同一份数据、换个箱宽就换个面孔，这正是解读直方图时必须记住的第一课。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：单个连续变量（可按分组分面）。
  • 因变量类型：不区分因变量。
  • 数据结构：一维数值数组。
  • 是否适合高维数据：仅单变量；高维需分面或改用其他图。
  • 是否适合缺失较多数据：需先剔除或说明缺失。
  • 是否适合删失数据：删失需专门处理，不宜直接画。
  • 是否适合重复测量数据：混合层级会掩盖结构，宜分层。

5.2 示例表格

bin 区间 频数 频率 频率密度
[110,120) 1 0.10 0.010
[120,130) 3 0.30 0.030
[130,140) 4 0.40 0.040
[140,150) 2 0.20 0.020

5.3 输入与产出

输入

  • 输入数据：一维数值向量。
  • 关键变量：待描述的连续变量、箱宽/箱数设置。
  • 需要预处理的内容：处理缺失、决定箱宽/规则、是否密度化。

产出

  • 模型对象/统计结果：各箱频数/频率/密度。
  • 参数估计：无（描述性）。
  • 预测结果：无。
  • 不确定性指标：可叠加分组或密度曲线辅助判断。
§6

适用场景

  • 适合：单变量分布的快速探索与质控。
  • 不适合：小样本、需要精确密度、比较多组分布（可用密度曲线或箱线图）。
  • 使用前需要特别检查的点：箱宽是否掩盖或制造了峰、坐标是否被异常值拉伸。
§7

实现

常用包：

  • matplotlib
  • numpy
import matplotlib.pyplot as plt
import numpy as np

x = np.array([118,122,125,128,130,131,134,138,142,149])
plt.hist(x, bins="fd", density=True,          # density=True 使面积=1
         color="#B0A895", edgecolor="white")
plt.xlabel("收缩压"); plt.ylabel("频率密度")
plt.show()

常用包：

  • graphics
x <- c(118,122,125,128,130,131,134,138,142,149)
hist(x, breaks = "FD", freq = FALSE,   # freq=FALSE 画密度
     xlab = "收缩压", main = "")
§8

结果如何解读

  • 核心结果看什么：中心位置、离散程度、偏态、峰数、异常值。
  • 每个主要参数如何解读：右偏（长尾在右）常见于浓度/时间类指标；双峰提示可能存在亚群。
  • 临床或医学意义如何表达：分布形状影响后续检验/模型选择（如是否需对数变换）。
  • 常见误读：把箱宽造成的锯齿当真实结构；用单一直方图比较多组分布。
§9

假设诊断与稳健性

  • 箱宽敏感性：至少用两三个箱宽（如 Sturges、FD）各画一张，看结论是否稳健。
  • 起点敏感性：小样本时移动起点会改变形状，稳健结论应不依赖起点（可转向 KDE）。
  • 异常值：极端值会拉长横轴、压扁主体，必要时截断显示或对数刻度。
  • 与理论分布对比：density=True 后可叠加正态密度曲线，直观看偏离。
§10

推荐可视化

  • 直方图叠加核密度曲线（互相印证）。
  • 分组分面直方图（比较多组）。
  • 与箱线图/小提琴图并列查看。
§11

优势、局限与常见坑

优势

  • 直观、计算简单，是分布探索的第一图。
  • 密度化后可叠加理论分布或 KDE。
  • 易于分组分面。

局限

  • 强烈依赖箱宽与起点。
  • 小样本下形状不稳。
  • 多组比较能力弱。

常见坑

  • 默认箱数不当导致误判分布。
  • 未密度化就跨不同箱宽比较纵轴。
  • 用直方图代替正态性检验下结论。
§12

与相近方法的区别

  • 和核密度估计（KDE）：KDE 用平滑核给出连续密度，避免箱边界的锯齿。
  • 和经验分布函数（ECDF）：ECDF 无需分箱、无信息损失地展示累积分布。
  • 和箱线图：箱线图概括分位数、便于多组比较，但看不出多峰。
  • 如何选择：单组看形状用直方图/KDE；多组比较用箱线图/小提琴图。
§13

医学研究中的典型应用

  • 基线连续变量（年龄、BMI、检验值）的分布描述。
  • 数据清洗中识别异常值、截断与录入错误。
  • 判断是否需要对数变换等预处理。
§14

关键术语

  • 箱 / bin：把取值范围切成的等宽区间。
  • 箱宽（Bin width, $h$）：每个区间的宽度，控制平滑程度。
  • 频率密度（Density）：频率除以箱宽，使柱形面积之和为 1。
  • 偏差-方差权衡：箱宽大偏差大、箱宽小方差大的取舍。
  • Freedman-Diaconis 规则：用四分位距定箱宽的稳健经验法则。
§15

相关方法

  • 核密度估计（Kernel Density Estimation, KDE）
  • 经验分布函数（Empirical Cumulative Distribution Function, ECDF）
