配对样本t检验

Paired t-Test
状态 · ◑ 已建 难度 · basic 标签 · 医学统计 / 数据科学 / 假设检验 / 配对设计 / 参数检验

配对样本t检验（Paired t-Test）

§1

方法概览

1.1 一句话本质

配对 t 检验把「每一对的差值」算出来，再对这一列差值做单样本 t 检验（看它是否偏离 0）——用「配对」抵消个体之间的差异，只留下处理效应。

1.2 定义

配对样本 t 检验用于同一批对象在两种条件下（如治疗前后、左右侧、两种方法）的连续测量比较。它把配对差值转化为单组问题，检验差值均值是否为 0。

1.3 它主要解决什么问题

  • 研究问题：同一批对象在两种条件下的测量是否有系统差异？
  • 适用任务：前后测比较、自身对照、两种测量方法在同一对象上的差异。
  • 常见医学场景：用药前后血压变化、同一患者左右眼指标、干预前后评分。

1.4 直觉与类比

想比较「运动前后体重」。人和人的基础体重差得很大（60 kg 与 90 kg），若直接比两堆数字，这些巨大的个体差异会把「运动减了多少」淹没。配对的高招是：只看每个人自己减了多少，把「你本来多重」这个无关因素消掉，信号一下子清晰。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

前后测数据不独立（同一个人的两次测量高度相关），直接套用两独立样本 t 会违背独立假设。更关键的是，个体间基线差异往往远大于处理效应，成为巨大的噪声。根本困难是：如何在个体差异的噪声里，捞出小小的处理效应？

2.2 关键洞察

对每对取差值 $d_i=x_{i,\text{后}}-x_{i,\text{前}}$。这一步同时解决两件事：差值之间相互独立了（回到单样本框架）；每个人的基线水平在相减中被消掉了（个体差异这个噪声被移除）。于是对 $d_i$ 做「$\mu_0=0$ 的单样本 t 检验」即可。当前后测正相关时，差值的方差远小于原始方差，配对检验的功效因此远高于不配对的做法。

2.3 与朴素/相邻做法的对比

  • 相对两独立样本 t：数据配对时必须用配对 t；误用独立样本 t 会浪费配对带来的功效、且违反独立性。
  • 相对单样本 t：配对 t 本质就是对差值的单样本 t。
  • 相对 Wilcoxon 符号秩：差值严重偏态或有序时改用非参数版。
§3

数学形式

3.1 核心公式

$$ t=\frac{\bar{d}}{s_d/\sqrt{n}},\qquad d_i=x_{i,\text{后}}-x_{i,\text{前}},\quad df=n-1 $$

这个式子在说：先把每对相减得到 $d_i$，再看这列差值的均值 $\bar{d}$ 距 0 有多少个标准误。

3.2 推导脉络

  • 设每对差值 $d_i\sim N(\mu_d,\sigma_d^2)$，原假设 $H_0:\mu_d=0$（前后无系统差异）。
  • 差值均值 $\bar{d}\sim N(\mu_d,\sigma_d^2/n)$，用样本 $s_d$ 估 $\sigma_d$。
  • 得 $t=\dfrac{\bar{d}}{s_d/\sqrt{n}}\sim t_{n-1}$——与单样本 t 完全同构。
  • 方差关系：$\sigma_d^2=\sigma_1^2+\sigma_2^2-2\rho\sigma_1\sigma_2$；正相关 $\rho\gt 0$ 使 $\sigma_d^2$ 变小，这就是配对提升功效的数学来源。

3.3 参数与统计量含义

  • $d_i$：第 $i$ 对的差值；$\bar{d}$：差值均值（即平均处理效应）。
  • $s_d$：差值的样本标准差。
  • $s_d/\sqrt{n}$：差值均值的标准误。
  • $df=n-1$（$n$ 为对子数，不是总观测数）。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
对子独立 不同对子互不影响 p 值失真 看设计
差值正态 $d_i$ 近似正态 小样本 p 不准 差值的 QQ 图
无强异常差 无极端差值 $\bar{d},s_d$ 被带偏 差值箱线图
§4

手把手算例

5 名高血压患者用药前后的收缩压（mmHg）：

患者 前 后 差 $d=$前−后
1 140 132 8
2 135 130 5
3 150 145 5
4 145 138 7
5 138 133 5

一步步计算：

  • 差值均值：$\bar{d}=(8+5+5+7+5)/5=30/5=6.0$。
  • 差值离均差：$2,-1,-1,1,-1$；平方和 $=4+1+1+1+1=8$。
  • 差值方差：$s_d^2=8/(5-1)=2.0$，$s_d=1.414$。
  • 标准误：$s_d/\sqrt{n}=1.414/\sqrt{5}=1.414/2.236=0.632$。
  • 统计量：$t=6.0/0.632=9.49$，自由度 $df=4$。
  • 临界值 $t_{0.975,4}=2.776$；因 $9.49\gg 2.776$，强烈拒绝（$p\lt 0.001$）。
  • 平均降幅 95% CI：$6.0\pm 2.776\times 0.632=(4.2,\ 7.8)$。

结论： 用药后收缩压平均下降 6.0 mmHg（95% CI 4.2–7.8），差异高度显著。注意 $t$ 高达 9.49 的关键在于差值波动极小（$s_d=1.4$）——因为消掉了患者间基线差异。若忽视配对、把 10 个数当两独立组比，个体差异会把信号淹没、$t$ 会小得多。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：配对条件（前/后、左/右、方法 A/B）。
  • 因变量类型：连续型。
  • 数据结构：每行一个对象，含两次测量。
  • 是否适合高维数据：否。
  • 是否适合缺失较多数据：仅用完整对子。
  • 是否适合删失数据：不适合。
  • 是否适合重复测量数据：两时点适用；三个及以上时点用重复测量 ANOVA/混合模型。

5.2 示例表格

患者 前 后
1 140 132
2 135 130
3 150 145

5.3 输入与产出

输入

  • 输入数据：配对的两列测量。
  • 关键变量：两次测量、对子标识。
  • 需要预处理的内容：构造差值、检查差值正态与异常值。

产出

  • 模型对象/统计结果：$t$、自由度、p 值。
  • 参数估计：差值均值 $\bar{d}$。
  • 预测结果：无。
  • 不确定性指标：差值均值的 95% 置信区间。
§6

适用场景

  • 适合：同一对象两条件比较、自身对照、前后测设计。
  • 不适合：两独立组（用两样本 t）、三时点以上（用重复测量 ANOVA）、差值严重偏态（用符号秩）。
  • 使用前需要特别检查的点：数据确为配对、差值正态、异常差值。
§7

实现

常用包：

  • scipy
from scipy import stats

before = [140, 135, 150, 145, 138]
after  = [132, 130, 145, 138, 133]
t, p = stats.ttest_rel(before, after)      # 配对 t
print(f"t={t:.2f}, p={p:.4f}")

常用包：

  • stats
before <- c(140,135,150,145,138)
after  <- c(132,130,145,138,133)
t.test(before, after, paired = TRUE)       # 含差值均值 95% CI
§8

结果如何解读

  • 核心结果看什么：差值均值、其 95% 置信区间、p 值。
  • 每个主要参数如何解读：$\bar{d}$ 是平均处理效应；CI 不含 0 ≈ 显著。
  • 临床或医学意义如何表达：报告「平均变化 + 区间 + 是否有临床意义」。
  • 常见误读：把配对 t 当独立样本 t；只报 p 值不报平均变化与区间。
§9

假设诊断与稳健性

  • 差值正态：对差值（而非原始两组）做 QQ 图/Shapiro-Wilk。
  • 异常差值：个别极端差会主导结论，用差值箱线图排查。
  • 学习/时间效应：前后设计易混入非处理因素（回归到均值、季节），必要时加对照组。
  • 非正态：差值严重偏态改用 Wilcoxon 符号秩检验。
§10

推荐可视化

  • 配对连线图（每个对象一条前后连线）。
  • 差值的直方图/箱线图 + 0 参照线。
  • 差值均值 ± 置信区间点区间图。
§11

优势、局限与常见坑

优势

  • 消除个体间差异，功效高于不配对比较。
  • 计算简单，直接给平均变化与区间。
  • 天然贴合前后测/自身对照设计。

局限

  • 需要真正的配对结构。
  • 对差值正态与异常值敏感。
  • 前后设计难分离处理效应与时间效应。

常见坑

  • 数据配对却误用独立样本 t（丢功效、违独立）。
  • 对原始数据而非差值检验正态。
  • 无对照的前后比较把自然变化当疗效。
§12

与相近方法的区别

  • 和两独立样本 t：配对 vs 独立由数据结构决定；配对时功效更高。
  • 和单样本 t：配对 t = 对差值、$\mu_0=0$ 的单样本 t。
  • 和 Wilcoxon 符号秩：差值非正态的非参数替代。
  • 和重复测量 ANOVA / 混合模型：三个及以上时点或含协变量时的推广。
§13

医学研究中的典型应用

  • 干预前后同一批患者的指标变化。
  • 同一患者两侧肢体/眼睛的对称比较。
  • 两种测量方法在同一批样本上的系统差异。
§14

关键术语

  • 配对差值（$d_i$）：同一对象两条件测量之差，检验的实际对象。
  • 自身对照：以对象自己作对照，消除个体间差异的设计。
  • 标准误（SE）：差值均值的抽样波动，$s_d/\sqrt{n}$。
  • 回归到均值：极端基线在复测时趋向平均的现象，会伪装成疗效。
  • 相关性与功效：前后正相关越强，差值方差越小、功效越高。
§15

相关方法

  • 单样本t检验（One-Sample t-Test）
  • 两独立样本t检验（Two-Sample t-Test）
  • Wilcoxon符号秩检验（Wilcoxon Signed-Rank Test）
  • 线性混合效应模型（Linear Mixed-Effects Model, LMM）
