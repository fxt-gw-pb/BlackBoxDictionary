样本量与检验效能

Sample Size and Power Analysis
状态 · ◑ 已建 难度 · intermediate 标签 · 医学统计 / 研究设计 / 样本量 / 检验效能 / 临床试验

样本量与检验效能（Sample Size and Power Analysis）

§1

方法概览

1.1 一句话本质

样本量估计是在收集数据之前，把「想检出的效应大小、数据的变异、可接受的假阳性率、想要的把握度」四者放进同一个等式，反解出「需要多少人」。

1.2 定义

检验效能（power）是「当真效应确实存在时，研究能把它检出（拒绝原假设）的概率」。样本量估计就是在给定效应量、显著性水平 $\alpha$ 与目标效能 $1-\beta$ 下反推所需样本量；也可反过来在给定样本量下评估能达到的效能。

1.3 它主要解决什么问题

  • 研究问题：这项研究要招募多少受试者，才有足够把握检出有临床意义的效应？
  • 适用任务：临床试验/队列/病例对照的样本量规划、事后效能评估、可行性论证。
  • 常见医学场景：RCT 主要终点的样本量计算、诊断研究所需病例数、基金申请与伦理审查的样本量依据。

1.4 直觉与类比

样本量像相机的曝光时间。光线弱（效应小、变异大）时，需要更长曝光（更大样本）才能拍清楚；光线强（效应大）时，短曝光就够。曝光不足（样本太少），照片一片噪点——真效应也被淹没，得出「无差异」的假阴性结论，这几乎是临床研究最常见、也最浪费的错误。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

假设检验有两类错误：把无效当有效（I 类错误 $\alpha$）和把有效当无效（II 类错误 $\beta$）。传统上大家只盯着 $\alpha=0.05$，却忽视 $\beta$。根本困难是：样本太小时，即使真有效应，检验统计量也常落不到拒绝域里——研究注定「看不见」真相。样本量估计就是在设计阶段把 $\beta$ 也控制住。

2.2 关键洞察

效能不是单独存在的，它由四个量互相牵制：效应量、变异、$\alpha$、样本量。固定其中三个，第四个就被决定了。关键机制是：备择假设为真时，检验统计量的分布会整体平移（非中心分布），平移距离正比于 $\sqrt{n}\cdot(\text{效应}/\text{标准差})$；样本越大，这个分布越深地探进拒绝域，落入拒绝域的概率（效能）就越高。

2.3 与朴素/相邻做法的对比

  • 相对「凭经验拍脑袋定 n」：正式计算把假设透明化，可复现、可审查。
  • 相对「事后再算效能」：事后效能（尤其用观测效应算的）几乎无信息量，规划必须在事前。
  • 与「精度导向」互补：也可按「置信区间半宽不超过某值」定样本量，思路等价。
§3

数学形式

3.1 核心公式

比较两组均值（等方差、双侧）时，每组所需样本量：

$$ n_{\text{每组}}=\frac{2\,(z_{1-\alpha/2}+z_{1-\beta})^2\,\sigma^2}{\delta^2} $$

这个式子在说：需要的样本量与「变异 $\sigma^2$」成正比、与「想检出的差 $\delta$ 的平方」成反比——差越小、越难测，所需样本以平方速度暴涨。

3.2 推导脉络

  • 检验统计量 $Z=\dfrac{\bar{x}_1-\bar{x}_2}{\sigma\sqrt{2/n}}$，原假设下 $Z\sim N(0,1)$。
  • 备择假设（真差为 $\delta$）下，$Z$ 的均值平移到非中心参数 $\lambda=\dfrac{\delta}{\sigma\sqrt{2/n}}=\dfrac{\delta}{\sigma}\sqrt{\dfrac{n}{2}}$。
  • 要求「在 $\alpha$ 下拒绝的概率」达到 $1-\beta$，即 $\lambda\ge z_{1-\alpha/2}+z_{1-\beta}$。
  • 代入 $\lambda$ 反解 $n$ 即得上式。其他设计（比例、生存、相关）思路相同，只是把「效应/标准差」换成对应效应量。

3.3 参数与统计量含义

  • $\delta$：想检出的均值差（应取有临床意义的最小差 MCID，而非「希望的」大差）。
  • $\sigma$：结局的标准差（来自既往文献或预试验）。
  • $\alpha$：显著性水平（I 类错误率，常 0.05）；$z_{1-\alpha/2}=1.96$。
  • $1-\beta$：目标效能（常 0.8 或 0.9）；效能 0.8 时 $z_{1-\beta}=0.84$。
  • 标准化效应量 $d=\delta/\sigma$：真正决定样本量的是它，而非 $\delta$ 或 $\sigma$ 单独。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
$\sigma$ 估计可靠 用的变异接近真实 低估 $\sigma$ → 样本不足、效能虚高 敏感性分析、取保守值
效应量合理 $\delta$ 取的是 MCID 取值过大 → 样本偏小 与临床共识对照
无过多脱落 实际可分析样本≈计划 有效样本缩水、效能下降 按脱落率上调 $n$
§4

手把手算例

设计一项 RCT，比较新药与安慰剂对收缩压的降幅：临床上认为降幅相差 5 mmHg 才有意义（$\delta=5$），已知收缩压标准差 $\sigma=10$，取 $\alpha=0.05$（双侧）、效能 $1-\beta=0.8$。

一步步计算：

  • 查标准正态分位：$z_{1-\alpha/2}=z_{0.975}=1.96$，$z_{1-\beta}=z_{0.80}=0.84$。
  • 两分位之和的平方：$(1.96+0.84)^2=2.80^2=7.84$。
  • 代入公式：$n=\dfrac{2\times 7.84\times 10^2}{5^2}=\dfrac{2\times 7.84\times 100}{25}=\dfrac{1568}{25}=62.7$。
  • 向上取整：每组 63 人，两组共 126 人。
  • 若预计 15% 脱落，需上调：$126/(1-0.15)\approx 149$ 人。

结论： 想稳定检出 5 mmHg 的差异，每组约需 63 人。若把目标差改成 3 mmHg，$n$ 会涨到约 $63\times(5/3)^2\approx175$/组——这直观展示了「越小的效应，样本量以平方暴涨」，也是为什么设计阶段必须诚实地设定 MCID。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：设计变量（分组、暴露）。
  • 因变量类型：连续、二分类、时间到事件、相关等（各有对应公式）。
  • 数据结构：尚无数据，基于假设参数计算。
  • 是否适合高维数据：多重比较需相应调整 $\alpha$。
  • 是否适合缺失较多数据：应按预期脱落率上调样本量。
  • 是否适合删失数据：生存设计按「所需事件数」而非人数计算。
  • 是否适合重复测量数据：需考虑组内相关（如 ICC、设计效应）。

5.2 示例表格

输入参数 取值 来源
最小重要差 δ 5 mmHg 临床共识
标准差 σ 10 mmHg 既往研究
α（双侧） 0.05 惯例
效能 1−β 0.80 惯例

5.3 输入与产出

输入

  • 输入数据：效应量、变异、$\alpha$、目标效能、分配比例。
  • 关键变量：结局类型、检验类型、单/双侧、脱落率。
  • 需要预处理的内容：把参数换算为标准化效应量。

产出

  • 模型对象/统计结果：所需样本量或达到的效能。
  • 参数估计：无（是设计计算）。
  • 预测结果：效能曲线（样本量 vs 效能）。
  • 不确定性指标：对参数假设的敏感性分析。
§6

适用场景

  • 适合：任何前瞻性研究的设计阶段、基金/伦理申请、可行性评估。
  • 不适合：用观测到的效应做「事后效能」为阴性结果开脱（无信息量）。
  • 使用前需要特别检查的点：效应量是否取 MCID、$\sigma$ 是否可靠、是否计入脱落与多重比较。
§7

实现

常用包：

  • statsmodels
from statsmodels.stats.power import TTestIndPower

analysis = TTestIndPower()
n = analysis.solve_power(effect_size=5/10,   # d = δ/σ
                         alpha=0.05, power=0.8, ratio=1.0,
                         alternative="two-sided")
print(f"每组样本量 ≈ {n:.1f}")

常用包：

  • pwr
library(pwr)
pwr.t.test(d = 5/10, sig.level = 0.05, power = 0.8,
           type = "two.sample", alternative = "two.sided")
§8

结果如何解读

  • 核心结果看什么：所需样本量（含脱落上调后的实际招募目标）。
  • 每个主要参数如何解读：效应量翻倍，样本量降为约 1/4；效能从 0.8 提到 0.9，样本量约增 34%。
  • 临床或医学意义如何表达：报告「为在 α=0.05、效能 80% 下检出 δ 的差异，需 N 人」，并给出参数来源。
  • 常见误读：把「未达显著」当「证明无效」；用夸大的效应量把样本量压小。
§9

假设诊断与稳健性

  • 效应量来源：优先用最小临床重要差（MCID），而非既往研究里可能被夸大的观测效应。
  • 变异来源：$\sigma$ 用文献或预试验；不确定时取偏大的保守值，并做敏感性分析（画多条效能曲线）。
  • 脱落与依从：按预期失访率上调样本量，长随访研究尤其重要。
  • 多重性与聚类：多主终点需调 $\alpha$；整群随机需乘设计效应 $1+(m-1)\rho$。
§10

推荐可视化

  • 效能曲线：横轴样本量、纵轴效能，标出目标 0.8/0.9。
  • 敏感性热图：效应量 × 变异 对样本量的影响。
  • 不同 α 或分配比例下的样本量对比。
§11

优势、局限与常见坑

优势

  • 让研究「有足够眼力」看见真效应，减少浪费与伦理风险。
  • 把设计假设透明化，可复现、可审查。
  • 是基金与伦理申请的硬性要求。

局限

  • 强依赖对效应量与变异的事前假设。
  • 简化公式未涵盖复杂设计（需模拟）。
  • 事后效能几乎无意义。

常见坑

  • 用「期望的大效应」而非 MCID，导致样本严重不足。
  • 忘记脱落、忘记多重比较调整。
  • 把 p 大解读为「证明等效」（等效性需专门设计）。
§12

与相近方法的区别

  • 和假设检验：检验是数据到手后的推断，样本量是数据之前的规划，二者用同一套分布理论。
  • 和置信区间/精度法：可按「区间半宽」定样本量，与效能法互为表里。
  • 和等效性/非劣效性设计：目标不是「检出差异」而是「排除有意义的差异」，公式与边界不同。
  • 如何选择：优效试验用效能法定 δ；纯估计研究用精度法定区间宽度。
§13

医学研究中的典型应用

  • RCT 主要终点的样本量计算与方案撰写。
  • 诊断准确性研究所需阳性/阴性病例数。
  • 队列研究在给定暴露率下检出某 RR 所需人数。
§14

关键术语

  • 检验效能（Power, $1-\beta$）：真效应存在时能检出它的概率。
  • I 类错误（$\alpha$）：无效应却判为有效（假阳性）。
  • II 类错误（$\beta$）：有效应却判为无效（假阴性）。
  • 效应量（Effect Size）：标准化的差异大小，如 $d=\delta/\sigma$，决定样本量的核心量。
  • 最小临床重要差（MCID）：小到临床仍在意的差异，应作为 $\delta$。
  • 非中心参数（Noncentrality）：备择假设下检验统计量分布的平移量，正比于 $\sqrt{n}\cdot d$。
