15·01
多重检验与错误率控制 Multiple Testing and Error Rate Control

多重检验校正不是“让 P 值更难显著”，而是把单次检验的错误承诺升级为对整组发现的错误承诺。

◑ 已建 intermediate
15·02
Bonferroni校正 Bonferroni Correction

Bonferroni 把总的第一类错误预算 $\alpha$ 平分给 $m$ 个检验，以最简单、最稳健的方式控制 FWER。

◑ 已建 basic
15·03
Holm程序 Holm Procedure

Holm 先审最小 P 值，再逐步放宽阈值；它与 Bonferroni 同样控制 FWER，却从不更差。

◑ 已建 basic
15·04
Benjamini-Hochberg程序 Benjamini-Hochberg Procedure

BH 不要求“一个假阳性都不能有”，而是控制发现集合中假发现比例的期望，从而在探索性高维研究中换取更高发现能力。

◑ 已建 basic
15·05
ROC曲线与AUC Receiver Operating Characteristic and AUC

ROC 扫描所有阈值观察灵敏度—假阳性权衡，AUC 则等于随机一名阳性得分高于随机一名阴性的概率。

◑ 已建 basic
15·06
校准曲线 Calibration Curve

校准回答“预测 20% 风险的人是否约有 20% 发病”，关注概率的绝对可信度，而不是排序。

◑ 已建 intermediate
15·07
决策曲线分析 Decision Curve Analysis, DCA

DCA 用阈值概率把漏诊与误治代价换成同一尺度，比较模型、全治和全不治谁带来更高净获益。

◑ 已建 intermediate
15·08
交叉验证 Cross-Validation

交叉验证轮流把每部分样本当作未见数据，用多次“训练—验证”近似模型对新样本的泛化表现。

◑ 已建 basic
15·09
SHAP Shapley Additive Explanations, SHAP

SHAP 把一次预测与参考基线之间的差额，按特征在所有可能加入顺序中的平均边际贡献进行分账。

◑ 已建 intermediate
15·10
一致性指数 Concordance Index, C-index

C-index 从所有能够判断结局先后的个体对中，计算模型把更早发生事件者排在更高风险位置的比例。

◑ 已建 intermediate
