Spearman秩相关

Spearman Rank Correlation
状态 · ◑ 已建 难度 · basic 标签 · 医学统计 / 数据科学 / 相关分析 / 非参数 / 单调关系

Spearman秩相关（Spearman Rank Correlation）

§1

方法概览

1.1 一句话本质

Spearman 相关就是「把数据换成名次后再算 Pearson 相关」：它衡量两个变量是否同增同减（单调关系），不要求关系是直线，且对异常值稳健。

1.2 定义

Spearman 秩相关系数 $\rho$ 度量两变量之间单调关联的强弱与方向。它把原始值替换为秩，再计算 Pearson 相关，取值 $[-1,1]$。

1.3 它主要解决什么问题

  • 研究问题：两个变量是否同向变化（哪怕不是线性）？
  • 适用任务：单调关联量化、有序变量相关、含异常值/偏态数据的关联。
  • 常见医学场景：疾病分级与某指标、剂量等级与反应、评分量表之间的关联。

1.4 直觉与类比

把两列数据各自换成「排名」（第 1 名、第 2 名……）。如果一个变量排名靠前的个体，另一个变量排名也靠前，两份排名就高度一致，$\rho$ 接近 1。因为只看名次不看具体数值，一个夸张的异常值最多占「最后一名」，不会像 Pearson 那样把结果带跑——这就是它稳健的来源。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

Pearson 只捕捉线性关系，且对异常值敏感。但很多真实关系是「单调但弯曲」的（如剂量-反应的 S 形），或数据本就是有序等级、或有极端值。根本困难是：如何度量「同增同减」这种单调关联，而不管它是否为直线、也不被异常值绑架？

2.2 关键洞察

单调关系在秩尺度上会被「拉直」——只要 $y$ 随 $x$ 单调递增，它们的秩就完全线性对应。于是把原值换成秩，再套用 Pearson，就把「单调」问题化归为「线性」问题。秩只保留顺序、丢弃具体间距，天然抗异常值，也直接适用于有序数据。

2.3 与朴素/相邻做法的对比

  • 相对 Pearson：Spearman 抓单调（更宽）、抗异常值；Pearson 抓线性、对异常值敏感。
  • 相对 Kendall：两者都测单调关联；Kendall 基于「和谐对」更易解释、结点/小样本更稳，Spearman 更常见、与 Pearson 同源。
  • 相对回归：相关是对称的关联度量，不建立预测方程。
§3

数学形式

3.1 核心公式

无结点时的简洁公式（$d_i$ 为两变量秩之差）：

$$ \rho=1-\frac{6\sum_{i=1}^{n} d_i^2}{n(n^2-1)} $$

这个式子在说：把每个观测在两变量上的名次相减、平方、求和；秩差越小（名次越一致），$\rho$ 越接近 1。它其实就是「在秩上算的 Pearson 相关」的等价简化。

3.2 推导脉络

  • 一般定义：$\rho=\mathrm{Pearson}\big(\mathrm{rank}(x),\ \mathrm{rank}(y)\big)$——对秩做 Pearson。
  • 当无结点时，秩是 $1\dots n$ 的排列，代入 Pearson 公式并化简，得到上面的 $d^2$ 简式。
  • 有结点时应回退到「对秩做 Pearson」的一般式（简式不再精确）。
  • 显著性：小样本查精确表，大样本用 $t=\rho\sqrt{\dfrac{n-2}{1-\rho^2}}$ 近似。

3.3 参数与统计量含义

  • $\rho$：秩相关系数，方向 + 单调强度。
  • $d_i$：第 $i$ 个观测在两变量上的秩之差。
  • $n$：样本量。
  • p 值：单调关联是否显著。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
单调 关系单调（可弯曲） 非单调（U 形）会被漏 看散点图
可排序 变量可排名 大量结点需用一般式 看重复值
观测独立 成对独立 p 值失真 看设计
§4

手把手算例

沿用 Pearson 卡的数据：$x=\{1,2,3,4,5\}$，$y=\{2,3,5,4,6\}$。

一步步计算：

  • $x$ 的秩：$\{1,2,3,4,5\}$（本就递增）。
  • $y$ 的秩：把 $y$ 从小到大 $2,3,4,5,6$ 对应名次 $1,2,3,4,5$，故 $y=\{2,3,5,4,6\}$ 的秩为 $\{1,2,4,3,5\}$。
  • 秩差 $d=x_{\text{秩}}-y_{\text{秩}}=\{0,0,-1,1,0\}$，$d^2=\{0,0,1,1,0\}$，$\sum d^2=2$。
  • 代入：$\rho=1-\dfrac{6\times 2}{5(25-1)}=1-\dfrac{12}{120}=1-0.1=0.90$。

结论： $\rho=0.90$，与该数据的 Pearson $r=0.90$ 恰好接近——因为这份数据既线性又单调。差别会在「弯曲但单调」或「有异常值」时显现：那时 Pearson 会下滑而 Spearman 仍高。对照 Kendall秩相关（Kendall Rank Correlation） 卡的同数据 $\tau=0.80$，可见三种系数量纲不同、不可直接互比大小。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：两个连续或有序变量。
  • 因变量类型：不区分。
  • 数据结构：成对观测。
  • 是否适合高维数据：可算秩相关矩阵。
  • 是否适合缺失较多数据：需成对完整。
  • 是否适合删失数据：不适合。
  • 是否适合重复测量数据：需考虑组内相关。

5.2 示例表格

i x y x秩 y秩 d²
3 3 5 3 4 1
4 4 4 4 3 1

5.3 输入与产出

输入

  • 输入数据：两列成对变量。
  • 关键变量：两变量。
  • 需要预处理的内容：查散点图、处理结点与缺失。

产出

  • 模型对象/统计结果：$\rho$、p 值。
  • 参数估计：$\rho$。
  • 预测结果：无。
  • 不确定性指标：$\rho$ 的置信区间（可选）。
§6

适用场景

  • 适合：单调（含非线性）关联、有序变量、含异常值/偏态数据。
  • 不适合：非单调关系（U 形等）、需要线性解释力时。
  • 使用前需要特别检查的点：散点图看单调性、结点比例。
§7

实现

常用包：

  • scipy
from scipy import stats

x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 4, 6]
rho, p = stats.spearmanr(x, y)
print(f"rho={rho:.2f}, p={p:.3f}")     # rho=0.90

常用包：

  • stats
x <- c(1,2,3,4,5); y <- c(2,3,5,4,6)
cor.test(x, y, method = "spearman")
§8

结果如何解读

  • 核心结果看什么：$\rho$ 的符号与绝对值、p 值。
  • 每个主要参数如何解读：$\rho$ 高说明单调一致强；与 Pearson 差距大提示非线性或异常值。
  • 临床或医学意义如何表达：报告单调关联方向与强度，强调非因果。
  • 常见误读：把 $\rho$ 当线性解释力（它是单调关联）；忽略非单调关系。
§9

假设诊断与稳健性

  • 单调性：散点图确认；非单调关系秩相关也会低。
  • Pearson 对比：$\rho\gg r$ 提示单调但非线性或有异常值。
  • 结点：大量结点用「对秩做 Pearson」的一般式。
  • 稳健性：对异常值远比 Pearson 稳健。

§11

优势、局限与常见坑

优势

  • 抓单调关系、不要求线性。
  • 对异常值稳健。
  • 适用有序数据。

局限

  • 不区分线性与非线性单调。
  • 无法捕捉非单调关系。
  • 不给线性解释力。

常见坑

  • 把 $\rho$ 当 $r^2$ 式的解释力。
  • 忽略非单调（U 形）关系。
  • 大量结点仍用简式公式。
§12

与相近方法的区别

  • 和 Pearson：Spearman 单调、稳健；Pearson 线性、敏感。
  • 和 Kendall：都测单调；Kendall 基于和谐对、更易解释与抗结点，Spearman 更常见。
  • 和回归：相关对称、不建预测方程。
  • 如何选择：单调/有序/异常值 → Spearman 或 Kendall；线性且干净 → Pearson。
§13

医学研究中的典型应用

  • 疾病严重度分级与某连续指标的关联。
  • 有序量表之间的相关。
  • 含异常值的实验室指标关联。
§14

关键术语

  • 秩（Rank）：数值排序后的名次。
  • 单调关系（Monotonic）：一个变量增另一个总体也增（或总体减）。
  • 秩差（$d_i$）：同一观测两变量名次之差。
  • $\rho$（rho）：Spearman 秩相关系数。
  • 结点（Ties）：相等观测，影响秩与公式选择。
