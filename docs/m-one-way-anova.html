单因素方差分析

One-Way ANOVA
状态 · ◑ 已建 难度 · basic 标签 · 医学统计 / 数据科学 / 假设检验 / 方差分析 / 多组比较

单因素方差分析（One-Way ANOVA）

§1

方法概览

1.1 一句话本质

ANOVA 用一个巧妙的对比回答「多组均值是否相同」：把数据的总变异拆成「组与组之间的差异」和「组内部的波动」，若前者远大于后者（$F$ 大），就说明组间真有差别。

1.2 定义

单因素方差分析比较三个及以上独立组的连续结局均值是否全部相等。它通过方差分解构造 $F$ 统计量，是多组均值比较的标准参数方法。名字叫「方差」分析，检验的却是「均值」。

1.3 它主要解决什么问题

  • 研究问题：多个组的平均水平是否存在差异（至少有一组与众不同）？
  • 适用任务：多组均值的整体（omnibus）检验，通常配合事后两两比较。
  • 常见医学场景：三种剂量组的疗效比较、多个年龄段的指标差异、不同治疗方案的结局均值。

1.4 直觉与类比

想象几支球队的身高。如果各队平均身高差不多，队与队之间的差就和「同队内部高矮不齐」差不多大；但如果某队明显更高，队间差就会盖过队内波动。ANOVA 就是比较这两把尺子：「组间差异」相对「组内波动」大到什么程度，才能断定各队真的不一样。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

三组比较若做三次两两 t 检验，每次 5% 假阳性会累积——3 次约 14%、5 组 10 次高达 40%，多重比较使总体假阳性率失控。根本困难是：如何用一次检验，判断「所有组均值相等」这个整体假设？

2.2 关键洞察

变异可以分解：总变异（每个点离总均值多远）= 组间变异（各组均值离总均值多远）+ 组内变异（各点离本组均值多远）。若各组均值真相等，「组间」只反映随机波动，应与「组内」同量级，比值 $F=\text{组间均方}/\text{组内均方}\approx 1$；若组间被真实差异撑大，$F$ 显著大于 1。这把「多组比较」浓缩成一个 $F$ 值。

2.3 与朴素/相邻做法的对比

  • 相对多次两两 t 检验：ANOVA 一次检验控制整体假阳性率，再由事后检验定位差异。
  • 相对两样本 t：两组时 ANOVA 与 t 检验等价（$F=t^2$）。
  • 相对 Kruskal-Wallis：数据非正态/有序时用其非参数版。
§3

数学形式

3.1 核心公式

$$ F=\frac{\text{MS}_{\text{组间}}}{\text{MS}_{\text{组内}}} =\frac{\text{SS}_{\text{组间}}/(k-1)}{\text{SS}_{\text{组内}}/(N-k)} $$

其中 $\text{SS}_{\text{组间}}=\sum_j n_j(\bar{x}_j-\bar{x})^2$，$\text{SS}_{\text{组内}}=\sum_j\sum_i (x_{ij}-\bar{x}_j)^2$。这个式子在说：把两种变异各自平均（除以自由度）成「均方」，再看组间均方是组内均方的几倍。

3.2 推导脉络

  • 平方和分解恒等式：$\text{SS}_{\text{总}}=\text{SS}_{\text{组间}}+\text{SS}_{\text{组内}}$（交叉项为 0）。
  • 在 $H_0$（各组均值相等）与正态等方差假设下，$\text{MS}_{\text{组内}}$ 与 $\text{MS}_{\text{组间}}$ 都是 $\sigma^2$ 的无偏估计，故 $F\approx 1$。
  • $H_0$ 下 $F\sim F_{k-1,\,N-k}$；组间被真实差异撑大时 $F$ 偏大，落入右尾即拒绝。
  • 拒绝只说明「至少两组不同」，不指明是哪两组——需事后比较（如 Tukey HSD）。

3.3 参数与统计量含义

  • $k$：组数；$N$：总样本量；$n_j,\bar{x}_j$：第 $j$ 组样本量与均值；$\bar{x}$：总均值。
  • $\text{SS}$：平方和；$\text{MS}=\text{SS}/df$：均方。
  • 自由度：组间 $k-1$、组内 $N-k$。
  • $F$：组间与组内均方之比，$\ge 0$。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
独立 组间、组内观测独立 p 值失真 看设计
正态 各组结局近似正态 小样本 p 不准 残差 QQ 图
方差齐 各组方差相等 F 偏差 Levene 检验；不齐用 Welch ANOVA
§4

手把手算例

三个剂量组各 3 名患者的疗效评分：A = {5,6,7}，B = {7,8,9}，C = {9,10,11}。

一步步计算：

  • 各组均值：$\bar{x}_A=6,\ \bar{x}_B=8,\ \bar{x}_C=10$；总均值 $\bar{x}=8$。
  • 组间平方和：$\text{SS}_{\text{组间}}=3[(6-8)^2+(8-8)^2+(10-8)^2]=3[4+0+4]=24$，$df=k-1=2$，$\text{MS}_{\text{组间}}=12$。
  • 组内平方和：每组离均差都是 $\{-1,0,1\}$，平方和各 2，合计 $\text{SS}_{\text{组内}}=6$，$df=N-k=6$，$\text{MS}_{\text{组内}}=1$。
  • $F=\text{MS}_{\text{组间}}/\text{MS}_{\text{组内}}=12/1=12$。
  • 临界值 $F_{0.05}(2,6)=5.14$；因 $12\gt 5.14$，拒绝「三组均值相等」。

结论： 组间差异是组内波动的 12 倍，远超随机能解释的范围，说明剂量组之间确有均值差异。但 ANOVA 只说「有组不同」，究竟哪两组不同，要接着做 TukeyHSD多重比较（Tukey Honest Significant Difference）（下一张卡用同一份数据继续算）。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：一个多分类分组变量（≥3 组）。
  • 因变量类型：连续型。
  • 数据结构：多组独立观测。
  • 是否适合高维数据：单因素单结局；多因素用多因素 ANOVA。
  • 是否适合缺失较多数据：按可用样本，注意缺失机制。
  • 是否适合删失数据：不适合。
  • 是否适合重复测量数据：不适合；重复测量用重复测量 ANOVA/混合模型。

5.2 示例表格

剂量组 评分
A 5, 6, 7
B 7, 8, 9
C 9, 10, 11

5.3 输入与产出

输入

  • 输入数据：分组变量 + 连续结局。
  • 关键变量：组别、结局值。
  • 需要预处理的内容：检查正态、方差齐、异常值。

产出

  • 模型对象/统计结果：ANOVA 表（SS、df、MS、$F$、p）。
  • 参数估计：各组均值。
  • 预测结果：无。
  • 不确定性指标：各组均值区间；事后比较的差值区间。
§6

适用场景

  • 适合：三组及以上独立组连续结局的整体均值比较。
  • 不适合：两组（直接用 t）、重复测量、严重偏态（用 Kruskal-Wallis）。
  • 使用前需要特别检查的点：独立性、正态性、方差齐性（不齐用 Welch ANOVA）。
§7

实现

常用包：

  • scipy
  • statsmodels
from scipy import stats

A = [5, 6, 7]; B = [7, 8, 9]; C = [9, 10, 11]
F, p = stats.f_oneway(A, B, C)
print(f"F={F:.2f}, p={p:.4f}")     # F=12.00
# 方差不齐时用 Welch: pingouin.welch_anova

常用包：

  • stats
score <- c(5,6,7, 7,8,9, 9,10,11)
group <- factor(rep(c("A","B","C"), each = 3))
summary(aov(score ~ group))        # ANOVA 表
oneway.test(score ~ group)         # Welch(不假设方差齐)
§8

结果如何解读

  • 核心结果看什么：$F$ 与 p，以及各组均值的差距。
  • 每个主要参数如何解读：p 小说明「至少有一组均值不同」，但不指明哪组。
  • 临床或医学意义如何表达：整体显著后，用事后比较报告具体哪两组差、差多少（带区间）。
  • 常见误读：把「整体显著」当成「每两组都不同」；只报 F/p 不看效应量（如 $\eta^2$）。
§9

假设诊断与稳健性

  • 正态性：看残差 QQ 图；小样本更关键，大样本较稳。
  • 方差齐性：Levene/Bartlett 检验；不齐改用 Welch ANOVA（推荐做法）。
  • 异常值：箱线图排查，极端值会撑大组内或组间。
  • 非正态：严重偏态/有序结局改用 Kruskal-Wallis。
§10

推荐可视化

  • 各组箱线图/小提琴图 + 抖动点。
  • 各组均值 ± 置信区间的点区间图。
  • 事后两两差值的置信区间图。

§11

优势、局限与常见坑

优势

  • 一次检验控制多组比较的整体假阳性率。
  • 方差分解框架可扩展到多因素、协方差分析。
  • 与回归/线性模型统一。

局限

  • 只给「是否有组不同」，需事后比较定位。
  • 对正态与方差齐敏感。
  • 不处理重复测量（需混合模型）。

常见坑

  • 显著后直接说「各组都不同」。
  • 方差明显不齐仍用经典 ANOVA（应 Welch）。
  • 多个结局各做一次 ANOVA 却不校正多重性。
§12

与相近方法的区别

  • 和两样本 t 检验：两组时等价（$F=t^2$）；三组以上才需 ANOVA。
  • 和 Tukey HSD 等事后检验：ANOVA 管整体，事后检验管两两定位。
  • 和 Kruskal-Wallis：非正态/有序时的非参数替代。
  • 和线性回归：ANOVA 是以分类自变量为主的线性模型的特例。
§13

医学研究中的典型应用

  • 多剂量/多方案组的连续结局比较。
  • 多个疾病亚型或年龄段的指标差异。
  • 多中心研究中各中心均值的整体比较。
§14

关键术语

  • 组间平方和（$\text{SS}_{\text{组间}}$）：各组均值离总均值的加权平方和，反映组间差异。
  • 组内平方和（$\text{SS}_{\text{组内}}$）：各点离本组均值的平方和，反映组内波动。
  • 均方（MS）：平方和除以自由度，是方差的估计。
  • F 统计量：组间均方与组内均方之比。
  • omnibus 检验：一次性检验「所有组相等」的整体检验。
  • 事后比较（Post hoc）：整体显著后定位具体差异的两两比较。
§15

相关方法

  • 两独立样本t检验（Two-Sample t-Test）
  • TukeyHSD多重比较（Tukey Honest Significant Difference）
  • Kruskal-Wallis检验（Kruskal-Wallis Test）
  • 线性回归（Linear Regression）
