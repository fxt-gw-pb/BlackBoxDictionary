经验分布函数

Empirical Cumulative Distribution Function, ECDF
状态 · ◑ 已建 难度 · basic 标签 · 医学统计 / 数据科学 / 描述统计 / 非参数 / 分布探索

经验分布函数（Empirical Cumulative Distribution Function, ECDF）

§1

方法概览

1.1 一句话本质

经验分布函数是「用数据本身当作分布」：在每个观测值处向上跳一小步 $1/n$，画出「有多大比例的数据不超过 $x$」这条阶梯曲线，它是真实累积分布函数（CDF）最诚实的估计。

1.2 定义

ECDF 是一个阶梯函数 $F_n(x)$，等于样本中「小于等于 $x$ 的观测所占比例」。它不假设任何分布形式，是完全非参数的分布描述工具，也是 KS 检验、分位数、生存曲线等方法的基石。

1.3 它主要解决什么问题

  • 研究问题：数据在各个取值以下的累积占比是多少？中位数、四分位数在哪？
  • 适用任务：分布探索、分位数读取、两组分布比较、分布拟合检验的可视化。
  • 常见医学场景：比较两组指标的整体分布、读取某阈值以下的患者比例、检查分布是否近似某理论分布。

1.4 直觉与类比

把所有数据从小到大排成一队，站到某个数值 $x$ 前面，回头数「队伍里有多大比例排在我（含）前面」——这个比例随 $x$ 从 0 爬到 1，就是 ECDF。它像一部「累积点票」的记录：每来一票（一个观测），累计比例就往上跳一格。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

我们想知道总体的 CDF $F(x)=P(X\le x)$，但总体不可见，只有样本。直方图和 KDE 要选箱宽/带宽这类「旋钮」，结果会随旋钮变。根本困难是：能不能不调任何旋钮、不损失信息地估计分布？ ECDF 正是答案——它把每个观测原封不动地放进曲线，无参数、无平滑、无信息损失。

2.2 关键洞察

用「频率」估「概率」：$P(X\le x)$ 用样本里「$\le x$ 的比例」来估。每个观测是否 $\le x$ 是一次伯努利试验，$F_n(x)$ 就是这些 0/1 的平均值。由此它无偏（期望正好是 $F(x)$），且由大数定律逐点收敛、由 Glivenko-Cantelli 定理一致收敛到真 CDF——样本越大，阶梯越贴合真实曲线。

2.3 与朴素/相邻做法的对比

  • 相对直方图：直方图要分箱、会丢信息且随箱宽变；ECDF 不分箱、无信息损失。
  • 相对核密度估计（KDE）：KDE 估的是密度（导数）、需带宽；ECDF 估的是累积分布、无需调参。
  • 代价：ECDF 是阶梯状（不平滑），看「密度形状」不如直方图/KDE 直观，但看「累积比例、分位数、分布比较」更准。
§3

数学形式

3.1 核心公式

$$ F_n(x)=\frac{1}{n}\sum_{i=1}^{n}\mathbf{1}\{x_i\le x\} $$

这个式子在说：$x$ 处的函数值 = 样本中「不超过 $x$ 的观测个数」除以总数 $n$；每遇到一个观测，曲线就向上跳 $1/n$。

3.2 推导脉络

  • 令指示变量 $I_i=\mathbf{1}\{x_i\le x\}$，它服从 $\mathrm{Bernoulli}(F(x))$。
  • $F_n(x)=\frac{1}{n}\sum_i I_i$ 是这些伯努利变量的样本均值。
  • 期望：$E[F_n(x)]=F(x)$（无偏）；方差：$\mathrm{Var}[F_n(x)]=\dfrac{F(x)(1-F(x))}{n}$（随 $n$ 收缩）。
  • 一致收敛：Glivenko-Cantelli 定理保证 $\sup_x|F_n(x)-F(x)|\to 0$（几乎必然）；DKW 不等式还给出置信带 $F_n(x)\pm\varepsilon$，其中 $\varepsilon=\sqrt{\ln(2/\alpha)/(2n)}$。

3.3 参数与统计量含义

  • $F_n(x)$：截至 $x$ 的累积比例，取值 0 到 1。
  • 跳跃高度 $1/n$：每个观测的「权重」；重复值处跳 $k/n$。
  • 分位数：$F_n$ 的反函数，如中位数 = $F_n(x)=0.5$ 处的 $x$。
  • DKW 半宽 $\varepsilon$：给整条曲线的同时置信带宽度。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
独立同分布 观测来自同一总体、互不影响 曲线不代表单一分布 分层看
无严重截断 样本能覆盖分布主体 尾部估计失真 结合数据来源判断
无删失 结局完整观测 有删失应改用 Kaplan-Meier 看是否随访删失
§4

手把手算例

对 5 个观测手工画 ECDF：数据 = 2, 4, 4, 6, 9（$n=5$，注意 4 出现两次）。

一步步计算（$F_n(x)$ = ≤x 的个数 / 5）：

$x$ ≤x 的个数 $F_n(x)$
1 0 0.0
2 1 0.2
3 1 0.2
4 3 0.6
6 4 0.8
9 5 1.0
  • 曲线在 2 处跳 $1/5=0.2$，在 4 处一次跳 $2/5=0.4$（因为有两个 4），在 6、9 处各跳 0.2。
  • 读中位数：$F_n$ 首次达到 0.5 的位置在 4（$F_n(4)=0.6$），故样本中位数为 4。

结论： 无需任何分箱或参数，5 个点就唯一确定了这条阶梯。重复值让台阶更高，说明 ECDF 忠实记录了「数据在哪扎堆」。样本越大，台阶越密、越逼近真实的平滑 CDF。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：单个数值变量（连续或离散）。
  • 因变量类型：不区分。
  • 数据结构：一维数值数组。
  • 是否适合高维数据：一维；多维需边际或专门方法。
  • 是否适合缺失较多数据：需先剔除或说明缺失。
  • 是否适合删失数据：不适合，删失用 Kaplan-Meier（它是删失版 ECDF）。
  • 是否适合重复测量数据：混合层级会掩盖结构，宜分层。

5.2 示例表格

$x$ $F_n(x)$
2 0.2
4 0.6
6 0.8
9 1.0

5.3 输入与产出

输入

  • 输入数据：一维数值向量。
  • 关键变量：待描述变量。
  • 需要预处理的内容：排序（软件自动）、处理缺失。

产出

  • 模型对象/统计结果：阶梯函数 $F_n$、各分位数。
  • 参数估计：无（非参数）。
  • 预测结果：任意 $x$ 处的累积比例。
  • 不确定性指标：DKW 置信带。
§6

适用场景

  • 适合：分布探索、分位数读取、两组分布整体比较、拟合检验可视化。
  • 不适合：直接看密度形状（用直方图/KDE）、删失数据（用 KM）。
  • 使用前需要特别检查的点：独立同分布、是否存在删失或截断。
§7

实现

常用包：

  • numpy
  • matplotlib
import numpy as np
import matplotlib.pyplot as plt

x = np.array([2, 4, 4, 6, 9])
xs = np.sort(x)
ys = np.arange(1, len(xs) + 1) / len(xs)   # 每步 1/n
plt.step(xs, ys, where="post")
plt.xlabel("x"); plt.ylabel("F_n(x)"); plt.ylim(0, 1)
plt.show()

常用包：

  • stats
x <- c(2, 4, 4, 6, 9)
plot(ecdf(x), main = "", xlab = "x", ylab = "F_n(x)")
quantile(x, 0.5)          # 中位数
§8

结果如何解读

  • 核心结果看什么：曲线的整体位置与陡缓、各分位数、两组曲线的分离程度。
  • 每个主要参数如何解读：曲线越靠左，分布整体越小；某处越陡，说明数据在该区间越密集。
  • 临床或医学意义如何表达：如「80% 的患者胆固醇在 X 以下」直接从曲线读出。
  • 常见误读：把阶梯的锯齿当分布细节（小样本本就如此）；忽略删失直接画 ECDF。
§9

假设诊断与稳健性

  • 与理论分布对比：把理论 CDF 叠加在 ECDF 上，最大缝隙即 KS 统计量。
  • 置信带：用 DKW 不等式画同时置信带，判断偏离是否显著。
  • 删失识别：若数据有随访删失，ECDF 会低估尾部，应改用 Kaplan-Meier。
  • 稳健性：ECDF 对异常值天然稳健（一个极端值只贡献 $1/n$ 的跳跃）。
§10

推荐可视化

  • 单组 ECDF 阶梯图（读分位数）。
  • 两组 ECDF 叠加（比较整体分布、标注最大缝隙）。
  • ECDF + DKW 置信带。

§11

优势、局限与常见坑

优势

  • 无参数、无平滑、无信息损失，唯一确定。
  • 对异常值稳健，便于读分位数。
  • 是分布比较与拟合检验的天然工具。

局限

  • 阶梯不平滑，看密度形状不直观。
  • 一维为主。
  • 有删失时不能直接用。

常见坑

  • 用 ECDF 判断「密度峰」（应看直方图/KDE）。
  • 小样本时过度解读锯齿。
  • 忽略删失导致尾部偏差。
§12

与相近方法的区别

  • 和直方图：直方图估密度、要分箱；ECDF 估累积分布、不分箱。
  • 和核密度估计：KDE 平滑估密度；ECDF 是密度的积分（累积）估计。
  • 和 Kaplan-Meier 曲线：KM 是能处理右删失的「生存版 ECDF」。
  • 如何选择：看累积比例/分位数/分布比较用 ECDF；看形状用直方图/KDE；有删失用 KM。
§13

医学研究中的典型应用

  • 比较治疗组与对照组某连续指标的整体分布。
  • 报告「某阈值以下的患者比例」。
  • 作为 KS 检验与分位数分析的可视化基础。
§14

关键术语

  • 累积分布函数（CDF）：$F(x)=P(X\le x)$，随机变量不超过 $x$ 的概率。
  • 经验分布函数（ECDF）：用样本频率估计的 CDF，阶梯函数。
  • 指示函数（$\mathbf{1}\{\cdot\}$）：条件成立取 1、否则取 0。
  • Glivenko-Cantelli 定理：ECDF 一致收敛到真 CDF 的定理。
  • DKW 不等式：给 ECDF 同时置信带宽度的不等式。
§15

相关方法

  • 直方图（Histogram）
  • 核密度估计（Kernel Density Estimation, KDE）
  • Kolmogorov-Smirnov检验（Kolmogorov-Smirnov Test）
  • Kaplan-Meier生存曲线（Kaplan-Meier Estimator）
