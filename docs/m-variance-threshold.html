状态 · ◑ 已建 难度 · basic 标签 · 医学统计 / 数据科学 / 特征选择 / 数据预处理

方差阈值法（Variance Threshold）

§1

方法概览

1.1 一句话本质

方差阈值法删除在训练样本中几乎不变化的特征，因为一个近似常数很难区分样本，却会增加存储、计算和数值负担。

1.2 定义

方差阈值法是一种无监督过滤式特征选择：逐列计算离散程度，保留方差高于预设阈值的列。它不看结局，因此只能判断“有没有变化”，不能判断“变化是否与结局有关”。

1.3 它主要解决什么问题

  • 清除常数列、近零方差哑变量和流水线遗留空特征。
  • 降低高维矩阵尺寸，避免无信息列干扰后续算法。
  • 医学场景：组学质控、one-hot 编码后稀有类别、跨中心数据字典中的恒定字段。

1.4 直觉与类比

若所有患者的“本中心代码”都为 1，这一列无法帮助区分患者；它像一把永远指向同一刻度的尺。方差阈值先把这种尺清走，但不会判断另一把变化很大的尺是否真的测到了疾病。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

自动合并数据、哑变量编码和组学预处理常产生大量恒定或极稀疏列。它们不提供样本间排序，却可能导致矩阵奇异、距离被无意义维度稀释或训练变慢。

2.2 关键洞察

特征对区分样本的最低必要条件是“至少有一定变化”。因此先用与模型无关的廉价统计量做粗筛，再把监督选择交给后续步骤。它是数据卫生规则，不是最终变量选择结论。

2.3 与朴素/相邻做法的对比

  • 删除常数列对应阈值 0，是最安全的版本。
  • 近零方差规则常同时考虑唯一值比例和最常见/次常见频数比，比单纯方差更适合类别变量。
  • 标准化后所有非恒定连续变量方差约为 1，因此方差筛选必须在标准化前拟合。
§3

数学形式

3.1 核心公式

总体形式的样本方差为

$$ s_j^2=\frac{1}{n}\sum_{i=1}^{n}(x_{ij}-\bar x_j)^2 $$

选择规则：

$$ \mathcal S=\{j:s_j^2>\tau\} $$

这个式子在说：只保留跨样本波动超过阈值 $\tau$ 的列。

二元特征若阳性比例为 $p$，方差为

$$ \operatorname{Var}(X)=p(1-p) $$

3.2 推导脉络

常数列所有偏差均为 0，因此方差为 0。二元特征在 $p=0.5$ 时方差最大 0.25，越接近全 0 或全 1 方差越小。由此可把“极稀有”转成方差阈值，但阈值依赖编码和尺度。

3.3 参数与统计量含义

  • $s_j^2$：第 $j$ 个特征在训练集中的波动。
  • $\tau$：保留阈值；不是普适常数，应结合尺度和任务。
  • $p$：二元特征阳性比例。
  • support mask：每列保留/删除的布尔结果。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
训练集代表部署分布 低方差状态在未来也相近 外部中心可能出现重要变化 分中心频数/方差
列尺度可比较 同一阈值具有共同意义 大单位变量天然更易保留 在原始语义尺度设规则
稀有不等于关键 极低频特征可安全粗筛 罕见病标志被误删 与临床白名单联用
选择仅在训练折拟合 测试分布不参与筛选 轻微信息泄漏 pipeline/嵌套验证
§4

手把手算例

6 名患者有三列：

患者 constant rare_flag marker
1–5 1 0 1,2,3,4,5
6 1 1 6

constant：

$$ s^2=0 $$

rare_flag： $p=1/6$，

$$ s^2=p(1-p)=\frac16\frac56=\frac5{36}=0.139 $$

marker： 均值 3.5，

$$ s^2=\frac{(-2.5)^2+\cdots+(2.5)^2}{6} =\frac{17.5}{6}=2.917 $$

阈值 $\tau=0.1$ 时删除 constant，保留 rare_flag 与 marker；若 $\tau=0.15$，rare_flag 也被删除。后一个决定可能删掉罕见但重要的临床信号，所以应由语境决定。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 数值化表格或稀疏高维矩阵。
  • 结局类型不限，因为方法不看结局。
  • 缺失值需先定义处理方式；NA 不能简单当作一个数值。
  • 不处理删失、混杂或重复测量相关性。

5.2 示例表格

patient_id site_B rare_variant biomarker
P01 0 0 1.2
P02 0 0 2.1
P03 0 1 3.4

5.3 输入与产出

输入训练特征矩阵与阈值；产出保留列、删除列、列方差和可复用转换器。不会产生结局关联或因果效应。

§6

适用场景

  • 适合：流水线第一道粗筛、常数/近常数清理。
  • 不适合：尺度差异巨大却共用阈值，或罕见信号具有明确临床意义。
  • 应与监督选择、临床白名单及外部中心检查结合。
§7

实现

import pandas as pd
from sklearn.feature_selection import VarianceThreshold

X = pd.DataFrame({
    "constant": [1, 1, 1, 1, 1, 1],
    "rare_flag": [0, 0, 0, 0, 0, 1],
    "marker": [1, 2, 3, 4, 5, 6],
})
selector = VarianceThreshold(threshold=0.1)
Xt = selector.fit_transform(X)  # 只在训练折 fit
print(pd.Series(selector.variances_, index=X.columns))
print(X.columns[selector.get_support()].tolist())

library(recipes)

train <- data.frame(
  constant = rep(1, 6),
  rare_flag = c(0, 0, 0, 0, 0, 1),
  marker = 1:6
)
rec <- recipe(~ ., data = train) |>
  step_zv(all_predictors()) |>
  prep(training = train)
bake(rec, new_data = train)
§8

结果如何解读

“删除”只表示训练集波动未过阈值，不表示变量在生物学上无意义。应报告阈值、方差定义、筛选前后列数和被白名单保留的例外。

§9

假设诊断与稳健性

  • 在交叉验证各折记录保留频率；不稳定列提示阈值敏感。
  • 比较多个合理阈值下的下游性能。
  • 按中心、批次和亚组计算方差，排查局部恒定。
  • 对稀有病、药物暴露和变异位点使用临床白名单。
§10

推荐可视化

  • 对数尺度方差排序图与阈值线。
  • 二元特征患病率/阳性率直方图。
  • 各折保留频率热图。
  • 中心 × 特征方差热图。
§11

优势、局限与常见坑

优势

  • 快速、无模型、适合高维稀疏矩阵。
  • 不使用标签，减少直接结局泄漏。

局限

  • 完全忽略结局和特征交互。
  • 对单位、编码、中心构成敏感。

常见坑

  • 标准化后才做筛选。
  • 在全数据上 fit。
  • 把低频等同无用。
  • 对 one-hot 各列使用不合语义的统一阈值。
§12

与相近方法的区别

  • 单变量选择看特征—结局关联；方差阈值不看结局。
  • 相关过滤删除特征间冗余；方差阈值只看单列。
  • 如何选择：先删除恒定列，再在训练流水线内做监督或冗余筛选。
§13

医学研究中的典型应用

  • 删除单中心数据中恒定设备/站点字段。
  • 清理组学低变异探针。
  • 删除 one-hot 后几乎从不出现的类别，但罕见病特征需保护。
§14

关键术语

  • 常数特征（Constant feature）：所有样本取值相同。
  • 近零方差（Near-zero variance）：只有极少数样本不同。
  • 过滤式选择（Filter method）：不依赖最终模型的预筛。
  • 支持掩码（Support mask）：标记保留列的布尔向量。
  • 数据泄漏（Leakage）：测试信息参与预处理拟合。
