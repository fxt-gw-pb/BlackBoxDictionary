限制性立方样条

Restricted Cubic Splines, RCS
状态 · ◑ 已建 难度 · intermediate 标签 · 医学统计 / 数据科学 / 非线性 / 剂量反应 / 样条

限制性立方样条（Restricted Cubic Splines, RCS）

§1

方法概览

1.1 一句话本质

限制性立方样条把连续变量切成几段，每段用三次曲线平滑拟合、段与段在结点处光滑衔接，并强制两端呈直线——既能灵活刻画弯曲关系，又不像高次多项式那样在两端狂摆。

1.2 定义

限制性立方样条是一种把连续协变量展开为若干基函数的非线性建模技术：结点之间用三次多项式，两端（边界结点外）约束为线性，可嵌入线性回归、Logistic、Cox 等模型。

1.3 它主要解决什么问题

  • 研究问题：某连续暴露与结局是否非线性（U 形、阈值、饱和），形状如何？
  • 适用任务：非线性剂量反应建模、连续协变量的灵活调整、非线性检验。
  • 常见医学场景：BMI 与死亡风险的 U 形、年龄效应、生物标志物阈值、Cox 模型中连续暴露的非线性风险。

1.4 直觉与类比

想用一根柔性的绘图曲线尺沿散点描边。多项式尺是「一整根硬尺」，你想让它中间弯，两端就会翘得很厉害。限制性立方样条像一根分段可弯、但两端被压平的尺子：在数据密集的中间灵活贴合，在数据稀疏的两端保持直线，避免尾部乱翘导致的荒谬外推。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

直接放连续变量假设它是直线，容易漏掉真实弯曲；高次多项式虽能弯，但两端剧烈震荡（龙格现象）、外推荒谬；分箱又丢失连续信息、结果依赖切点。根本困难是：如何灵活刻画非线性，又稳健、不在稀疏尾部失控？

2.2 关键洞察

用分段低次（三次）多项式代替全局高次：在少数结点之间各用三次曲线，并施加连续性约束（函数值、一阶、二阶导在结点处都连续），使拼接处光滑无折角。关键的「限制」是在两端边界结点之外强制线性——这消除了尾部的高次摆动，是它比普通三次样条稳健的原因。把变量展开成 $k-2$ 个样条基后，仍是普通回归，只是多了几列特征；结点数（3–5）与位置（取分位数）决定灵活度。

2.3 与朴素/相邻做法的对比

  • 相对多项式回归：样条分段局部、两端线性，尾部稳；多项式全局、尾部乱摆。
  • 相对分箱/分类：样条保留连续信息，不因任意切点丢功效、造阶梯。
  • 相对 LOWESS：样条给可推断的参数模型（能算 OR/HR 曲线与检验），LOWESS 偏探索。
§3

数学形式

3.1 核心公式

给定结点 $t_1\lt\dots\lt t_k$，把 $x$ 展开为 $k-1$ 个项，模型的线性预测子为：

$$ g\big(E[Y\mid x]\big)=\beta_0+\beta_1 x+\sum_{j=1}^{k-2}\gamma_j\,s_j(x) $$

其中 $s_j(x)$ 是由结点构造、两端线性的三次样条基。这个式子在说：先放一个线性项，再加几个「样条基」项去刻画弯曲；$g$ 随结局类型取恒等/logit/log，从而嵌入线性、Logistic、Cox 模型。

3.2 推导脉络

  • 三次样条：分段三次多项式，在结点处 0/1/2 阶导连续 → 光滑拼接。
  • 「限制/自然」条件：边界结点外二阶、三阶导为 0，即两端线性，减少边界方差。
  • 把 $x$ 变换成 $k-2$ 个基函数列，代入所在回归模型用常规方法（OLS/MLE）估计。
  • 非线性检验：联合检验所有 $\gamma_j=0$（Wald/似然比）；不显著则可退回线性。

3.3 参数与统计量含义

  • $t_1,\dots,t_k$：结点，常取分位数（Harrell 建议 4 结点取 5/35/65/95 百分位）。
  • $\beta_1,\gamma_j$:样条系数，单个无直接意义，需整体解读为效应曲线。
  • 非线性检验 p 值：是否存在非线性。
  • $g$：连接函数，决定嵌入哪种模型。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
结点合理 数量/位置得当 过拟合或欠拟合 敏感性、交叉验证
沿用所在模型假设 如 Cox 的 PH 相应偏差 对应诊断
看整体形状 不解读单系数 误读 画效应曲线
§4

手把手算例

以「在 Logistic 模型里用 RCS 刻画 BMI 与患病风险的非线性」为例，走一遍工作流（样条基的逐项手算不实用，重点是结点、自由度与非线性检验）。

一步步走：

  • 选结点：$n$ 较大时取 4 个结点，位置放在 BMI 的 5/35/65/95 百分位（如 18.5 / 24 / 28 / 35）。
  • 数自由度：4 结点 → 样条项 $k-2=2$ 个，加上线性项，共 3 个自由度（比 2 次多项式多一点灵活度、但尾部线性）。
  • 拟合与检验：拟合 logit(P) = β0 + β1·BMI + γ1·s1 + γ2·s2；联合检验 $H_0:\gamma_1=\gamma_2=0$。若 $p=0.002$，则存在显著非线性。
  • 读结果：不看单个 $\gamma$，而是画「OR（相对参考 BMI=24）随 BMI 变化」的效应曲线——例如呈 U 形：BMI 24 附近风险最低，向两侧升高。

结论： RCS 的价值不在某个系数，而在整条效应曲线 + 一个非线性检验。本例用 3 个自由度就捕捉到 U 形关系，且因两端线性，BMI=15 或 45 处的外推不会像高次多项式那样发散。报告时务必给出参考值（如 BMI=24）与置信带——脱离参考值，样条系数无法解读。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：至少一个连续协变量（做样条展开）。
  • 因变量类型：连续、二分类、计数、时间到事件（依所嵌回归）。
  • 数据结构：标准回归表格。
  • 是否适合高维数据：对少数关键连续变量用，不宜全部展开。
  • 是否适合缺失较多数据：先处理缺失。
  • 是否适合删失数据：嵌入 Cox 时适用。
  • 是否适合重复测量数据：可嵌入混合模型。

5.2 示例表格

结点数 k 样条项 (k−2) 总自由度 结点位置(百分位)
3 1 2 10/50/90
4 2 3 5/35/65/95
5 3 4 5/27.5/50/72.5/95

5.3 输入与产出

输入

  • 输入数据：连续协变量 + 结局 + 其他调整变量。
  • 关键变量：待展开变量、结点数与位置、参考值。
  • 需要预处理的内容：确定结点（分位数）、设定参考值。

产出

  • 模型对象/统计结果：样条系数、非线性检验 p 值。
  • 参数估计:样条基系数（整体解读）。
  • 预测结果：效应形状曲线（相对参考值的 OR/HR）。
  • 不确定性指标：效应曲线的置信带。
§6

适用场景

  • 适合：怀疑连续暴露非线性、需灵活又稳健地刻画形状。
  • 不适合：关系确为线性、样本太小、需要单一斜率解释时。
  • 使用前需要特别检查的点：结点数/位置、是否过拟合、参考值设定。
§7

实现

常用包：

  • patsy
  • statsmodels
import statsmodels.formula.api as smf

# cr() 为自然(限制性)三次样条基, df 控制结点数
fit = smf.logit("event ~ cr(BMI, df=3) + age", data=df).fit()
print(fit.summary())

常用包：

  • rms
library(rms)
dd <- datadist(df); options(datadist = "dd")
fit <- lrm(event ~ rcs(BMI, 4) + age, data = df)  # 4 个结点
anova(fit)                     # 含非线性检验
plot(Predict(fit, BMI, ref.zero = TRUE))   # 效应曲线
§8

结果如何解读

  • 核心结果看什么：效应形状曲线、非线性检验 p、是否有阈值/拐点。
  • 每个主要参数如何解读：单个样条系数无意义，看整条曲线相对参考值的 OR/HR。
  • 临床或医学意义如何表达：如「BMI 在 24 附近风险最低，两侧升高，呈 U 形」。
  • 常见误读：解读单个基系数；把非线性显著等同于临床重要。
§9

假设诊断与稳健性

  • 结点敏感性：换 3/4/5 结点看曲线形状是否稳健。
  • 过拟合：结点过多或样本小易过拟合，用交叉验证/AIC 节制。
  • 外推：两端线性已缓解，但仍避免推到数据稀疏区。
  • 沿用所在模型诊断：如嵌 Cox 需查比例风险。
§10

推荐可视化

  • 效应形状曲线（含 95% 置信带、标注参考值）。
  • 结点位置标注。
  • 与线性拟合叠加对比。
§11

优势、局限与常见坑

优势

  • 灵活刻画非线性且两端稳健，避免多项式尾部摆动。
  • 可嵌入线性/Logistic/Cox 等多种模型。
  • 提供正式的非线性检验。

局限

  • 结点选择带主观性。
  • 需图形化解读，不如单斜率直观。
  • 结点过多易过拟合。

常见坑

  • 结点过多导致过拟合与不稳。
  • 外推到数据稀疏尾部。
  • 不报参考值，使 OR/HR 曲线无法解读。
§12

与相近方法的区别

  • 和多项式回归：多项式全局、尾部易摆；RCS 分段且两端线性更稳。
  • 和局部加权回归：LOWESS 探索性；RCS 给可推断的参数模型。
  • 和分箱/分类:样条保留连续信息，避免任意切点损失。
  • 如何选择：需推断+非线性 → RCS；纯探索 → LOWESS；轻度弯曲 → 低次多项式。
§13

医学研究中的典型应用

  • BMI、血压、生物标志物与结局的非线性剂量反应。
  • Cox 模型中连续暴露的非线性风险刻画。
  • 年龄等连续混杂因素的灵活调整。
§14

关键术语

  • 结点（Knot）:样条分段的连接点，常取分位数。
  • 分段三次多项式：每段用三次曲线拟合。
  • 自然/限制样条：两端强制线性的样条。
  • 非线性检验：联合检验样条项是否为 0。
  • 参考值：解读效应曲线（OR/HR）的基准点。
§15

相关方法

  • 多项式回归（Polynomial Regression）
  • 局部加权回归（Locally Weighted Regression）
  • Cox比例风险模型（Cox Proportional Hazards Model）
