01·01
样本量与检验效能 Sample Size and Power Analysis

样本量估计是在收集数据之前，把「想检出的效应大小、数据的变异、可接受的假阳性率、想要的把握度」四者放进同一个等式，反解出「需要多少人」。

01·02
直方图 Histogram

直方图把连续变量的取值范围切成一排等宽格子，数一数每格里落了多少观测——它其实是密度函数最朴素的「分段常数」估计。

◑ 已建 basic
01·03
经验分布函数 Empirical Cumulative Distribution Function, ECDF

经验分布函数是「用数据本身当作分布」：在每个观测值处向上跳一小步 $1/n$，画出「有多大比例的数据不超过 $x$」这条阶梯曲线，它是真实累积分布函数（CDF）最诚实的估计。

◑ 已建 basic
01·04
核密度估计 Kernel Density Estimation, KDE

核密度估计给每个观测放上一个小小的、光滑的「山包」（核），再把所有山包叠加平均，得到一条连续平滑的密度曲线——本质是把直方图的硬台阶抹圆。

◑ 已建 intermediate
01·05
Kolmogorov-Smirnov检验 Kolmogorov-Smirnov Test

KS 检验用「两条累积分布曲线之间的最大垂直距离」作为分布是否相同的证据——距离越大，越有理由说它们来自不同分布。
