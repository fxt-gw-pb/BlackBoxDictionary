样本量与检验效能 Sample Size and Power Analysis

样本量与检验效能分析在研究开始前，根据预期效应、显著性水平和目标把握度，计算需要多少样本，或评估给定样本量下能检出多大效应。

01·02
直方图 Histogram

直方图把连续变量的取值范围切成若干等宽区间（bin），统计每个区间内的观测数（或频率），用相邻柱形展示数据的分布形状。

◑ 已建 basic
01·03
经验分布函数 Empirical Cumulative Distribution Function, ECDF

经验分布函数是对总体分布函数 $F(x)=P(X\le x)$ 的非参数估计。它直接用样本中“不大于某个阈值的比例”来近似总体累计概率。

◑ 已建 basic
01·04
核密度估计 Kernel Density Estimation, KDE

核密度估计是一种用局部加权平均来估计连续型变量概率密度函数的非参数方法。它常用来替代粗糙的直方图，给出更平滑的分布形状。

◑ 已建 basic
01·05
Kolmogorov-Smirnov检验 Kolmogorov-Smirnov Test

Kolmogorov-Smirnov 检验是一种基于经验分布函数的非参数检验，用经验分布与理论分布（单样本）或两组经验分布之间的最大距离，判断分布是否一致。
