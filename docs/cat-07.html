07·01
决策树 Decision Tree

决策树把复杂分类问题拆成一连串「如果满足某条件，就往左；否则往右」的问题，最后在叶节点用多数类别做预测。

07·02
ID3决策树 ID3 Decision Tree

ID3 每一步都选择「最能减少类别不确定性」的离散特征来分裂节点。

◑ 已建 intermediate
07·03
C4.5决策树 C4.5 Decision Tree

C4.5 是 ID3 的实用升级版：它用信息增益率惩罚「把数据切得过碎」的特征，并支持连续变量、缺失处理和剪枝。

◑ 已建 intermediate
07·04
决策树回归 Decision Tree Regression

决策树回归把连续结局分成一块块区域，每个叶节点用该区域的平均值作为预测。

◑ 已建 basic
07·05
Bagging算法 Bootstrap Aggregating

Bagging 让许多个「看过不同重抽样训练集」的模型各自预测，再投票或平均，以抵消单个模型的偶然波动。

◑ 已建 basic
07·06
随机森林 Random Forest

随机森林是在 Bagging 多棵树的基础上，再让每个节点只看一部分随机特征，从而让树与树更不相似、投票更稳。

◑ 已建 basic
07·07
随机森林回归 Random Forest Regression

随机森林回归训练许多带随机性的回归树，再把它们的连续预测取平均，得到比单棵树更稳定的预测值。

◑ 已建 basic
07·08
极端随机树 Extremely Randomized Trees

极端随机树比随机森林更随机：每个节点不仅随机抽特征，还随机抽切分阈值，再从随机候选中挑一个较好的。

◑ 已建 intermediate
07·09
Boosting算法 Boosting

Boosting 让弱学习器依次上场，每一轮都针对当前模型尚未解决的错误做一次小修正，最后把这些修正相加成强模型。

◑ 已建 intermediate
07·10
AdaBoost Adaptive Boosting

AdaBoost 通过提高错分样本的权重，让后一轮弱分类器更关注“难题”，最后按弱分类器质量加权投票。

◑ 已建 intermediate
07·11
梯度提升决策树 Gradient Boosting Decision Tree, GBDT

GBDT 把“拟合残差”推广为“拟合损失函数的负梯度”，让浅决策树在函数空间中逐步走向更低损失。

◑ 已建 intermediate
07·12
梯度提升回归 Gradient Boosting Regression

梯度提升回归用一连串浅回归树逐步拟合当前损失的负梯度，从而不断修正连续结局预测。

◑ 已建 intermediate
07·13
XGBoost Extreme Gradient Boosting, XGBoost

XGBoost 是带二阶近似、显式树复杂度惩罚、采样与高效实现的梯度提升树。

◑ 已建 intermediate
07·14
LightGBM Light Gradient Boosting Machine

LightGBM 用直方图近似分裂搜索，并优先分裂全树中收益最大的叶节点，以较低内存和较高速度训练梯度提升树。

◑ 已建 intermediate
07·15
投票集成 Voting Ensemble

投票集成用固定规则合并多个分类模型：硬投票数类别票，软投票平均经过校准且可比较的类别概率。

◑ 已建 basic
07·16
Stacking集成 Stacked Generalization

Stacking 把多个基础模型的 out-of-fold 预测当作新特征，再训练一个元学习器决定何时、以多大程度相信每个模型。

◑ 已建 intermediate
