Pearson卡方独立性检验

Pearson Chi-Squared Test of Independence
状态 · ◑ 已建 难度 · basic 标签 · 医学统计 / 数据科学 / 分类数据 / 列联表 / 假设检验

Pearson卡方独立性检验（Pearson Chi-Squared Test of Independence）

§1

方法概览

1.1 一句话本质

卡方检验把列联表里「实际观测到的频数」和「假如两变量毫无关系时本该出现的期望频数」逐格相比：两者差得越大，越说明两个分类变量之间存在关联。

1.2 定义

Pearson 卡方独立性检验用于判断两个分类变量是否独立（无关联）。它基于列联表，比较观测频数与独立假设下的期望频数，构造近似服从卡方分布的统计量。

1.3 它主要解决什么问题

  • 研究问题：两个分类特征是否相关（如暴露与疾病、性别与结局）？
  • 适用任务：列联表关联检验、率的组间比较。
  • 常见医学场景：暴露组与非暴露组的患病率比较、不同性别的并发症发生率、分类危险因素与结局的关联。

1.4 直觉与类比

先假设「两件事完全无关」，据此推算「本该」出现的表格（期望频数）。再看真实表格偏离它多远。就像先假设硬币公平、算出正反各半的期望，再看实际抛掷偏离多少。偏离若在随机波动之内，就没有证据说两者相关；偏离若大得离谱，就推翻「无关」的假设。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

列联表里各格频数天然有随机波动。即便两变量真的独立，观测频数也不会精确等于「行列比例相乘」的期望。根本困难是：如何判断观测到的偏离，是真实关联还是抽样噪声？

2.2 关键洞察

「独立」有精确的数学含义：$P(A,B)=P(A)P(B)$，于是每格的期望频数 = 行合 × 列合 ÷ 总数。把每格的偏离标准化（除以期望）再累加，得到卡方统计量 $\sum (O-E)^2/E$——每格贡献与「偏离平方相对期望的大小」成正比。在独立假设与大样本下，这个和近似服从自由度 $(r-1)(c-1)$ 的卡方分布，于是可算 p 值。

2.3 与朴素/相邻做法的对比

  • 相对直接比较各格比例：卡方给出整体的显著性与不确定性。
  • 相对 Fisher 精确检验：卡方是大样本近似，期望频数小时不准，此时用 Fisher。
  • 相对 McNemar：卡方要求两变量的观测相互独立；配对二分类要用 McNemar。
§3

数学形式

3.1 核心公式

$$ \chi^2=\sum_{\text{所有格}}\frac{(O-E)^2}{E},\qquad E_{ij}=\frac{(\text{行}_i\text{合})\times(\text{列}_j\text{合})}{N} $$

自由度 $df=(r-1)(c-1)$。这个式子在说：每格把「观测减期望」平方后除以期望（对偏离标准化），再全表求和；期望频数由「独立即行列比例相乘」给出。

3.2 推导脉络

  • 独立假设 $P(\text{行}_i,\text{列}_j)=P(\text{行}_i)P(\text{列}_j)$，用边际频率估计概率，得 $E_{ij}$。
  • 每格 $O_{ij}$ 近似服从泊松/二项，标准化偏离 $(O-E)/\sqrt{E}$ 近似标准正态。
  • 其平方和在独立假设下近似卡方分布，自由度为 $(r-1)(c-1)$（表格被边际约束后的自由格数）。
  • 大 $\chi^2$（右尾）意味观测系统偏离独立，即拒绝独立假设。$2\times2$ 表小样本可加 Yates 连续性校正。

3.3 参数与统计量含义

  • $O,E$：各格观测、期望频数。
  • $\chi^2$：累积标准化偏离，$\ge 0$。
  • $df=(r-1)(c-1)$：自由度。
  • p 值：独立假设下出现如此大偏离的概率；效应量用 Cramér's V / OR。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
期望频数够大 各格 $E\ge 5$（多数格） 近似失准、p 偏差 看期望频数表；小则用 Fisher
观测独立 每个个体只计一次、互不影响 p 失真 看设计；配对用 McNemar
随机抽样 样本代表总体 外推受限 看抽样方式
§4

手把手算例

比较吸烟与患病的关联（2×2 列联表）：

患病 未患 行合
吸烟 30 70 100
不吸 15 85 100
列合 45 155 200

一步步计算：

  • 期望频数：$E(\text{吸烟,病})=\dfrac{100\times45}{200}=22.5$；同理 $E(\text{吸烟,未})=77.5$，$E(\text{不吸,病})=22.5$，$E(\text{不吸,未})=77.5$。
  • 各格贡献 $(O-E)^2/E$：
  • 吸烟·病：$(30-22.5)^2/22.5=56.25/22.5=2.5$
  • 吸烟·未：$(70-77.5)^2/77.5=56.25/77.5=0.726$
  • 不吸·病：$(15-22.5)^2/22.5=2.5$
  • 不吸·未：$(85-77.5)^2/77.5=0.726$
  • $\chi^2=2.5+0.726+2.5+0.726=6.45$，$df=(2-1)(2-1)=1$。
  • 临界值 $\chi^2_{0.05,1}=3.84$；因 $6.45\gt 3.84$，拒绝独立，即吸烟与患病相关。

结论： 吸烟组患病率 30% 高于不吸组 15%，卡方 6.45 超过临界值，关联显著。注意卡方只说「有关联」，方向与强度要看 比值比 $\mathrm{OR}=\dfrac{30\times85}{70\times15}=\dfrac{2550}{1050}=2.43$——吸烟者患病优势约为不吸者的 2.4 倍。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：两个分类变量。
  • 因变量类型：分类（列联表频数）。
  • 数据结构：$r\times c$ 列联表。
  • 是否适合高维数据：大表需注意稀疏格。
  • 是否适合缺失较多数据：缺失需先处理。
  • 是否适合删失数据：不适合。
  • 是否适合重复测量数据：不适合；配对用 McNemar。

5.2 示例表格

患病 未患
吸烟 30 70
不吸 15 85

5.3 输入与产出

输入

  • 输入数据：列联表或原始分类记录。
  • 关键变量：两个分类变量。
  • 需要预处理的内容：制表、检查期望频数。

产出

  • 模型对象/统计结果：$\chi^2$、自由度、p 值。
  • 参数估计：无（关联强度用 Cramér's V / OR）。
  • 预测结果：无。
  • 不确定性指标：OR/RR 的置信区间（另算）。
§6

适用场景

  • 适合：中大样本的列联表独立性检验、率的比较。
  • 不适合：期望频数小（用 Fisher）、配对数据（用 McNemar）、有序变量趋势（用 Cochran-Armitage）。
  • 使用前需要特别检查的点：期望频数、观测独立、是否配对。
§7

实现

常用包：

  • scipy
import numpy as np
from scipy.stats import chi2_contingency

table = np.array([[30, 70], [15, 85]])
chi2, p, dof, expected = chi2_contingency(table, correction=False)
print(f"chi2={chi2:.2f}, df={dof}, p={p:.4f}")   # chi2≈6.45
print("期望频数:\n", expected)

常用包：

  • stats
table <- matrix(c(30, 70, 15, 85), nrow = 2, byrow = TRUE)
chisq.test(table, correct = FALSE)      # correct=TRUE 加 Yates 校正
§8

结果如何解读

  • 核心结果看什么：$\chi^2$/p，以及关联的方向与强度（OR/RR/Cramér's V）。
  • 每个主要参数如何解读：p 小说明有关联；单看卡方不含方向，需配 OR。
  • 临床或医学意义如何表达：报告率的差异与 OR/RR 区间，而非仅 p。
  • 常见误读：把卡方大小当关联强度（它受样本量影响）；期望频数小仍用卡方。
§9

假设诊断与稳健性

  • 期望频数：查期望频数表，若多格 $\lt 5$（尤其 2×2 有格 $\lt 5$），改用 Fisher。
  • 独立性：确认每个个体只计一次；配对数据改用 McNemar。
  • 大样本敏感：极大样本下微不足道的关联也会显著，务必看效应量。
  • 有序变量：忽略顺序会损失功效，用 Cochran-Armitage 趋势检验。

§11

优势、局限与常见坑

优势

  • 简单通用，适用任意 $r\times c$ 表。
  • 计算与解释门槛低。
  • 与残差分析结合可定位关联来源。

局限

  • 大样本近似，小期望频数失准。
  • 只测关联、不给方向与强度。
  • 对极大样本过度敏感。

常见坑

  • 期望频数小仍用卡方（应 Fisher）。
  • 对配对数据误用（应 McNemar）。
  • 把统计显著当临床重要。
§12

与相近方法的区别

  • 和 Fisher 精确检验：小样本/小期望用 Fisher，大样本用卡方。
  • 和 McNemar：配对二分类用 McNemar。
  • 和 Cochran-Armitage：有序暴露的趋势用 CA。
  • 和 Mantel-Haenszel：需控制分层混杂时用 MH。
§13

医学研究中的典型应用

  • 暴露与疾病的关联初筛。
  • 不同组别并发症/结局率比较。
  • 分类危险因素与二分类结局关联。
§14

关键术语

  • 列联表（Contingency Table）：两分类变量的频数交叉表。
  • 期望频数（$E$）：独立假设下每格应有的频数。
  • 卡方统计量（$\chi^2$）：标准化偏离的累加。
  • 自由度：$(r-1)(c-1)$。
  • Cramér's V：从卡方导出的关联强度（0–1）。
  • Yates 校正：2×2 表的连续性校正。
§15

相关方法

  • Fisher精确检验（Fisher Exact Test）
  • McNemar检验（McNemar Test）
  • Mantel-Haenszel检验（Mantel-Haenszel Test）
  • Logistic回归（Logistic Regression）
