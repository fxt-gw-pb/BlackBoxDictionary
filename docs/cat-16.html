16·01
K近邻算法 K-Nearest Neighbors, KNN

KNN 不学习显式方程，而是在标准化后的特征空间里寻找最相似的 $K$ 个历史样本，用局部多数票决定新样本类别。

16·02
K近邻回归 K-Nearest Neighbors Regression

K 近邻回归把新样本的连续结局估计为其局部邻域中已观察结局的平均或距离加权平均。

◑ 已建 basic
16·03
朴素贝叶斯 Naive Bayes

朴素贝叶斯把每项特征提供的类条件证据相乘，再与类别先验结合，选择后验概率最大的类别。

◑ 已建 basic
16·04
线性判别分析 Linear Discriminant Analysis, LDA

LDA 寻找“类间中心尽量远、类内样本尽量紧”的线性投影，并在共享协方差假设下形成线性分类边界。

◑ 已建 intermediate
16·05
感知机 Perceptron

感知机从一个线性边界出发，每遇到误分类样本就把边界朝纠正该样本的方向移动。

◑ 已建 basic
