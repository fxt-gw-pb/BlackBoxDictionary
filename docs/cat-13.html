收录条目
13·01
方差阈值法 Variance Threshold

方差阈值法删除在训练样本中几乎不变化的特征，因为一个近似常数很难区分样本，却会增加存储、计算和数值负担。

◑ 已建 basic
13·02
单变量特征选择 Univariate Feature Selection

单变量特征选择让每个特征单独参加一次“与结局的考试”，再按得分或校正后的显著性筛选，但看不到联合效应和交互。

◑ 已建 basic
13·03
相关系数特征选择 Correlation-based Feature Selection

相关过滤把高度同步变化的特征视为重复测量，在每组冗余变量中保留代表，以降低共线性与重复信息。

◑ 已建 basic
13·04
信息增益 Information Gain

信息增益衡量知道某特征后，类别的不确定性平均减少了多少；减少越多，特征越能把不同类别分开。

◑ 已建 intermediate
13·05
互信息特征选择 Mutual Information Feature Selection

互信息衡量知道一个特征后，结局的不确定性平均减少多少，因此能发现不限于线性或均值差异的边际依赖。

◑ 已建 intermediate
13·06
嵌入式特征选择 Embedded Feature Selection

嵌入式选择把“哪些变量留下”写进模型训练目标或结构，让预测拟合与特征取舍同时完成。

◑ 已建 intermediate
13·07
基于树模型的特征选择 Tree-based Feature Selection

树模型选择按特征在分裂中减少多少预测误差或打乱后损失多少性能来排序，能捕捉非线性与交互，但重要性会受基数和相关性偏倚。

◑ 已建 intermediate
13·08
多重插补 Multiple Imputation

多重插补不假装缺失值有唯一答案，而是生成多份合理完整数据，分别分析，再把“数据内不确定性”和“不同填法之间的不确定性”一起合并。

◑ 已建 intermediate
