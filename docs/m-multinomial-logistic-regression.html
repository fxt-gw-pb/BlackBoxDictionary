多项Logistic回归（Multinomial Logistic Regression）

§1

方法概览

1.1 一句话本质

多项 Logistic 把二分类 Logistic 推广到三类以上无序结局：选一类作参照，对其余每一类各建一条「该类 vs 参照类」的 logit，用 softmax 保证所有类别概率之和为 1。

1.2 定义

多项 Logistic 回归用于无序多分类结局，通过对每个非参照类别相对参照类别建立 logit 模型，估计协变量对各类别相对概率的影响，属 GLM 家族的多分类扩展。

1.3 它主要解决什么问题

  • 研究问题：协变量如何影响个体落入若干无自然顺序类别中的哪一个？
  • 适用任务：无序多分类结局建模、相对风险比估计、多类别概率预测。
  • 常见医学场景：疾病亚型（无序）、多种治疗方案选择、就诊去向/结局分类。

1.4 直觉与类比

若结局是几个平行、没有大小之分的类别（如乳腺癌的分子亚型），就像面前有几扇并排的门。以其中一扇为「参照门」，逐一比较「走进其他每扇门相对参照门」的倾向。每个协变量因此对每扇非参照门各有一组系数，softmax 再把这些倾向归一成加起来等于 1 的概率。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

结局是三类以上、且无顺序（用有序模型不合适）时，二分类 Logistic 不够用。根本困难是：如何把 Logistic 的「对数优势线性建模」推广到多个互斥类别，并保证输出是一组和为 1 的概率？

2.2 关键洞察

固定一个参照类别，对每个非参照类别 $j$ 建「$j$ vs 参照」的对数优势 $\log\frac{P(Y=j)}{P(Y=J)}=\beta_j^\top x$——这就把多分类拆成 $J-1$ 条二分类式的 logit。反解并归一化得到 softmax，天然满足所有概率非负、和为 1。参照类别的选择只影响系数的呈现（都是「相对参照」），不影响拟合出的概率。

2.3 与朴素/相邻做法的对比

  • 相对二分类 Logistic：推广到多类别，输出多个类别概率。
  • 相对有序 Logistic：多项不假设顺序、更灵活但参数更多；有序利用顺序更省参数。
  • 相对逐一「一对多」分别拟合：多项模型联合估计、概率自动归一，比拼凑多个二分类模型更连贯。
§3

数学形式

3.1 核心公式

以类别 $J$ 为参照，对 $j=1,\dots,J-1$：

$$ \log\frac{P(Y=j\mid x)}{P(Y=J\mid x)}=\beta_j^\top x,\qquad P(Y=j\mid x)=\frac{\exp(\beta_j^\top x)}{1+\sum_{k=1}^{J-1}\exp(\beta_k^\top x)} $$

这个式子在说：每个非参照类别相对参照类别的对数优势是线性的；softmax 把这些线性预测子转成一组和为 1 的概率。

3.2 推导脉络

  • 对每个非参照类别建 baseline-category logit，共 $J-1$ 组系数。
  • 反解概率并强制归一，得到 softmax 形式（参照类别的线性预测子取 0）。
  • 用最大似然联合估计所有 $\beta_j$。
  • $\exp(\beta_j)$ 是「相对参照类别」的相对风险比（RRR）。

3.3 参数与统计量含义

  • $\beta_j$：类别 $j$ 相对参照的系数向量。
  • $\exp(\beta_{j})$：相对风险比 RRR。
  • 参照类别：只影响系数呈现，不影响拟合概率。
  • $J$：无序类别数。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
类别无序互斥 无自然顺序、不重叠 用错模型/损失效率 领域判断
观测独立 无相关结构 SE 失真 看设计
IIA 加减类别不改其余相对比 相对比估计偏 Hausman-McFadden 检验
§4

手把手算例

3 个无序类别 A、B、C（C 为参照）。对某协变量取值的个体，其非参照类别线性预测子 $\eta_A=0.5$、$\eta_B=-0.5$（参照 C 的 $\eta_C=0$）。用 softmax 算三类概率。

一步步计算：

  • 指数化：$e^{\eta_A}=e^{0.5}=1.649$，$e^{\eta_B}=e^{-0.5}=0.607$，$e^{\eta_C}=e^{0}=1$。
  • 分母（归一化常数）：$1.649+0.607+1=3.256$。
  • $P(A)=1.649/3.256=0.506$；$P(B)=0.607/3.256=0.186$；$P(C)=1/3.256=0.307$。
  • 校验：$0.506+0.186+0.307=1.00$ ✓。

结论： 该个体最可能属类别 A（概率 0.51）。系数含义：若 A 相对 C 的某协变量系数 $\beta=0.5$，则 $\exp(0.5)=1.65$——该协变量每升 1 单位，「属 A 相对属 C」的相对概率增大约 65%。softmax 保证三类概率自动加起来等于 1，这正是多分类概率建模的关键；参照类别 C 只是「量尺原点」，换个参照不改变这组概率、只改变系数怎么写。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：连续、二分类、多分类均可。
  • 因变量类型：无序多分类（≥3 类）。
  • 数据结构：每行一个体，含多分类结局与协变量。
  • 是否适合高维数据：需正则化。
  • 是否适合缺失较多数据：需先处理。
  • 是否适合删失数据：不适合。
  • 是否适合重复测量数据：需多分类混合模型。

5.2 示例表格

类别 线性预测子 η exp(η) 概率
A 0.5 1.649 0.506
B −0.5 0.607 0.186
C(参照) 0 1.000 0.307

5.3 输入与产出

输入

  • 输入数据：无序多分类结局 + 协变量。
  • 关键变量：类别集合、参照类别、协变量矩阵。
  • 需要预处理的内容：指定参照类别、编码、检查各类样本量。

产出

  • 模型对象/统计结果：各非参照类别系数与 RRR。
  • 参数估计：$\beta_j$。
  • 预测结果：各类别概率、最可能类别。
  • 不确定性指标：系数标准误、RRR 区间。
§6

适用场景

  • 适合：无序、平行的多分类结局。
  • 不适合：有序结局（用有序 Logistic 更省参数）、二分类（用普通 Logistic）。
  • 使用前需要特别检查的点：结局是否真无序、各类别样本是否充足、IIA 是否合理。
§7

实现

常用包：

  • statsmodels
import statsmodels.api as sm

X = sm.add_constant(df[["age", "marker"]])
mod = sm.MNLogit(df["subtype_code"], X).fit(disp=False)  # 结局编码 0..J-1
print(mod.summary())          # 各非参照类别系数

常用包：

  • nnet
library(nnet)
df$subtype <- relevel(factor(df$subtype), ref = "LuminalA")
fit <- multinom(subtype ~ age + marker, data = df)
exp(coef(fit))                # 相对风险比 RRR
§8

结果如何解读

  • 核心结果看什么：每个非参照类别相对参照的 RRR、方向与区间。
  • 每个主要参数如何解读：RRR>1 表示协变量升高使「属该类相对参照」的相对概率增大。
  • 临床或医学意义如何表达：如「标志物阳性者相对 LuminalA，更可能为 Basal 亚型」。
  • 常见误读：把 RRR 当绝对概率；忽视参照类别改变会改变系数呈现。
§9

假设诊断与稳健性

  • IIA：Hausman-McFadden 或 small-Hsiao 检验；违反时考虑嵌套 Logit/多项 probit。
  • 稀疏类别：某类样本太少估计不稳，可合并或收集更多数据。
  • 完全分离：某协变量完美区分某类时系数发散，用惩罚似然。
  • 参照选择：报告时明确参照类别，便于复现。
§10

推荐可视化

  • 各类别 RRR 的分面森林图。
  • 协变量变化下各类别预测概率曲线。
  • 预测概率的堆叠面积图。
§11

优势、局限与常见坑

优势

  • 直接建模无序多分类，输出各类别概率。
  • 系数可解释为相对风险比。
  • 软件成熟、易扩展。

局限

  • 参数随类别数线性增多，需较大样本。
  • 依赖 IIA 假设。
  • 忽略结局顺序（若有序则低效）。

常见坑

  • 对有序结局误用多项模型，损失效率。
  • 某些类别样本太少致估计不稳。
  • 不声明参照类别，结果难复现。
§12

与相近方法的区别

  • 和二分类 Logistic：推广到多分类。
  • 和有序 Logistic：有序利用顺序更省参数；多项更灵活。
  • 和多层感知机分类器：多项 Logistic 线性可解释；神经网络非线性、可解释性弱。
  • 如何选择：无序多类 → 多项 Logistic；有序 → 有序 Logistic；二类 → Logistic。
§13

医学研究中的典型应用

  • 肿瘤分子亚型（无序）的危险因素分析。
  • 多种治疗方案选择的影响因素。
  • 多类别结局去向（治愈/转院/失访等）建模。
§14

关键术语

  • 参照类别（Baseline Category）：作比较基准的类别。
  • 相对风险比（RRR）：$\exp(\beta_j)$，相对参照类别的比值。
  • softmax：把线性预测子转为和为 1 的概率的函数。
  • IIA：无关方案独立性假设。
  • baseline-category logit：多项 Logistic 的建模形式。
