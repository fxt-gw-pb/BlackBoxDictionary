Kruskal-Wallis检验

Kruskal-Wallis Test

Kruskal-Wallis检验（Kruskal-Wallis Test）

§1

方法概览

1.1 一句话本质

它是单因素方差分析的非参数版：把所有组混在一起排名，比较各组的平均名次是否相当——若某些组的名次系统性地偏高或偏低，就说明组间分布不同。

1.2 定义

Kruskal-Wallis 检验比较三个及以上独立组的分布位置，基于混合排秩，构造近似服从卡方分布的 $H$ 统计量，不要求数据正态，适用于连续或有序结局。

1.3 它主要解决什么问题

  • 研究问题：多个独立组的水平是否不同（在不敢假设正态时）？
  • 适用任务：多组非参数整体比较，通常配合事后两两比较（Dunn 检验）。
  • 常见医学场景：多组偏态指标比较、多个有序等级组比较、含异常值的多组数据。

1.4 直觉与类比

几个班级的考试排名混在一起。若各班水平相当，每班学生的名次应上下均匀分布、平均名次接近整体中点。若某班明显更强，他们会更多占据前排、平均名次更靠前。Kruskal-Wallis 就是检验「各班平均名次的差异」是否超出随机。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

ANOVA 依赖正态与方差齐；多组偏态、有序或含异常值时不可靠。而多组又不能靠多次秩和检验（会膨胀假阳性）。根本困难是：如何用一次非参数检验，判断「多组分布位置是否全相同」？

2.2 关键洞察

把全部 $N$ 个观测混合排秩，计算各组的平均秩 $\bar{R}_j$。若各组同分布，所有组的平均秩都应接近总平均秩 $(N+1)/2$；$H$ 统计量正是各组平均秩偏离总平均秩的加权平方和。它可看作「在秩上做的 ANOVA」——把 ANOVA 的组间平方和搬到秩尺度，于是继承了 ANOVA 的整体检验逻辑，又获得了秩方法的稳健性。

2.3 与朴素/相邻做法的对比

  • 相对单因素 ANOVA：不要求正态、抗异常值。
  • 相对多次秩和检验：一次检验控制整体假阳性率。
  • 相对 Wilcoxon 秩和检验：Kruskal-Wallis 是它从两组到多组的推广（两组时二者等价）。
§3

数学形式

3.1 核心公式

$$ H=\frac{12}{N(N+1)}\sum_{j=1}^{k}\frac{R_j^2}{n_j}-3(N+1) $$

其中 $R_j$ 是第 $j$ 组的秩和，$n_j$ 是其样本量，$N=\sum n_j$。这个式子在说：把各组秩和平方后按样本量归一再求和，衡量组间平均秩的分散程度；常数项使 $H$ 在原假设下以卡方分布为参照。

3.2 推导脉络

  • 混合排秩后，$H$ 等价于「对秩做单因素 ANOVA 的组间平方和」除以秩的总方差。
  • 原假设（各组同分布）下，$H$ 近似服从自由度 $k-1$ 的卡方分布。
  • 大 $H$ 意味某些组平均秩系统偏离总体，落入卡方右尾即拒绝。
  • 有结点时需除以结点校正因子；小样本可用精确分布。

3.3 参数与统计量含义

  • $k$：组数；$N$：总样本量；$n_j,R_j$：第 $j$ 组样本量与秩和。
  • $\bar{R}_j=R_j/n_j$：各组平均秩，越偏离 $(N+1)/2$ 越异常。
  • $H$：检验统计量，$\ge 0$；自由度 $k-1$。
  • p 值：卡方分布右尾概率。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
独立 组间、组内独立 p 值失真 看设计
可排序 结局可比大小 大量结点降低效力 看重复值
形状相近 各组分布形状类似 「位置」解释需谨慎 各组分布图
§4

手把手算例

三组各 3 例的花费评分：A = {1, 2, 3}，B = {4, 5, 6}，C = {7, 8, 9}（已从小到大，故秩即数值位次）。

一步步计算：

  • $N=9$，混合秩 $1\dots9$。各组秩和：$R_A=1+2+3=6$，$R_B=4+5+6=15$，$R_C=7+8+9=24$。
  • 代入：$\dfrac{12}{9\times10}\left(\dfrac{6^2}{3}+\dfrac{15^2}{3}+\dfrac{24^2}{3}\right)-3\times10$。
  • 括号内：$\dfrac{36}{3}+\dfrac{225}{3}+\dfrac{576}{3}=12+75+192=279$。
  • $H=\dfrac{12}{90}\times279-30=0.1333\times279-30=37.2-30=7.2$。
  • 自由度 $k-1=2$，卡方临界值 $\chi^2_{0.05,2}=5.99$；因 $7.2\gt 5.99$，拒绝「三组同分布」。

结论： 三组的平均秩分别是 2、5、8，单调递增且分散明显，$H=7.2$ 超过临界值，说明组间存在差异。与 ANOVA 一样，Kruskal-Wallis 只说「有组不同」，具体哪两组要用 Dunn 事后检验（并做多重比较校正）。注意每组仅 3 例，实务中应以精确检验复核。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：多分类分组变量（≥3 组）。
  • 因变量类型：连续或有序等级。
  • 数据结构：多组独立观测。
  • 是否适合高维数据：否。
  • 是否适合缺失较多数据：按可用样本。
  • 是否适合删失数据：不适合。
  • 是否适合重复测量数据：不适合；重复测量多组用 Friedman。

5.2 示例表格

组 值 秩和 平均秩
A 1,2,3 6 2
B 4,5,6 15 5
C 7,8,9 24 8

5.3 输入与产出

输入

  • 输入数据：分组变量 + 结局。
  • 关键变量：组别、结局值。
  • 需要预处理的内容：处理结点、确认独立。

产出

  • 模型对象/统计结果：$H$、自由度、p 值。
  • 参数估计：各组平均秩。
  • 预测结果：无。
  • 不确定性指标：事后两两比较的校正 p。
§6

适用场景

  • 适合：三组及以上、非正态或有序结局的整体比较。
  • 不适合：两组（用秩和检验）、重复测量（用 Friedman）。
  • 使用前需要特别检查的点：独立性、各组形状是否相近、是否需事后比较。
§7

实现

常用包：

  • scipy
from scipy import stats

A = [1, 2, 3]; B = [4, 5, 6]; C = [7, 8, 9]
H, p = stats.kruskal(A, B, C)
print(f"H={H:.2f}, p={p:.4f}")     # H≈7.2
# 事后: scikit_posthocs.posthoc_dunn

常用包：

  • stats
val <- c(1,2,3, 4,5,6, 7,8,9)
grp <- factor(rep(c("A","B","C"), each = 3))
kruskal.test(val ~ grp)
# 事后: dunn.test::dunn.test(val, grp, method = "bh")
§8

结果如何解读

  • 核心结果看什么：$H$/p，以及各组平均秩的排序。
  • 每个主要参数如何解读：p 小说明至少一组分布位置不同；平均秩显示方向。
  • 临床或医学意义如何表达：整体显著后用 Dunn 检验报告具体哪两组差。
  • 常见误读：把整体显著当成「每两组都不同」；形状差异大却硬说「中位数不同」。
§9

假设诊断与稳健性

  • 分布形状：各组形状相近时可解释为「中位数/位置差」。
  • 结点：大量相等值需结点校正，软件自动处理。
  • 稳健性：对异常值远比 ANOVA 稳健。
  • 事后与校正：Dunn 检验做两两比较，务必对多重性校正（如 BH）。

§11

优势、局限与常见坑

优势

  • 不依赖正态、抗异常值。
  • 一次检验控制多组假阳性率。
  • 适用有序结局。

局限

  • 只给整体结论，需事后定位。
  • 形状差异大时解释受限。
  • 大量结点降低效力。

常见坑

  • 显著后说「各组都不同」。
  • 事后两两比较不校正多重性。
  • 与 Friedman（重复测量）混用。
§12

与相近方法的区别

  • 和单因素 ANOVA：ANOVA 比均值、要正态；KW 比位置、更稳健。
  • 和 Wilcoxon 秩和检验：KW 是其多组推广。
  • 和 Friedman 检验：Friedman 用于重复测量多组。
  • 如何选择：多独立组且非正态 → KW；正态 → ANOVA。
§13

医学研究中的典型应用

  • 多组偏态实验室指标的整体比较。
  • 多个有序分级组的比较。
  • 多中心/多亚型含异常值数据的比较。
§14

关键术语

  • 平均秩（Mean Rank）：某组在混合排序中的名次均值。
  • H 统计量：衡量组间平均秩分散度、近似卡方的统计量。
  • 卡方近似：$H$ 在原假设下近似卡方分布（自由度 $k-1$）。
  • Dunn 检验：Kruskal-Wallis 的事后两两比较方法。
  • 结点校正：存在相等观测时对 $H$ 的修正。
§15

相关方法

  • 单因素方差分析（One-Way ANOVA）
  • Wilcoxon秩和检验（Wilcoxon Rank-Sum Test）
