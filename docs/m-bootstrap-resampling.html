Bootstrap Resampling

Bootstrap重抽样（Bootstrap Resampling）

§1

方法概览

1.1 一句话本质

Bootstrap 用「从手上的样本里有放回地反复重抽」来模拟「把整个研究重做很多遍」，从而不靠任何公式，就能估计任意统计量的抽样波动、标准误和置信区间。

1.2 定义

Bootstrap（自助法）是一种重抽样方法：把观测样本当作总体的替身，有放回地抽取大量「自助样本」，在每个上重算统计量，用这些重复值的分布来近似该统计量的真实抽样分布。

1.3 它主要解决什么问题

  • 研究问题：某个统计量（尤其没有简单公式的）有多不确定？置信区间是多少？
  • 适用任务：复杂统计量的标准误与区间估计、正态近似不成立时的推断。
  • 常见医学场景：中位数/分位数的区间、相关系数或 AUC 的区间、比值/比率、非线性组合指标的不确定性。

1.4 直觉与类比

理想情况下，想知道「估计有多准」，应该把整个研究重复上千次，看每次结果散多开。现实只做得起一次。Bootstrap 的妙招是：既然样本是总体最好的缩影，就把样本当「小总体」，从里面有放回地反复抽同样大小的新样本——每次抽都像「重做一次研究」。上千次之后，这些结果的散布就近似了「真重复研究」的散布。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

均值有现成的标准误公式，但中位数、相关系数、AUC、两个估计的比值……很多统计量要么没有简洁的方差公式，要么其正态近似在小样本/偏态下失效。根本困难是：如何在没有解析公式、也不敢用正态近似时，得到一个统计量的不确定性？

2.2 关键洞察

用经验分布代替未知的总体分布。既然真总体不可见，就用「样本的经验分布」（每个观测各占 $1/n$）当它的最佳估计；「从总体再抽一个样本」就用「从样本有放回地抽 $n$ 个」来模拟。对每个自助样本重算统计量 $\hat{\theta}^*$，重复 $B$ 次（几千次），这 $B$ 个值的分布就近似 $\hat{\theta}$ 的抽样分布——其标准差是标准误，其百分位给置信区间。它把「解析推导」换成了「计算模拟」。

2.3 与朴素/相邻做法的对比

  • 相对解析标准误：Bootstrap 通用、不需为每个统计量推公式；代价是要算力、且小样本时不可靠。
  • 相对正态近似 CI：偏态或小样本时 Bootstrap（尤其 BCa）区间更准。
  • 相对置换检验：置换用于假设检验（打乱标签），Bootstrap 用于估计不确定性（有放回重抽）。
§3

数学形式

3.1 核心公式

自助标准误与百分位区间（$\hat{\theta}^{*(b)}$ 为第 $b$ 个自助样本上的统计量）：

$$ \widehat{\mathrm{SE}}=\sqrt{\frac{1}{B-1}\sum_{b=1}^{B}\big(\hat{\theta}^{*(b)}-\bar{\theta}^{*}\big)^2},\qquad \mathrm{CI}_{95\%}=\big[\hat{\theta}^{*}_{(2.5\%)},\ \hat{\theta}^{*}_{(97.5\%)}\big] $$

这个式子在说：标准误就是 $B$ 个自助统计量的标准差；百分位区间就是把这 $B$ 个值排序、取第 2.5 与 97.5 百分位。

3.2 推导脉络

  • 记经验分布 $\hat{F}_n$（每个观测权重 $1/n$）；它是总体 $F$ 的一致估计。
  • 「Bootstrap 原理」：用 $\hat{F}_n$ 下 $\hat{\theta}^*$ 相对 $\hat{\theta}$ 的分布，近似 $F$ 下 $\hat{\theta}$ 相对真值 $\theta$ 的分布。
  • 无法解析积分，就用蒙特卡洛：抽 $B$ 个自助样本、各算 $\hat{\theta}^*$，用经验分布逼近。
  • 区间有多种构造：百分位法（简单）、基本法、BCa（校正偏差与偏度，最常推荐）、自助 t。

3.3 参数与统计量含义

  • $B$：自助重复次数（标准误用几百、区间用 $\ge 2000$）。
  • $\hat{\theta}^{*(b)}$：第 $b$ 个自助样本上的统计量。
  • $\widehat{\mathrm{SE}}$：自助标准误；百分位：区间端点。
  • 分布形状：自助值的直方图直接展示抽样分布是否偏态。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
样本代表总体 经验分布近似总体 小样本时不可靠 看样本量
观测独立 i.i.d. 抽样 需分块/分层自助 看数据结构
统计量平滑 对数据平滑依赖 极值/边界统计量失效 看统计量类型
§4

手把手算例

用 5 个观测 $\{2,4,6,8,10\}$（均值 6）演示自助抽样如何模拟重复研究。

一步步计算：

  • 有放回抽 $n=5$ 得若干自助样本，各算均值：
  • 自助样本 1：$\{4,4,8,10,2\}\to$ 均值 $5.6$
  • 自助样本 2：$\{2,6,6,8,10\}\to$ 均值 $6.4$
  • 自助样本 3：$\{10,8,6,4,2\}\to$ 均值 $6.0$
  • 重复 $B=2000$ 次，得到 2000 个均值。它们的标准差就是自助标准误，约 $1.3$（与解析值 $s/\sqrt{n}=3.16/\sqrt5=1.41$ 接近）。
  • 把 2000 个均值排序，取第 2.5% 与 97.5% 百分位，即得 95% 自助置信区间（约 $3.6$ 到 $8.4$）。

结论： 只靠「反复重抽 + 重算」，就复现了均值的标准误与区间，完全没用到任何标准误公式。对均值这种有公式的量，两者结果吻合；真正的价值在于——把「均值」换成「中位数」「相关系数」「AUC」等没有简单公式的量，同一套流程照样给出不确定性。注意每个自助样本会重复或漏掉某些原始观测，这正是它模拟抽样波动的机制。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：任意（对统计量重抽）。
  • 因变量类型：任意。
  • 数据结构：独立观测；相关数据用分块/分层自助。
  • 是否适合高维数据：可，但计算量大。
  • 是否适合缺失较多数据：需把缺失处理纳入每次重抽。
  • 是否适合删失数据：需专门的生存自助。
  • 是否适合重复测量数据：需按对象整块重抽（cluster bootstrap）。

5.2 示例表格

自助样本 抽到的值 均值
1 4,4,8,10,2 5.6
2 2,6,6,8,10 6.4
3 10,8,6,4,2 6.0

5.3 输入与产出

输入

  • 输入数据：原始样本 + 一个「计算统计量」的函数。
  • 关键变量：统计量、重复次数 $B$、区间方法。
  • 需要预处理的内容：确认独立性、决定重抽单元（个体/整群）。

产出

  • 模型对象/统计结果：自助分布、标准误、置信区间。
  • 参数估计：原统计量。
  • 预测结果：无。
  • 不确定性指标：自助 SE 与区间（百分位/BCa）。
§6

适用场景

  • 适合：复杂/无公式统计量的不确定性、偏态或小样本下的区间。
  • 不适合：极值/边界统计量（最大值）、强相关未处理、样本太小。
  • 使用前需要特别检查的点：重抽单元是否正确、$B$ 是否足够、区间方法选择。
§7

实现

常用包：

  • numpy
  • scipy
import numpy as np
from scipy.stats import bootstrap

x = np.array([2, 4, 6, 8, 10])
res = bootstrap((x,), np.mean, n_resamples=2000,
                confidence_level=0.95, method="BCa")
print("SE =", round(res.standard_error, 2))
print("95% CI =", res.confidence_interval)

常用包：

  • boot
library(boot)
x <- c(2,4,6,8,10)
b <- boot(x, statistic = function(d, i) mean(d[i]), R = 2000)
boot.ci(b, type = "bca")
§8

结果如何解读

  • 核心结果看什么：自助分布形状、标准误、置信区间是否含无效值。
  • 每个主要参数如何解读：区间偏离对称提示统计量抽样分布偏态；宽度反映精度。
  • 临床或医学意义如何表达：报告点估计 + 自助 95% CI（注明区间方法与 $B$）。
  • 常见误读：把 Bootstrap 当「凭空造数据」（它只是重排现有信息）；小样本上盲信。
§9

假设诊断与稳健性

  • 收敛：增大 $B$（如 2000→10000）看区间是否稳定。
  • 分布检查：画自助值直方图，偏态时优先 BCa 而非百分位。
  • 相关/聚类：普通自助会低估不确定性，改用按整群重抽。
  • 边界统计量：最大/最小值等对重抽极敏感，Bootstrap 不适用。
§10

推荐可视化

  • 自助分布直方图 + 区间标注。
  • 多方法区间（百分位/BCa/正态）对比。
  • 统计量随 $B$ 增大的收敛曲线。

§11

优势、局限与常见坑

优势

  • 通用，不需为每个统计量推公式。
  • 偏态/小样本下区间常优于正态近似。
  • 直接展示抽样分布形状。

局限

  • 计算量大。
  • 小样本时经验分布本身不可靠。
  • 对极值/边界统计量失效。

常见坑

  • 相关/聚类数据用普通自助（低估不确定性）。
  • $B$ 太小导致区间不稳。
  • 把自助当成「增加了样本量」。
§12

与相近方法的区别

  • 和解析标准误/CI：Bootstrap 通用但耗算力；有公式时解析更快。
  • 和置换检验：置换检验假设、Bootstrap 估不确定性。
  • 和贝叶斯后验：都给不确定性，但框架与解释不同。
  • 如何选择：有可靠公式用解析；复杂/偏态统计量用 Bootstrap。
§13

医学研究中的典型应用

  • AUC、C-index、敏感度/特异度的置信区间。
  • 中位生存/分位数的区间估计。
  • 复杂派生指标（比值、差的比）的不确定性。
§14

关键术语

  • 重抽样（Resampling）：从样本反复抽取新样本的方法。
  • 有放回抽样（With Replacement）：抽后放回，允许重复，Bootstrap 的核心。
  • 经验分布：样本各点等权重的分布，总体的估计。
  • 百分位区间：用自助值分位数构造的置信区间。
  • BCa 区间：校正偏差与偏度的自助区间，常最优。
  • 蒙特卡洛：用大量随机模拟逼近难以解析的量。
§15

相关方法

  • 置信区间（Confidence Interval）
  • 一致性指数（Concordance Index, C-index）
  • ROC曲线与AUC（Receiver Operating Characteristic and AUC）
