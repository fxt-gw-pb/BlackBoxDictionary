线性回归（Linear Regression）

§1

方法概览

1.1 一句话本质

线性回归用一条直线（或超平面）去拟合「给定自变量时因变量的平均值」，办法是找到让所有点到直线的竖直残差平方和最小的那条线。

1.2 定义

线性回归假设结局的条件期望是自变量的线性组合，用最小二乘估计系数，量化每个自变量对连续结局的平均贡献。它是统计建模的基石，也是 GLM、正则化、混合模型等的出发点。

1.3 它主要解决什么问题

  • 研究问题：哪些因素、以多大幅度影响一个连续结局？给定自变量能预测多少？
  • 适用任务：连续结局的关联分析、效应估计、预测、混杂调整。
  • 常见医学场景：年龄/BMI 对血压的影响、剂量对某连续指标的效应、校正混杂后估计治疗效应。

1.4 直觉与类比

一堆散点像天上的星星，线性回归就是画一条「最贴合大多数星星」的直线。「最贴合」的标准是：把每颗星星到直线的竖直距离（残差）平方后加起来，选让这个总和最小的直线——平方是为了让大偏差受更重的惩罚，也让正负偏差不相互抵消。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

自变量与结局的关系被噪声掩盖：同样的 BMI，血压也高低不一。根本困难是：如何从带噪声的散点里，提取出「平均趋势」，并把每个因素的独立贡献分离、量化出来？

2.2 关键洞察

用「最小二乘」把「拟合得好」变成一个可求解的优化：最小化残差平方和 $\sum(y_i-\hat{y}_i)^2$。这个目标是系数的二次函数，有唯一的闭式解（正规方程）。几何上，最优拟合就是把结局向量 $Y$正交投影到自变量张成的空间上——残差与所有自变量垂直，这正是「已用尽自变量里的线性信息」的含义。多元回归里每个系数是「固定其他变量后」的偏效应，从而实现混杂调整。

2.3 与朴素/相邻做法的对比

  • 相对只算相关系数：回归给出带单位的斜率、可预测、可放多个自变量做调整。
  • 相对 GLM：线性回归是 GLM 在「正态分布 + 恒等连接」下的特例；结局非正态时用 GLM。
  • 相对正则化回归：高维/共线时加 L1/L2 惩罚（Lasso/Ridge）稳定估计。
§3

数学形式

3.1 核心公式

$$ Y=X\boldsymbol{\beta}+\boldsymbol{\varepsilon},\qquad \hat{\boldsymbol{\beta}}=(X^\top X)^{-1}X^\top Y $$

一元情形：$\hat{\beta}_1=\dfrac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sum(x_i-\bar{x})^2}$，$\hat{\beta}_0=\bar{y}-\hat{\beta}_1\bar{x}$。这个式子在说：斜率是「x 与 y 的协动」除以「x 的离散」，截距使直线过点 $(\bar{x},\bar{y})$。

3.2 推导脉络

  • 目标：最小化 $S(\boldsymbol{\beta})=\|Y-X\boldsymbol{\beta}\|^2$。
  • 对 $\boldsymbol{\beta}$ 求导置零得正规方程 $X^\top X\boldsymbol{\beta}=X^\top Y$，解出闭式 $\hat{\boldsymbol{\beta}}$。
  • 高斯-马尔可夫定理：在零均值、同方差、无自相关误差下，OLS 是最优线性无偏估计（BLUE）。
  • 关系提示：$\hat{\beta}_1=r\cdot\dfrac{s_y}{s_x}$，即斜率 = 相关系数按标准差比例还原到原尺度。

3.3 参数与统计量含义

  • $\beta_0$：截距（所有 $x=0$ 时的期望结局）。
  • $\beta_j$：$x_j$ 每增一单位、其他变量固定时结局的平均变化。
  • $R^2$：模型解释的结局方差比例（一元时 $=r^2$）。
  • 残差标准误、系数标准误 / p 值 / CI：拟合精度与推断。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
线性(L) 均值与 X 线性 效应有偏、预测差 残差 vs 拟合值图
独立(I) 误差互不相关 SE 失真 看设计/自相关
同方差(E) 误差方差恒定 SE 有偏 残差散布图
正态(N) 误差近正态 小样本推断不准 残差 QQ 图
§4

手把手算例

5 对观测（沿用相关分析的数据）：$x=\{1,2,3,4,5\}$，$y=\{2,3,5,4,6\}$。

一步步计算：

  • 均值：$\bar{x}=3$，$\bar{y}=4$。
  • 离均差乘积和：$\sum(x-\bar{x})(y-\bar{y})=9$；$\sum(x-\bar{x})^2=10$。
  • 斜率：$\hat{\beta}_1=9/10=0.90$。
  • 截距：$\hat{\beta}_0=\bar{y}-\hat{\beta}_1\bar{x}=4-0.90\times3=1.30$。
  • 拟合方程：$\hat{y}=1.30+0.90x$。预测 $x=3$：$\hat{y}=1.30+2.70=4.0$。
  • $R^2=r^2=0.90^2=0.81$。

结论： 每增加 1 单位 $x$，$y$ 平均升高 0.90；模型解释了 $y$ 约 81% 的方差。注意此处斜率 0.90 恰等于相关系数 $r=0.90$——这是因为本例 $s_x=s_y$，一般情形下 $\hat{\beta}_1=r\cdot s_y/s_x$，相关系数与回归斜率是同一信息在不同尺度上的两种表达。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：连续、二分类、分类（编码后）均可。
  • 因变量类型：连续型。
  • 数据结构：每行一个观测的表格。
  • 是否适合高维数据：$p$ 接近或超过 $n$ 时需正则化。
  • 是否适合缺失较多数据：需先处理缺失。
  • 是否适合删失数据：不适合（用生存模型）。
  • 是否适合重复测量数据：不适合；用混合模型/GEE。

5.2 示例表格

i x y
1 1 2
3 3 5
5 5 6

5.3 输入与产出

输入

  • 输入数据：自变量矩阵 + 连续结局。
  • 关键变量：结局、协变量、交互项。
  • 需要预处理的内容：编码分类变量、查线性与异常值、必要时变换。

产出

  • 模型对象/统计结果：系数、标准误、t 检验、$R^2$、F 检验。
  • 参数估计：$\hat{\boldsymbol{\beta}}$。
  • 预测结果：拟合值与预测区间。
  • 不确定性指标：系数 CI、预测区间。
§6

适用场景

  • 适合：连续结局的关联/预测、混杂调整、效应量化。
  • 不适合：二分类/计数/时间到事件结局（用相应 GLM/生存模型）、强非线性未处理、重复测量。
  • 使用前需要特别检查的点：线性、同方差、异常/杠杆点、共线性。
§7

实现

常用包：

  • statsmodels
import statsmodels.formula.api as smf
import pandas as pd

df = pd.DataFrame({"x": [1,2,3,4,5], "y": [2,3,5,4,6]})
fit = smf.ols("y ~ x", data=df).fit()
print(fit.summary())          # 系数、R²、p
print(fit.predict({"x": [3]}))   # 4.0

常用包：

  • stats
df <- data.frame(x = 1:5, y = c(2,3,5,4,6))
fit <- lm(y ~ x, data = df)
summary(fit)                  # 系数、R²
confint(fit)                  # 系数 95% CI
§8

结果如何解读

  • 核心结果看什么：系数方向与大小、CI、$R^2$、残差诊断。
  • 每个主要参数如何解读：$\hat{\beta}_1=0.9$ 即「$x$ 每增 1，$y$ 平均增 0.9」；截距常无独立意义。
  • 临床或医学意义如何表达：报告效应量 + 区间，并说明单位与调整了哪些变量。
  • 常见误读：把关联当因果；$R^2$ 高当模型正确（可能非线性/过拟合）；外推到数据范围外。
§9

假设诊断与稳健性

  • 线性与同方差：残差 vs 拟合值图应无结构、无喇叭形；喇叭形提示异方差，用稳健 SE 或变换。
  • 正态性：残差 QQ 图；大样本靠 CLT 较稳。
  • 影响点：Cook 距离、杠杆值排查单点主导。
  • 共线性：VIF 过高时合并/剔除变量或用正则化。
§10

推荐可视化

  • 散点图 + 拟合线 + 置信带。
  • 残差 vs 拟合值图、残差 QQ 图。
  • 系数森林图（多元）。

§11

优势、局限与常见坑

优势

  • 简单、可解释、有闭式解与完整推断理论。
  • 是几乎所有高级模型的基础。
  • 系数直接给带单位的效应。

局限

  • 只建模线性均值关系。
  • 对异常值/杠杆点敏感。
  • 要求同方差、正态误差（推断时）。

常见坑

  • 不看残差图就信结果。
  • 对二分类/计数结局误用线性回归。
  • 把 $R^2$ 当模型正确性的证据。
§12

与相近方法的区别

  • 和 GLM：线性回归是 GLM 的正态+恒等连接特例；结局非正态用 GLM。
  • 和 Pearson 相关：回归有方向、带单位、可多变量调整；相关对称无量纲。
  • 和 Ridge/Lasso：高维/共线时加惩罚更稳、可变量选择。
  • 和混合模型：重复测量/分层数据用混合模型。
§13

医学研究中的典型应用

  • 连续生物标志物的危险因素分析。
  • 剂量-连续反应的效应估计。
  • 校正混杂后估计治疗对连续结局的效应。
§14

关键术语

  • 最小二乘（OLS）：最小化残差平方和的估计法。
  • 残差（Residual）：观测值与拟合值之差。
  • 决定系数（$R^2$）：模型解释的结局方差比例。
  • 同方差（Homoscedasticity）：误差方差恒定。
  • 杠杆/影响点：对拟合影响过大的观测。
  • 正规方程：$X^\top X\beta=X^\top Y$，OLS 的解方程。
