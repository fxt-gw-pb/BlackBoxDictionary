02·01
置信区间 Confidence Interval

置信区间给点估计配上一段范围，回答「这个数字有多可信」；95% 指的是方法的长期表现——反复抽样构造的区间里，约 95% 会罩住真值。

02·02
单样本t检验 One-Sample t-Test

单样本 t 检验问一个问题：这组数据的均值，离某个「参照值」是不是远到不像抽样运气？做法是把「均值与参照值之差」除以它的标准误，得到信噪比 $t$。

◑ 已建 basic
02·03
配对样本t检验 Paired t-Test

配对 t 检验把「每一对的差值」算出来，再对这一列差值做单样本 t 检验（看它是否偏离 0）——用「配对」抵消个体之间的差异，只留下处理效应。

◑ 已建 basic
02·04
两独立样本t检验 Two-Sample t-Test

它把「两组均值的差」除以「这个差本身的随机波动（标准误）」，得到一个信噪比 $t$：差异相对波动足够大，就判断两组总体均值真的不同。

◑ 已建 basic
02·05
单因素方差分析 One-Way ANOVA

ANOVA 用一个巧妙的对比回答「多组均值是否相同」：把数据的总变异拆成「组与组之间的差异」和「组内部的波动」，若前者远大于后者（$F$ 大），就说明组间真有差别。

◑ 已建 basic
02·06
TukeyHSD多重比较 Tukey Honest Significant Difference

Tukey HSD 在 ANOVA 显著之后，对所有组两两比较均值，并用一个「统一加高」的临界值，保证整族比较的假阳性率仍控制在 5%——回答「到底是哪两组不同」。

◑ 已建 intermediate
02·07
Wilcoxon符号秩检验 Wilcoxon Signed-Rank Test

它是配对 t 检验的非参数版：不去比差值的「平均大小」，而是比差值的符号和秩（谁大谁小的顺序），因此对偏态和异常值稳健。

◑ 已建 basic
02·08
Wilcoxon秩和检验 Wilcoxon Rank-Sum Test

它是两独立样本 t 检验的非参数版：把两组数据混在一起从小到大排名，看某一组的「名次总分」是否系统性地偏高或偏低——等价于问「随机各取一个，A 大于 B 的概率是不是 0.5」。

◑ 已建 basic
02·09
Kruskal-Wallis检验 Kruskal-Wallis Test

它是单因素方差分析的非参数版：把所有组混在一起排名，比较各组的平均名次是否相当——若某些组的名次系统性地偏高或偏低，就说明组间分布不同。

◑ 已建 intermediate
02·10
Pearson相关 Pearson Correlation

Pearson 相关系数 $r$ 是「标准化的协方差」：把两个变量各自的单位除掉，得到一个 $-1$ 到 $1$ 之间、无量纲的数，衡量它们线性关系的方向与强度。

◑ 已建 basic
02·11
Spearman秩相关 Spearman Rank Correlation

Spearman 相关就是「把数据换成名次后再算 Pearson 相关」：它衡量两个变量是否同增同减（单调关系），不要求关系是直线，且对异常值稳健。

◑ 已建 basic
02·12
Kendall秩相关 Kendall Rank Correlation

Kendall $\tau$ 直接数「排序一致的对」减去「排序矛盾的对」，再归一化：它回答「随机挑两个个体，两个变量给出相同大小顺序的概率，比相反顺序高多少」。

◑ 已建 intermediate
02·13
Pearson卡方独立性检验 Pearson Chi-Squared Test of Independence

卡方检验把列联表里「实际观测到的频数」和「假如两变量毫无关系时本该出现的期望频数」逐格相比：两者差得越大，越说明两个分类变量之间存在关联。

◑ 已建 basic
02·14
Fisher精确检验 Fisher Exact Test

Fisher 精确检验不靠大样本近似：它固定行列合计，直接用超几何分布把「所有可能的表」一一枚举，加总「比观测更极端」的那些表的概率，得到精确的 p 值。

◑ 已建 intermediate
02·15
McNemar检验 McNemar Test

McNemar 检验用于配对的二分类数据：它只看发生了改变的那些对（前阳后阴、前阴后阳），忽略两次结果一致的对，判断「改变」是否偏向某一方向。

◑ 已建 intermediate
02·16
Mantel-Haenszel检验 Mantel-Haenszel Test

Mantel-Haenszel 把数据按一个混杂因素（如年龄）分层，在每一层里干净地看暴露与结局的关联，再把各层的关联加权合并成一个「已调整」的比值比——从而在不建回归模型的情况下

◑ 已建 intermediate
02·17
Bootstrap重抽样 Bootstrap Resampling

Bootstrap 用「从手上的样本里有放回地反复重抽」来模拟「把整个研究重做很多遍」，从而不靠任何公式，就能估计任意统计量的抽样波动、标准误和置信区间。

◑ 已建 intermediate
02·18
Cohen's Kappa一致性系数 Cohen's Kappa

Kappa 衡量两名评分者分类判断的一致性，但先扣掉「就算瞎猜也能碰巧一致」的部分：它回答「在机遇一致之上，两人还多达成了多少一致」。

◑ 已建 basic
02·19
组内相关系数 Intraclass Correlation Coefficient, ICC

ICC 是「真实的个体间差异」占「总变异」的比例：若同一个人重复测量都很接近、而不同人之间差异明显，说明测量可靠，ICC 就高。

◑ 已建 intermediate
02·20
Bland-Altman分析 Bland-Altman Analysis

Bland-Altman 直接盯着「两种方法测同一对象的差值」：算出平均偏差（系统差）和 95% 一致性界限（个体差能有多大），据此判断两法能不能互换——而不是问它们相不相关。

◑ 已建 basic
