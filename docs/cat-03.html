回归与广义线性模型

Regression & Generalized Linear Models

以线性预测子连接期望与协变量的通用框架。

收录条目
03·01
线性回归 Linear Regression

线性回归通过把连续型结局表示为若干自变量的线性组合，来估计变量与结局之间的方向、大小和不确定性，是医学统计最基础的建模方法之一。

◑ 已建 basic
03·02
广义线性模型 Generalized Linear Model, GLM

广义线性模型是把线性模型推广到非高斯结局的一套统一框架，通过“分布族 + 线性预测子 + 连接函数”来建模连续、二分类和计数等不同类型的结局。

◑ 已建 intermediate
03·03
Logistic回归 Logistic Regression

Logistic 回归是最经典的二分类结局回归模型，用 logit 连接函数把事件概率映射到线性预测子上，从而估计协变量与事件发生 odds 的关系。

◑ 已建 basic
03·04
Poisson回归 Poisson Regression

Poisson 回归用于建模非负整数计数结局，核心思想是在 log 尺度上把期望计数表示成协变量的线性组合。

◑ 已建 basic
03·05
Quasi-Likelihood与过度离散 Quasi-Likelihood and Overdispersion

Quasi-Likelihood 是 GLM 的半参数扩展，不要求完整指定响应分布，只需要指定均值模型和方差模型，因此常被用来处理过度离散和稳健方差估计问题。

◑ 已建 intermediate
03·06
多项式回归 Polynomial Regression

多项式回归是在普通线性回归的基础上加入自变量的高次项，从而用一条更灵活的曲线来拟合非线性关系。

◑ 已建 basic
03·07
贝叶斯回归 Bayesian Regression

贝叶斯回归是一类把回归参数本身视为随机变量、通过先验分布和观测数据共同得到后验分布的回归方法。

◑ 已建 intermediate
03·08
局部加权回归 Locally Weighted Regression

局部加权回归是一种在每个预测点附近单独拟合局部模型的非参数回归方法，距离预测点越近的样本权重越大。

◑ 已建 intermediate
03·09
负二项回归 Negative Binomial Regression

负二项回归是计数结局的回归模型，通过在 Poisson 基础上引入一个额外的离散参数，允许方差大于均值，从而处理过度离散的计数数据。

◑ 已建 intermediate
03·10
有序Logistic回归 Ordinal Logistic Regression

有序 Logistic 回归（比例优势模型）用于结局为有序类别的建模，通过对「累积概率」建 logit，估计协变量如何把个体推向更高等级。

◑ 已建 intermediate
03·11
限制性立方样条 Restricted Cubic Splines, RCS

限制性立方样条是一种把连续协变量展开为若干基函数的建模技术，在结点之间用三次多项式平滑拟合，并强制两端呈线性，从而灵活刻画非线性关系而不过度摆动。

◑ 已建 intermediate
03·12
多项Logistic回归 Multinomial Logistic Regression

多项 Logistic 回归把二分类 Logistic 推广到三类及以上的无序结局，通过对每个类别相对参考类别建 logit，估计协变量对各类别相对概率的影响。

◑ 已建 intermediate
