回归与广义线性模型

Regression & Generalized Linear Models

以线性预测子连接期望与协变量的通用框架。

收录条目
03·01
线性回归 Linear Regression

线性回归用一条直线（或超平面）去拟合「给定自变量时因变量的平均值」，办法是找到让所有点到直线的竖直残差平方和最小的那条线。

◑ 已建 basic
03·02
广义线性模型 Generalized Linear Model, GLM

GLM 把线性回归推广到「非正态结局」：它不直接对结局本身做线性拟合，而是先用一个连接函数把「结局的期望」变换到实数轴上，再对这个变换后的量做线性回归，并用合适的分布族描述随机波动

◑ 已建 intermediate
03·03
Logistic回归 Logistic Regression

Logistic 回归先把「事件发生的概率」翻译成一个可以取任意实数的量——对数优势（log-odds），再对这个量做普通的线性回归；预测时反向翻译回 0 到 1 的概率。

◑ 已建 basic
03·04
Poisson回归 Poisson Regression

Poisson 回归对「事件发生的次数」建模：用 log 连接把「平均计数」连到协变量的线性组合，系数取指数后就是率比（某因素让事件发生率变为几倍）。

◑ 已建 intermediate
03·05
Quasi-Likelihood与过度离散 Quasi-Likelihood and Overdispersion

拟似然不承诺结局的完整分布，只规定「均值-方差关系」：它用一个离散因子 $\phi$ 把标准误按实际观测到的分散程度放大，从而在过度离散时得到诚实的推断，而系数点估计不变。

◑ 已建 intermediate
03·06
多项式回归 Polynomial Regression

多项式回归是线性回归最直接的「弯曲」扩展：把 $x,x^2,x^3,\dots$ 当作新的自变量一起塞进线性回归，就能拟合曲线——而模型对系数依然是线性的，仍用普通最小二乘求解。

◑ 已建 basic
03·07
贝叶斯回归 Bayesian Regression

贝叶斯回归把回归系数当作有先验分布的随机量，用数据把先验更新成后验分布——输出不再是一个点估计加区间，而是参数与预测的整条概率分布，不确定性天然内建。

◑ 已建 intermediate
03·08
局部加权回归 Locally Weighted Regression

局部加权回归不去拟合一条全局曲线，而是「走到哪拟合到哪」：在每个查询点附近，只用它的近邻、按远近加权做一个小小的线性回归，把这些局部拟合值串起来，就得到一条自动贴合数据形状的光滑曲

◑ 已建 intermediate
03·09
负二项回归 Negative Binomial Regression

负二项回归是 Poisson 回归的「加宽版」：它多带一个离散参数，允许方差大于均值，从而正确处理真实计数里普遍存在的「过度离散」。

◑ 已建 intermediate
03·10
有序Logistic回归 Ordinal Logistic Regression

有序 Logistic 回归（比例优势模型）对「累积概率」建 logit，并让所有切点共享同一组斜率——于是一个比值比就能概括「协变量把个体推向更高等级」的整体倾向。

◑ 已建 intermediate
03·11
限制性立方样条 Restricted Cubic Splines, RCS

限制性立方样条把连续变量切成几段，每段用三次曲线平滑拟合、段与段在结点处光滑衔接，并强制两端呈直线——既能灵活刻画弯曲关系，又不像高次多项式那样在两端狂摆。

◑ 已建 intermediate
03·12
多项Logistic回归 Multinomial Logistic Regression

多项 Logistic 把二分类 Logistic 推广到三类以上无序结局：选一类作参照，对其余每一类各建一条「该类 vs 参照类」的 logit，用 softmax 保证所有类别

◑ 已建 intermediate
