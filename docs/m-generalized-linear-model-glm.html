广义线性模型

Generalized Linear Model, GLM
状态 · ◑ 已建 难度 · intermediate 标签 · 医学统计 / 数据科学 / GLM / 建模框架 / 连接函数

广义线性模型（Generalized Linear Model, GLM）

§1

方法概览

1.1 一句话本质

GLM 把线性回归推广到「非正态结局」：它不直接对结局本身做线性拟合，而是先用一个连接函数把「结局的期望」变换到实数轴上，再对这个变换后的量做线性回归，并用合适的分布族描述随机波动。

1.2 定义

广义线性模型是一个统一框架，由三部分组成：随机成分（结局服从某指数族分布）、系统成分（线性预测子 $\eta=X\boldsymbol{\beta}$）、连接函数（$g(\mu)=\eta$）。线性回归、Logistic 回归、Poisson 回归都是它的特例。

1.3 它主要解决什么问题

  • 研究问题：结局不是正态连续（而是二分类、计数、比例…）时，如何统一地建模？
  • 适用任务：为各类结局选对分布与连接函数，做关联分析与预测。
  • 常见医学场景：二分类疾病结局（Logistic）、事件计数（Poisson）、比例/率、正偏连续结局（Gamma）。

1.4 直觉与类比

线性回归是一台只认「正态连续结局」的机器。GLM 给它加了两个可换的旋钮：一个选「结局的随机性长什么样」（分布族），一个选「怎么把有界/离散的期望翻译到能做线性回归的实数轴上」（连接函数）。换不同旋钮组合，同一台机器就分别变成线性回归、Logistic、Poisson——它们本是一家。

§2

核心思想与原理

2.1 它到底在解决什么根本困难

线性回归假设结局正态、方差恒定、期望可取任意实数。但二分类结局的期望是 $[0,1]$ 的概率、计数结局的期望是非负数、且它们的方差随均值变化。直接线性回归会预测出界、且方差假设错误。根本困难是：如何用一套框架，处理各种「有界、离散、方差不恒定」的结局？

2.2 关键洞察

拆成三块、各自可替换：

  1. 随机成分：结局服从某个指数族分布（正态、二项、泊松、Gamma…），它决定了「均值-方差关系」。
  2. 系统成分：协变量的线性组合 $\eta=X\boldsymbol{\beta}$，永远是线性的。
  3. 连接函数 $g$：把有界/离散的期望 $\mu$ 映到无界实数 $\eta$（如 logit 把 $[0,1]$ 映到 $\mathbb{R}$）。

关键在于「线性」被保留在连接后的尺度上，而非结局本身。选定 (分布, 连接)，就选定了一个具体模型，全部用最大似然（IRLS）统一估计。

2.3 与朴素/相邻做法的对比

  • 相对线性回归：GLM 是其推广（正态+恒等连接即线性回归）。
  • 相对给结局硬做变换再线性回归：GLM 在「均值的连接尺度」建模、方差结构正确；对结局本身取对数会改变解释、且遇零失效。
  • 相对 GLMM/GEE：数据相关（重复测量）时，用它们把 GLM 扩展到相关结构。
§3

数学形式

3.1 核心公式

$$ g\big(E[Y\mid X]\big)=g(\mu)=\eta=X\boldsymbol{\beta} $$

这个式子在说：不是 $\mu=X\beta$，而是「$\mu$ 经连接函数 $g$ 变换后」等于线性预测子。预测时用反连接 $\mu=g^{-1}(\eta)$ 还原。

3.2 推导脉络

  • 指数族分布的均值-方差有统一形式 $\mathrm{Var}(Y)=\phi\,V(\mu)$，$V$ 为方差函数。
  • 每个分布有一个「典范连接」（正态→恒等、二项→logit、泊松→log），使数学最简洁。
  • 参数用最大似然估计，得分方程通过迭代重加权最小二乘（IRLS）求解——每步都像做一次加权线性回归。
  • 拟合优度用偏差（deviance） 衡量（饱和模型与当前模型对数似然之差的两倍）。

3.3 参数与统计量含义

  • $\mu=E[Y\mid X]$：结局条件期望。
  • $g$：连接函数；$g^{-1}$：反连接（预测用）。
  • $\boldsymbol{\beta}$：连接尺度上的效应；反连接后有各自解释（OR、率比等）。
  • 偏差、AIC：拟合与模型比较；$\phi$：离散/尺度参数。

3.4 关键假设（含违反后果）

假设 含义 违反后会怎样 如何粗查
分布族正确 均值-方差关系对 过度离散、SE 失真 偏差/自由度、残差
连接正确 连接尺度上线性 效应有偏 连接尺度残差图
观测独立 无相关结构 SE 失真 看设计；相关用 GEE/GLMM
§4

手把手算例

同一套「三件套」在不同结局上落地为不同模型：

结局类型 分布族 典范连接 反连接 $\mu=g^{-1}(\eta)$ 得到的模型
连续正态 正态 恒等 $\mu=\eta$ $\mu=\eta$ 线性回归
二分类 二项 logit $\log\frac{\mu}{1-\mu}$ $\mu=\frac{1}{1+e^{-\eta}}$ Logistic 回归
计数 泊松 log $\log\mu$ $\mu=e^{\eta}$ Poisson 回归
正偏连续 Gamma log（常用） $\mu=e^{\eta}$ Gamma 回归

一个数值示例（log 连接的 Poisson）： 设线性预测子 $\eta=\beta_0+\beta_1 x$，某人 $\eta=1.6$，则预测均值 $\mu=e^{1.6}\approx4.95$——永远为正，天然适合计数；若 $\beta_1=0.69$，则 $x$ 每增 1，$\mu$ 变为原来的 $e^{0.69}\approx2$ 倍（率比 2）。

结论： GLM 的威力在于「一套框架、换旋钮即得不同模型」。二分类为何用 logit？因为它把概率映到实数、且反连接是 sigmoid 保证预测在 $[0,1]$；计数为何用 log？因为反连接 $e^\eta$ 恒正、且系数指数化后是率比。理解了连接函数，就理解了 Logistic 的 OR、Poisson 的 IRR 为什么是那样解释的。

§5

数据形式与输入输出

5.1 适合的数据形式

  • 自变量类型：连续、分类均可。
  • 因变量类型：连续、二分类、计数、比例、正偏连续。
  • 数据结构：独立观测的表格。
  • 是否适合高维数据：可结合正则化。
  • 是否适合缺失较多数据：需先处理。
  • 是否适合删失数据：不适合（用生存模型）。
  • 是否适合重复测量数据：不适合；用 GLMM/GEE。

5.2 示例表格

结局 选择
是否患病 二项 + logit
发作次数 泊松 + log
住院费用 Gamma + log

5.3 输入与产出

输入

  • 输入数据：协变量 + 结局。
  • 关键变量：结局类型（定分布与连接）。
  • 需要预处理的内容：选分布族与连接、编码、查过度离散。

产出

  • 模型对象/统计结果：系数、SE、偏差、AIC。
  • 参数估计：连接尺度系数及其反连接解释。
  • 预测结果：$\mu=g^{-1}(\eta)$。
  • 不确定性指标：系数/预测 CI。
§6

适用场景

  • 适合：为非正态结局统一建模、需可解释系数。
  • 不适合：相关数据（用 GLMM/GEE）、删失（用生存模型）、强非线性未处理。
  • 使用前需要特别检查的点：分布族是否匹配、是否过度离散、连接尺度线性。
§7

实现

常用包：

  • statsmodels
import statsmodels.api as sm
import statsmodels.formula.api as smf

# 二分类 → 二项族 + logit
smf.glm("y ~ x", data=df, family=sm.families.Binomial()).fit()
# 计数 → 泊松族 + log
smf.glm("count ~ x", data=df, family=sm.families.Poisson()).fit()

常用包：

  • stats
glm(y ~ x, family = binomial(link = "logit"), data = df)   # Logistic
glm(count ~ x, family = poisson(link = "log"), data = df)  # Poisson
§8

结果如何解读

  • 核心结果看什么：连接尺度系数 + 其反连接解释（OR/IRR）、偏差/AIC、CI。
  • 每个主要参数如何解读：logit 尺度系数指数化为 OR；log 尺度系数指数化为率比。
  • 临床或医学意义如何表达：用反连接后的可解释量（OR/IRR/均值）报告。
  • 常见误读：忽视过度离散把 SE 当准；在错误尺度上线性解释。
§9

假设诊断与稳健性

  • 过度离散：泊松/二项族看「偏差/自由度」是否远大于 1；大则用 quasi 或负二项。
  • 连接与线性：偏差残差 vs 拟合值图、部分残差图。
  • 影响点：GLM 也有 Cook 距离等诊断。
  • 相关数据：改用 GLMM（个体效应）或 GEE（边际）。
§10

推荐可视化

  • 反连接后的预测曲线（如概率/率随 x 变化）。
  • 偏差残差诊断图。
  • 系数（OR/IRR）森林图。
§11

优势、局限与常见坑

优势

  • 统一框架，涵盖多种结局。
  • 系数经反连接有清晰解释。
  • 估计与推断理论成熟（MLE/IRLS）。

局限

  • 假设结局独立。
  • 分布族选错会失真（过度离散）。
  • 连接尺度的线性仍是限制。

常见坑

  • 结局类型与分布族不匹配。
  • 忽视过度离散。
  • 直接对结局取对数代替 log 连接（遇零失效、改变解释）。
§12

与相近方法的区别

  • 和线性回归：正态+恒等连接的 GLM 即线性回归。
  • 和 Logistic/Poisson 回归：它们是 GLM 的具体成员。
  • 和 GLMM/GEE：相关数据下对 GLM 的扩展。
  • 和广义可加模型（GAM）：GAM 允许非线性光滑项，放宽线性假设。
§13

医学研究中的典型应用

  • 二分类疾病结局的 Logistic 建模。
  • 事件计数/发病率的 Poisson 建模。
  • 医疗费用等正偏连续结局的 Gamma 建模。
§14

关键术语

  • 连接函数（Link）：把期望 $\mu$ 映到线性预测子 $\eta$ 的函数。
  • 线性预测子（$\eta=X\beta$）：协变量的线性组合。
  • 指数族（Exponential Family）：GLM 允许的分布族。
  • 偏差（Deviance）：GLM 的拟合优度度量。
  • IRLS：迭代重加权最小二乘，GLM 的估计算法。
  • 典范连接：使数学最简的默认连接。
§15

相关方法

  • 线性回归（Linear Regression）
  • Logistic回归（Logistic Regression）
  • Poisson回归（Poisson Regression）
  • 广义估计方程（Generalized Estimating Equations, GEE）
