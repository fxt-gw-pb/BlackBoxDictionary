- **方法**：简单线性回归、单因素 ANOVA、$R^2$、残差诊断、事后检验

## 研究问题

以**对数变换后的胆固醇水平**（log-transformed cholesterol level）为结局。

### (a) 以 BMI 为单一协变量的线性模型

- **i.** 检验"BMI 无效应"的原假设。用图形可视化数据以检查二者关系。
- **ii.** 报告 $R^2$ 值，并验证：当只有一个预测变量时，$R^2$ 等于 $Y$ 与 $X$ 的 Pearson 相关系数的平方。
- **iii.** 画出残差图，检查：线性模型设定是否正确、误差是否正态且同方差、是否存在离群点（outlier）。

### (b) 以年龄组为因子的 ANOVA 模型

- **i.** 检验"年龄无效应"的原假设。用图形可视化数据以检查二者关系。
- **ii.** 画出残差图，评估误差的正态性与同方差性。
- **iii.** 若原假设被拒绝，进行事后检验（post-hoc test）。

### (c) 以性别为因子的 ANOVA 模型

- **i.** 检验"性别无效应"的原假设。用图形可视化数据以检查二者关系。
- **ii.** 画出残差图，评估误差的正态性与同方差性。
