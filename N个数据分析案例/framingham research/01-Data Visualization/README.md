# 01 · 数据可视化（Data Visualization）

> 通过图形探索结局与协变量的分布及其边际关系，为后续建模提供直觉与假设。

📘 **完整分析报告 / 学习笔记**：[report.md](report.md)（源码 [report.Rmd](report.Rmd)，纯代码 [code.R](code.R)）

- **数据**：[`../rawdata/Framingham_data.csv`](../rawdata/Framingham_data.csv)，基线 PERIOD 1
- **方法**：分布图、分组对比图、散点/箱线/柱状图等描述性可视化

## 研究问题

围绕主线问题——理解个体 **BMI** 与其**血清总胆固醇**水平在基线（第 1 期）的关系——本模块用图形探索结局与协变量的分布及边际关系。胆固醇既可看作连续变量，也可看作二分类变量（总胆固醇 ≥200 mg/dL 为 "undesirable"，否则 "desirable"）；可视化需关注年龄组与性别，并初步考察它们是否修饰 BMI 与胆固醇状态的关系。

1. **分布**：展示结局变量胆固醇水平（连续型与二分类型）的分布，以及三个协变量 **BMI、年龄组、性别**的分布。

2. **边际关系**：展示结局变量分别对三个协变量的（边际）关系。

3. **图形规范**：为图形添加清晰的标签（labels）与图例（legends）。
