## 本机数据集位置

原始数据集体积较大（约 1.1GB），不放在 Obsidian 库和 git 仓库内，实际存放于：

```text
~/Datasets/黑箱字典/dataset/
├── 001/GlobalTemperatures.csv
├── 002/（电影票房多表：movies_metadata、credits、keywords、ratings 等）
├── 003/house-prices-advanced-regression-techniques/
├── 004/Churn_Modelling.csv
├── 005/all_stocks_5yr.csv
├── 006/primary_results.csv
├── 007/data.csv（乳腺癌分类数据）
├── 008/Mall_Customers.csv
├── 009/creditcard.csv
└── 010/spam.csv
```

`code/dataset` 是指向上述目录的符号链接，案例脚本中的 `./dataset/NNN/...` 相对路径仍可正常使用（在 `code/` 目录下运行即可）。换机或重装时，从上方网盘链接下载数据后重建该目录与符号链接：

```bash
ln -s ~/Datasets/黑箱字典/dataset "本库路径/05_数据分析案例/code/dataset"
```

Framingham 数据（512K）体积小且为医学统计主线的优先演示数据，仍保留在库内：`Framingham研究/rawdata/Framingham_data.csv`。
