Cox 比例风险模型

Cox Proportional Hazards Model
状态 · ● 已校 难度 · intermediate 标签 · 生存分析 / 半参数 / 风险比 / 删失
§1

方法概览

Cox 比例风险模型是生存分析中最常用的回归工具，用于量化一组协变量对事件发生风险（hazard）的影响。它之所以被称为「半参数」，是因为它不对基线风险 h₀(t) 的形状作任何假设，只对协变量的作用形式作出设定。

这一特性使它在处理删失数据（censoring）时既灵活又稳健，成为临床随访研究里估计治疗与预后因素效应的标准方法。

§2

数学形式

模型将个体在时刻 t、给定协变量 x 的风险函数写作基线风险与指数项之积：

$$h(t \mid \mathbf{x}) = h_0(t)\,\exp\!\left(\beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_p x_p\right)$$

参数 β 通过最大化 Cox 的偏似然（partial likelihood）估计，它只依赖事件发生的次序而非具体时点：

$$L(\boldsymbol{\beta}) = \prod_{i:\,\delta_i = 1} \frac{\exp(\boldsymbol{\beta}^{\top}\mathbf{x}_i)}{\displaystyle\sum_{j \in R(t_i)} \exp(\boldsymbol{\beta}^{\top}\mathbf{x}_j)}$$

其中 R(tᵢ) 为 tᵢ 时刻的风险集，δᵢ 为事件指示。系数的指数即风险比 $\mathrm{HR} = \exp(\beta_k)$，是模型解释的核心量。

§3

数据形式与输入输出

变量 角色 说明
time生存时间观测到事件或删失的时长
event事件指示1 = 发生，0 = 右删失
x₁…xₚ协变量连续或分类的预后 / 治疗因素

输出 → 各协变量系数 β、风险比 HR、95% 置信区间、Wald / 似然比 p 值、C-index 与基线累积风险。

§4

适用场景

  • —随访研究中评估治疗或暴露对生存 / 复发风险的影响；
  • —存在右删失、且事件时点分布未知或不便参数化时；
  • —需要在多个预后因素间调整、构建风险评分或列线图。
§5

实现

Python · lifelines R · survival 
from lifelines import CoxPHFitter
import pandas as pd

df = pd.read_csv("rossi.csv")

cph = CoxPHFitter()
cph.fit(df, duration_col="week", event_col="arrest")

cph.print_summary()          # HR、95% CI、p 值
cph.plot()                   # 森林图
cph.check_assumptions(df)    # 比例风险假设检验
 
library(survival)
library(survminer)

fit <- coxph(Surv(week, arrest) ~ fin + age + prio,
             data = rossi)

summary(fit)                 # HR = exp(coef)、95% CI、p
cox.zph(fit)                 # 比例风险假设 (Schoenfeld)
ggforest(fit, data = rossi)  # 森林图
§6

结果解释

系数 β 本身无直觉意义，需取指数还原为风险比。以 HR 读数：HR > 1 表示协变量升高使风险增大，HR < 1 为保护性，HR = 1 无影响。

「比例」二字是全部解释的前提：模型假设任意两组的风险比不随时间改变。报告结果前应以 Schoenfeld 残差检验（cox.zph）核验此假设，否则 HR 会失去恒定的意义。

§8

优势 · 局限与常见坑

优势

无需设定基线风险形状；自然处理删失；系数解释为风险比，临床沟通友好；成熟软件与诊断工具完备。

局限 / 坑

比例风险假设常被忽视；对时依效应无能为力（需时依协变量）；处理大量结点（ties）需近似；不直接给出绝对风险。

§10

医学应用

在肿瘤学随访中，Cox 模型被用于同时评估分期、分子标志物与治疗方案对总生存与无进展生存的独立影响。例如某三期临床试验以治疗组为主协变量，调整年龄、ECOG 评分后得 HR = 0.72（95% CI 0.58–0.90），提示试验方案将死亡风险降低约 28%。
