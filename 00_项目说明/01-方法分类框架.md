# 方法分类框架

这份框架用于统一整个“医学统计与数据科学方法字典”的组织方式。当前更推荐按“具体方法家族”来划分，而不是设置一个过于宽泛的 `机器学习` 总目录。

## 一级分类

| 一级分类 | 核心问题 | 典型方法 |
| --- | --- | --- |
| 研究设计与数据理解 | 研究怎么设计，数据怎么认识 | 样本量估计、偏倚评估、变量定义、数据质控 |
| 描述统计与统计推断 | 数据有什么特征，组间是否存在差异 | 描述统计、t 检验、卡方检验、方差分析、非参数检验 |
| 回归与广义线性模型 | 自变量与结局之间如何建模 | 线性回归、Logistic 回归、Poisson 回归、GLM |
| 生存分析与纵向数据 | 存在删失或重复测量时如何分析 | Kaplan-Meier、Cox、AFT、混合效应模型、GEE |
| 因果推断 | 如何减少混杂并尽量靠近因果解释 | 倾向评分、IPTW、双重稳健、工具变量 |
| 正则化与变量选择 | 如何筛选变量并控制模型复杂度 | Ridge、LASSO、Elastic Net、逐步选择 |
| 树模型与集成学习 | 如何用树结构或集成方法建模 | 决策树、随机森林、XGBoost、GBDT |
| 支持向量机与核方法 | 如何通过间隔最大化和核技巧建模 | SVM、SVR、RBF 核、多项式核 |
| 聚类与无监督学习 | 如何发现潜在亚群与结构 | K-means、层次聚类、DBSCAN、GMM |
| 降维与表征学习 | 如何压缩维度并保留主要结构 | PCA、ICA、t-SNE、UMAP、PLS |
| 神经网络与深度学习 | 如何建模复杂非线性映射 | MLP、CNN、RNN、Transformer |
| 时间序列与时序建模 | 如何处理带顺序依赖的数据 | ARIMA、VAR、Holt-Winters、Prophet |
| 数据预处理与特征工程 | 如何把原始数据变成可建模输入 | 标准化、归一化、编码、缺失处理、构造特征 |
| 高维数据与多组学 | 变量很多、样本较少时如何建模 | 稀疏建模、组学特征选择、批次效应校正 |
| 模型评估与解释 | 模型好不好、稳不稳、能否解释 | 交叉验证、ROC/AUC、校准曲线、SHAP、重采样 |
| 分类与判别方法 | 如何把个体分到预定义类别中 | K近邻、朴素贝叶斯、LDA、QDA |

## 每一类里优先收录什么

### 1. 研究设计与数据理解

- 研究设计本身不是“模型”，但决定后续方法能否正确使用
- 建议收录样本量估计、偏倚类型、变量类型、数据清洗原则、缺失机制判断

### 2. 描述统计与统计推断

- 这是医学统计最常见的入口层
- 建议把参数检验和非参数检验都纳入，并强调适用前提

### 3. 回归与广义线性模型

- 这是医学研究中最重要的一类方法
- 建议先按结局变量类型组织，再补变量选择和正则化方法

### 4. 生存分析与纵向数据

- 医学数据常见删失、随访和重复测量
- 建议单独成类，避免被普通回归方法掩盖

### 5. 因果推断

- 医学观察性研究里非常重要
- 建议每个方法都强调识别假设与局限，而不只写算法步骤

### 6. 正则化与变量选择

- 建议优先收录 LASSO、Ridge、Elastic Net、逐步选择等高频方法
- 这一类适合作为回归建模与高维建模之间的桥梁

### 7. 树模型与集成学习

- 建议优先收录决策树、随机森林、XGBoost、LightGBM 一类方法
- 这一类更适合作为独立门类，而不是笼统放在“机器学习”下

### 8. 支持向量机与核方法

- 建议把 SVM、SVR 及常见核函数放在一起，便于读者统一查找

### 9. 聚类与无监督学习

- 适合承接客户分群、疾病亚型发现、表型聚类等内容

### 10. 降维与表征学习

- 建议单独收录 PCA、ICA、t-SNE、UMAP、PLS 等方法
- 这样比“聚类与降维”混放更清晰

### 11. 神经网络与深度学习

- 建议先从多层感知机这类前馈神经网络入手，再逐步扩展到 CNN、RNN、Transformer
- 当项目后续涉及图像、文本、多模态和复杂非线性预测时，这一类会变得越来越重要

### 12. 时间序列与时序建模

- 适合容纳 ARIMA、VAR、状态空间模型、时序深度学习等

### 13. 数据预处理与特征工程

- 很适合作为网站里“建模前准备”专题，而不是散落在各方法之间

### 14. 高维数据与多组学

- 这类问题常见于基因组学、蛋白组学、代谢组学
- 建议强调小样本高维、批次效应、特征选择和稳定性

### 15. 模型评估与解释

- 这一类很容易被忽视，但在医学建模里非常关键
- 建议把“区分度、校准、临床效用、可解释性”拆开写清楚

### 16. 分类与判别方法

- 这一类适合收纳以“个体判别”或“模式识别”为主线的方法
- 当某个方法不自然属于回归、树模型、支持向量机等现有家族时，可以放在这里
- 典型条目包括 K 近邻分类、朴素贝叶斯、LDA、QDA、感知机

## 后续可扩展的二级专题

- 医学影像分析
- 临床文本与 NLP
- 贝叶斯统计
- 外部验证与模型更新
- 报告规范与证据评价

## 组织原则

- 一个方法只保留一个主条目，避免重复
- 相近方法通过“相关方法”互相链接
- 分类以“主要用途”为准，不强求唯一绝对归属
- 如果一个方法横跨多类，主分类放最常见用法，其余通过标签补充
- 对已经在其他家族中有稳定主条目的分类方法，如 Logistic 回归、支持向量机、树模型，不必强行再迁入本类
