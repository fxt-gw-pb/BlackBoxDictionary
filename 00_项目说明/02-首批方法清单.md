# 首批方法清单

这份清单用于指导项目的第一阶段建设。原则是先覆盖医学研究中最常用、最容易混淆、最值得沉淀的方法。

## 第一优先级：基础高频方法

这些方法最适合作为第一批样板。

| 方法 | 主要任务 | 典型结局/数据 | 优先级 |
| --- | --- | --- | --- |
| t 检验 | 两组均值比较 | 连续型结局 | P0 |
| 卡方检验 | 分类变量关联分析 | 列联表 | P0 |
| 方差分析 | 多组均值比较 | 连续型结局 | P0 |
| 相关分析 | 线性或秩相关 | 连续或等级变量 | P0 |
| 线性回归 | 连续结局建模 | 表格数据 | P0 |
| Logistic 回归 | 二分类结局建模 | 表格数据 | P0 |
| Cox 比例风险模型 | 时间到事件建模 | 生存数据 | P0 |
| Kaplan-Meier 曲线 | 生存概率描述 | 生存数据 | P0 |
| 混合效应模型 | 重复测量/分层数据 | 纵向数据 | P0 |
| 倾向评分方法 | 混杂调整 | 观察性研究 | P0 |
| 多重插补 | 缺失数据处理 | 表格数据 | P0 |
| LASSO | 变量筛选/正则化 | 中高维表格数据 | P0 |
| 随机森林 | 非线性预测建模 | 表格数据 | P0 |
| XGBoost | 高性能预测建模 | 表格数据 | P0 |
| 主成分分析 | 降维 | 高维连续变量 | P0 |
| ROC/AUC | 区分度评估 | 二分类模型 | P0 |
| 校准曲线 | 预测概率校准 | 风险预测模型 | P0 |
| 决策曲线分析 | 临床效用评估 | 风险预测模型 | P0 |

## 第三优先级：专题扩展

适合在字典形成一定规模后继续扩展。

- 贝叶斯回归
- 因果森林
- 工具变量
- 目标试验模拟
- 深度神经网络
- 生物标志物筛选
- 批次效应校正
- 医学影像分割与分类
- 临床文本分类与信息抽取

## 建议的起步组合

如果只先写 5 篇，我建议优先写：

1. Logistic 回归
2. Cox 比例风险模型
3. 混合效应模型
4. 倾向评分匹配
5. 随机森林

这 5 篇可以覆盖：

- 经典统计建模
- 生存分析
- 纵向数据
- 因果推断
- 树模型与集成学习
